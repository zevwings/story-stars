# 都市星辰契约题材口径

> 本文件是本项目实际采用的都市超凡题材口径正典。它定义了现代秩序与隐秘星辰契约超凡的融合规则、主角心理咨询师的咨询循环以及星辰契约能力的局限性。具体世界观设定、角色关系、历史因果与专名，以 `BIBLE.md` 与 `worldbuilding/` 为准。

## 1. 核心承诺
都市星辰契约的叙事承诺是：**冰冷的星辰契约超凡力量嵌套于高度写实的现代都市中，而所有超凡对抗与心理咨询，最终都服务于人物间的情感羁绊、创伤救赎与秩序重建。**

它不同于传统“装逼打脸”的都市高武，超凡能力并非直接的杀戮武器，而是探寻人心创伤与灵魂契约的探针。

---

## 2. 主循环
每一卷或每一个故事弧线应遵循以下循环：
1. **现代日常/咨询切入**：男主在依法登记的私人心理咨询工作室接待来访者，或在日常都市生活（职场、学院）中遇到异常状况。
2. **异变显影**：来访者或目标表现出精神海动荡、星辰异变（同源异化）或受到黑暗契约的精神控制（造梦/污染/篡改记忆）。
3. **限知介入**：男主以心理咨询师身份访谈侧写，同时以隐秘的“双子星契约者”身份动用梦境/幻境进行探查或咨询辅助。
4. **博弈与代价**：在隐秘的世界中与敌对势力、黑暗契约者对抗，使用截断/逆流自保或反制，承受星辰异变的反噬风险。
5. **救赎与后果**：通过建立信任与情感链接（引发亲密），协助对方稳固心神、降解异变，并引发社会层面（管理局/家族）与人际关系层面（修罗场/协商成阵）的连锁反应。

## 3. 与传统都市玄幻的对比

| 维度 | 传统都市玄幻 | 都市星辰契约（本项目） |
| :--- | :--- | :--- |
| **力量源泉** | 灵气、天地元气、功法修炼 | **星空结契**：一星一契，目视结契，主动权在星辰侧 |
| **战力体系** | 筑基/金丹等境界，升级换地图 | **无综合修炼境界**：精神海五境仅描述承载状态，无“破境”仪式 |
| **战斗机制** | 肉体强化、法术对轰、法宝飞天 | **物理层结算**：星辰契约不提供肉搏强化，动作伤害靠物理对抗结算 |
| **主角定位** | 最强战神、仙帝重修、杀伐果断 | **感知/辅助流**：隐秘双子星契约者，医生身份，非正面最强战士 |
| **亲密关系** | 战力附庸、收女涨数值 | **亲密驱动**：先有情感后有欲望，亲密是救赎异变的唯一防线 |

## 4. 核心设定边界与红线

### 4.1 结契与星辰之力
*   **一星一契**：普通人一生只能与一颗星结契。男主是全书唯一的“双子星契约”特例（亮星梦境/幻境 + 暗伴星截断/逆流），此设定为核心独特性支柱，不可泛滥。
*   **起源模糊**：不解释星辰契约的宇宙起源，不设定正典起源史。
*   **星表与隐秘**：管理局维护的“星表”仅是事后登记工具，而非结契通道。男主只将亮星登记成普通单星梦境契约，隐瞒幻境、暗伴星、截断与逆流；“官方知道他是契约者，却错误理解了他的完整结构”是关键剧情张力。

### 4.2 战斗语法与物理限制
*   **星辰契约不强化肉体**：不存在能让人跑得像火车、一拳打穿钢板的肉体强化星辰契约。近身搏斗完全依赖现实物理格斗。
*   **药剂限制**：药剂是受管制的肉体兴奋剂（有严重副作用），只短时刺激身体机能，不介入精神层战斗。
*   **精神系收敛**：精神能力的交锋是双向的。男主的幻境不修改人格、不改记忆；逆流和截断只能对正在攻击或已暴露的通道生效。

### 4.3 精神海五境定义
精神之海按承载分为：**泉境、湖境、川境、海境、浩瀚境**。
*   此境界只描述承载深广状态，不是修炼阶段，不存在境界壁垒和“破境”仪式。
*   战力高低由精神海状态、控制力、权柄及现场条件综合决定，不可用境界高低直接判定输赢。

## 5. 现代化改造与势力层级
*   **表层（日常/官方）**：普通市民社会，对超凡力量基本无知；星辰管理局作为官方监管机构，负责信息外泄管控。
*   **中层（民间/交易）**：私人心理咨询工作室、特殊药剂流通渠道、黑市交易网络。
*   **深层（传统/敌对）**：掌握血缘与星表资源的传统星辰家族，以及进行非法精神实验或掠夺的黑暗契约者。

---

## 6. 禁区
1.  **禁“修仙化”描写**：禁止出现“御剑飞行”、“打坐吐纳一整天”、“破境雷劫”等传统修仙词汇和桥段。
2.  **禁越界操控**：男主绝对不能使用梦境或幻境非法操纵、篡改自己人或女主角的意识。
3.  **禁脱离法律与痕迹**：超凡争斗绝非无法无天，每次出手必须考虑管理局的干预、警方的刑事调查以及社交网络的外泄风险。
4.  **禁战斗数值化**：不要在战斗中出现具体的战力值、灵力百分比或功法层数，多写现场感知与精神消耗的疲惫感。
