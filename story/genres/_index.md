## 本书采用

- `都市星辰契约`: 架空都市背景、一星一契规则、无境界升级、近身搏斗走物理层结算。
- `18+HD纯肉`: B 场正向 HD 为全书主色带与兑现主体（最高频、连续、有美感），生理感官细节与性心理描写的精细化写作规范，彻底排除油腻病灶词。
- `都市日常`: 私人心理咨询工作室执业、职场与社交博弈、跨人物日常场域、共同生活边界、幽默甜宠拉扯与亲密蓄势。
- `深色情欲带`: 正向线暗色情趣（支配 / 臣服 / 蒙眼 / 束缚 / 绝对主导）的技法与安全边界；双方成立、清醒可拒绝，与黑暗场靠五条分界线区分，档位按角色 + 协作启用。
- `黑暗压迫场`: 反派 NC / 压迫场的立场与边界口径，双声部铁律，仅对敌 / 反派、不落男主对女主；占比提升为常规张力色带，是口径页、执行走 hd-dark skill。
- `后宫群像经营`: 多女主群像的平衡、横向关系与三阶段总弧（隐秘并行 → 修罗场 → 协商成阵），防工具化 / 龙傲天 / 雌竞羞辱。

## 本书不采用

- 传统仙侠/修真式的升级境界、境界壁垒和破境雷劫仪式。
- 强化肉体力量/速度 of 星辰或异能设定。
- 功利性双修机制（即亲密行为能双修回蓝、提升战力或突破境界）。
