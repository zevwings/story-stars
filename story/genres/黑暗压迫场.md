# 黑暗压迫场 题材口径

> 本文件是本项目**黑暗压迫场**（反派 NC / 压迫场）的机制模块口径与**立场页**，不替代 story-writer 的 `intimacy-hd-dark-planner / writer` skill——skill 管执行流程（`design.b.dark` 设计卡 → 正文），本页管本项目黑暗场的立场、边界、禁区与词汇口径。场型分工、尺度红线以 `BIBLE.md` 为准，双声部铁律继承 `BIBLE.md` / `STYLE.md`。

---

## 1. 定位与触发

黑暗压迫场 ＝ **反派 / 对敌 / 黑暗契约者**施加的性压迫场，是色带谱最深的一档。触发即**强制走 hd-dark skill 流程**（先出 `design.b.dark` 设计卡再写正文），不得即兴。

- **仅落黑暗线**：施加方永远是反派侧；**男主不对自己的女主施真实非自愿压迫**（那不属本页）。
- **触发按三轴，不看处境也不看尺度**（`BIBLE.md`「场型归属判据」）：**目的轴**——以造成不可逆后果为目的，或明知会造成仍施加；**意愿轴**——任意一方不同意，含中途撤回后仍继续；**判断轴**——当场完全不能判断或完全不能选择。任一触发即本页，三轴都不触发即不进本页。
  - **伤害＝不可逆**：本场结束后仍然存在、当事人自己无法消除的后果（影像把柄、伤残、疾病、妊娠相关的不可逆后果、被公开的身份）。疼痛、瘀青、痕迹、当场的羞耻会消退，不是伤害——**强度高不等于本页**，深色带 C 档在三轴清白时仍属正向线。
  - **只判参与方的行为**：由场外第三方对参与者造成的不可逆伤害（偷拍、泄露、外部胁迫）不把参与者的场拉进本页——他们是被害方，其场仍属正向线；**加害的第三方按其自身行为进本页或案件线**。
  - **催化不是剥夺**：酒精、欲望权柄、情绪只放大既有欲望，判断仍在即不进本页；捡尸（已断片、无反应）与恐吓下的「同意」才进。
  - **处境不利本身不触发**：当次自愿而处境不自由的兑现（利益交换、生计压力、身份落差）不进本页，走 `BIBLE.md`「结构失衡不抹除」。把它写进本页会同时冤枉正向线人物并稀释本页的批判立场。
  - **实施方即定性**：触发三轴而实施的一方即按黑暗线侧处理，不论其在人物结构中的位置。判定是**逐行为方**的——同一事件里，一方可属本页，另一方仍属正向场。
- **常规色带，占比提升**：较旧口径不再收敛为罕见特例，可作为常规张力 / 剧情重量按需出现，每卷可有若干张力点；但「占比提升」指频率，不指立场——**立场永远是批判性的，不写成情色爽点**。

## 2. 双声部铁律（本项目黑暗场的地基）

继承 `BIBLE.md` / `STYLE.md` 定义，不改动，只落地：

> 快感是施暴方的、账单是受害方的；文本不认同施暴方；受害方痛苦与施暴方快感同重量在场；必落后果，不用甜蜜余韵收尾；不把身体反应写成愿意、求生配合写成迎合。

**落地写法**——两个声部**并置**在场：施暴方的快感 / 掌控写出来（不回避），但同一段里受害方的痛苦、抵抗、屈辱以**同等重量**在场；镜头不站在施暴方那侧欣赏，叙述的体温始终在受害方一边。读者可以看到施暴方爽，但**文本不请读者一起爽**。

---

## 3. 施暴方不美化

- 不把施暴写成「强大有魅力」：不给他情欲滤镜、不写成霸道总裁式的危险吸引。
- 他的快感可以写，但**不奖赏**——不让施暴带来正向的情欲美感，不给温存、不给「其实她也……」。
- 力量是压迫工具，不是魅力值。星辰之力若参与，是**加害手段**（如欲望 / 侵蚀类权柄），写其恶，不写其炫。

---

## 4. 受害方主体性与反物化

- **痛苦同重量在场**：受害方不是道具，她的恐惧、疼痛、屈辱、抵抗必须有分量，不被施暴方的视角盖过。
- **求生配合 ≠ 迎合**：为活命 / 减少伤害的配合，不得写成「身体背叛」「其实享受」。身体的非自主反应（若写）必须明确框为**生理机制 / 施暴产物**，不是愿意。
- **不写成沦陷**：黑暗场不走「被强到爱上」「驯服」的路径——那是对双声部的背叛。

---

## 5. 失衡后果（必落账单）

黑暗场**必须**留下后果，不能像正向场那样温存收尾：

- **身体**：伤、痕、需要恢复 / 治疗（可接共鸣救赎线）。
- **心理**：创伤、应激、信任受损——不速愈，给恢复以时间和过程。
- **剧情**：成为仇恨 / 复仇 / 救赎的账单，推动主线；施暴方最终要在剧情里付代价。

后果的存在本身，就是文本「不认同施暴方」的证明。

---

## 6. 反派专用词汇边界

参考库标注的粗俗 / 物化词汇（见 `18+HD纯肉.md` §6），在黑暗场**施暴方声部完全启用**，作为塑造施暴方污秽、让读者厌恶施暴方的手段。**「完全启用」指档位，不指位置——使用边界不变**：

- 只进施暴方的语言 / 视角，不进叙述本体，不进受害方声部。
- 服务于塑造与批判，不是无目的堆砌感官。
- 正向线一律不用，不得借黑暗场把粗俗词漏进正向兑现。

## 7. 与深色情欲带的区分

同样有支配 / 束缚 / 疼痛，深色带与黑暗场靠「意愿根、可撤回、动机、事后、文本立场」五条分界（见 `深色情欲带.md` §2）。**一句话**：深色带里她能喊停且喊停被尊重、事后被接住、文本认同双方；黑暗场三者全无。写前先判这是哪一侧，再选流程。

**⚠️ 扮演式强制不属于本页**：正向线允许约定内的「假装不愿意」，它归深色带（`深色情欲带.md` §4），受三项硬前置约束。它表面与本页最像，**判断不看表面动作，看三件事**——有没有在正文里可见的约定、有没有另设并当场兑现过的真实停止信号、结尾走照护还是走账单。三项齐备走深色带流程；缺任何一项都不成立为情趣，且**它不因此变成本页的场**——本页只收反派侧的真实压迫，男主对女主的场不论如何都不进 hd-dark 流程（§8.2）。

**⚠️ 自愿交换场也不属于本页**：当次自愿、但参与者处于结构性不自由中的兑现（利益交换、生计压力、身份落差），既不是深色带（内核是交换与失衡，不是权力交换的情趣），也不是本页（三轴均不触发）。它归正向线，走 `BIBLE.md`「结构失衡不抹除」——失衡真实在场、不推定后续授权、必落现实后果，但**不套双声部**。判断口径同上：过三轴，不看处境是否不利。

## 8. 禁区

1. **年龄硬底线**：所有参与者年满 18，**不得借黑暗场规避**。
2. **不落男主对女主**：男主对自己人无真实非自愿压迫。
3. **不滑成爽点**：不写成情色奖赏、不给施暴方魅力、不走沦陷驯服。
4. **不消费受害者**：痛苦不是用来点缀的调味，受害方是人不是剧情道具。
5. **不用甜蜜余韵收尾**：黑暗场不套正向场的温存、迎合、余韵。

### 未成年人保护边界

- 未成年人不得成为黑暗压迫场、性交易过程、客户偏好、身体反应或情色镜头的参与者与观察对象。
- 若案件涉及未成年人，叙事只进入发现、脱离危险、医疗与心理稳定、安置、监护核验、教育衔接、证据保护和责任追查。
- 不通过回忆、证言复现、能力读取、录像材料或施暴方视角补写色情过程；必要事实以非过程化、最小充分的信息确认。
- 未成年人保护与成年性交易、成年当事人恢复、白玥私人关系及任何正向亲密线严格分开，不以“案件需要”建立情色交叉。
- 在出现具有独立变化主体、保护接口与完成判据的正式人物或事件以前，不为未成年人保护预建空剧情节点。
