# 18+ HD 纯肉题材口径

> 本文件是本项目 18+ HD 场景（B 场正向 HD 为主体）的机制模块口径。场型分工、尺度红线以 `BIBLE.md` 为准，文风三基调（写实＋美感修辞、纯写实不用异能滤镜、张弛结合）以 `STYLE.md` 为准；具体写法素材按需调用 story-writer 亲密参考库（`story-writer://references/intimacy/_index.md`），本文只保存项目侧口径，不复制参考库内容。

## 1. 核心承诺
本题材的叙事承诺是：**极致的感官细节还原、随着本能交互而步步失陷的性心理状态，以及压在这一切之下的关系重量。**

**B 场（正向 HD）是全书的主色带与兑现主体，也是最高频、最高优先的场型**：兑现要多、要连续、要有美感，是常态而非稀客。但高频不等于流水账——HD 场景仍是关系推进与欲望兑现的高潮节点，不是孤立肉戏：先有日常蓄势与情感信任（关系立住后可用短铺垫，靠关系存量供能，不必每场重铺），写完之后，关系必须和进场时不一样。拒绝空洞生硬的流水账式生理动作描写，也拒绝与人物关系脱钩的纯器官特写。

## 2. 场型分流
写前先判场型，本文件只直接约束 B 场：

*   **A 场（高暧昧 / 软切黑 / 正文亲密，非 HD）**：走正文亲密设计流程，美学与张力参考 `19_A场美学与性张力`；不适用本文件的兑现节奏。**A 场保留为连续兑现里的呼吸留白、过渡与过场**——重点场写满 B 场，过渡场用软切黑或写实含蓄承接，靠详略搭配做节奏，而不是拿软切黑当偷懒默认。
*   **B 场（正向 HD，男主 × 女主正向线）**：本文件的主口径，**全书主色带与兑现主体，最高频、最高优先**。参考库配方：`01_B` + `02_B` + `03_B` + `13_节奏` + `17_进阶技法`。
*   **深色情欲带（正向线暗色情趣）**：支配 / 臣服 / 蒙眼 / 绝对主导 / 暗色情趣，在 B 场框架内写，走本文件节奏；**双方成立、清醒可拒绝——那是情趣不是压迫**，真实的选择权失衡只属于黑暗场（见第 4 节底线）。**技法、与黑暗场的分界线与尺度档位详见 `深色情欲带.md`。**
*   **黑暗 HD / 压迫场（仅对敌 / 反派，可按需常规使用，不落男主对女主）**：必须先走黑暗 HD 流程与黑暗设计卡，执行双声部铁律（快感是施暴方的、账单是受害方的、必落后果）；**严禁**套用本文件的正向节奏、迎合心理与温存余韵。**立场、边界与词汇口径详见 `黑暗压迫场.md`。**

**参考库剔除项**：本项目无双修机制，参考库中 `08_双修场景` 与各文件的“双修改编”段一律不适用；经脉 / 识海 / 丹田 / 灵力光晕类修真反馈词不得进入正文——亲密描写纯写实，星辰之力不做性爱滤镜。

## 3. 五感感官矩阵
为了避免干瘪的动作描写，必须在每一幕中交叉使用五感细节：
*   **触觉**：皮肤的温差（微凉与滚烫）、汗水的黏腻、指甲掐入肌肉的力度、压迫感与包裹感。
*   **视觉**：局部特写（泛红的眼角、因紧绷而起伏的锁骨）、光影的投射（阴影下的交缠）、体液折射的微光。
*   **听觉**：呼吸的粗重与短促、低声的喘息、无法自抑的呜咽、黏潮的碰撞声、被刻意压低的求饶或命令。
*   **嗅味**：汗味与体温烘出的气息、发间与颈侧的味道、亲吻里的津液与呼吸交换。
*   **温度与重量**：体温上升的过程、呼出热气的吐息、交界处的滚烫；肢体压迫的沉重感、掌控与被掌控的实体力量对比。

## 4. 性心理双向轴
性爱不仅仅是生理碰撞，更是心理防线的拉扯与交互。写作时需在以下心理状态中切换：
*   **主导方**：占有欲的膨胀、恶劣的试探、看对方失控的愉悦、克制之后的爆发、怜惜与掌控的夹杂。
*   **被动方**：心理防线的崩溃、羞耻感的挣扎、感官超载的迷失、被迫承受的顺从、主动迎合的羞涩。
*   **交互状态**：别扭、拉扯、求饶与不予、理智崩塌前的最后一次推拒。

**清醒可拒绝底线（正向线专属）**：女主的推拒只来自羞耻、别扭与情趣拉扯，不是真实非自愿；绝对主导可以成立，但那是双方成立的情趣，不是压迫。真实的选择权失衡只属于黑暗场，不得渗入 B 场。

## 5. HD 场景推进线与节奏
推进线：**前奏 → 确认 → 升级 → 收束 → 余波**。不必每场走满，但每场都要能说清当前在哪一步。（细化技法见参考库 `13_节奏`、`11_结构`）

**连续兑现的详略节奏**：当多场亲密连续发生时，按关系节点分详略——里程碑场（首次、破防、关系质变）写满五步兑现；之间的日常兑现可只取前奏 → 升级 → 余波的骨架，或用软切黑 / 写实含蓄承接，避免每场同强度堆到底。高频靠的是“常发生”，不是“每场都最重”。

*   **前奏（边缘试探）**
    *   *动作*：亲吻、抚摸敏感带、衣物半褪、呼吸变调。
    *   *心理*：心跳失速、羞耻感累积、目光试探与躲闪。
*   **确认**：关键接触前后要有意愿信号在场——不一定是语言，可以是动作、停顿、眼神或主动靠近；一方退缩就降速改写。
*   **升级（防线破裂 → 感官超载）**
    *   *动作*：关键的结合与切入、物理阻力、身体的适应与接纳；持续的物理撞击、体位转换、生理失控（战栗、痉挛）。
    *   *心理*：强烈的异物感与充实感并存，心理防线瞬间失陷，羞耻被本能的潮水淹没；意识模糊，除了快感与对方的体温外失去一切感知。
    *   *节奏*：走波浪式（上升 → 小高潮 → 回落 → 再上升），用故意中断、体位变换、刺激叠加制造起伏，不写匀速到底的直线。
*   **收束（余韵温存）**
    *   *动作*：肢体瘫软、汗水交融的拥抱、温存清理与照顾。
    *   *心理*：极度疲惫后的心安、依恋感倍增、理智回归时的微弱羞涩。
    *   *要求*：优先写人物怎么缓下来，不重复前面的强度；身体负担要交代照顾与恢复。
*   **余波**：关系变化必须落在动作、对白或沉默里，给后续剧情留空间——这是 HD 场景对主线的交付物。

**句式与呼吸感**：句式长短随烈度联动——铺垫用长句沉浸，冲刺用短句密集，极致高潮允许碎片化断句；高压段善用“压拍点”（动作峰值处硬生生停住，写静止里的心跳与汗珠）与声音延迟（动作先发、那声破碎的呻吟慢半拍才溢出）。

## 6. 禁用油腻词与写实替换词表

| 禁用词 | 导致问题 | 推荐替换方案 |
| :--- | :--- | :--- |
| **娇躯、玉体** | 悬浮、没有实感 | *写实部位*：肩膀、后背、腰肢、绷紧的大腿、脚踝 |
| **樱唇、檀口** | 陈腐、同质化 | *动作细节*：微张的唇缝、被咬出白印的下唇、唇齿厮磨 |
| **电流窜过、酥麻** | 夸张、套路化 | *生理反应*：脊背不可自抑地战栗、指尖发麻、头皮炸开、物理颤抖 |
| **花蕊、幽径、分身、巨龙** | 俗气、出戏、粗鄙 | *写实/借代*：结合处、最深处、滚烫的源头、湿软的内里 |
| **娇喘连连** | 模板化敷衍 | *具体声音*：细碎的呜咽、压在喉咙里的闷哼、带哭腔的急喘、断续的呼吸 |
| **巨乳、罩杯数字、淫穴类粗俗命名** | 物化、廉价 | 正向线一律不用；参考库标注的“反派专用词汇”只允许出现在黑暗场施暴方声部 |
| **灵力翻涌、识海震荡、丹田发烫** | 机制滤镜、违反纯写实基调 | 回到身体本身：呼吸、心跳、肌肉、体温、失神 |

**淫语使用口径**：允许分级使用（见参考库 `04_词汇库_B`），但少而准、以点爆破——重度淫语只做单点，必须有角色语气与人设兜住，说完要有符合人物的回响，不整段堆砌，不当自动替换器。

## 7. 禁区
1.  **唯一硬底线**：场景所有参与者必须年满 18 岁，正邪场同等适用，不得借黑暗场规避。
2.  **禁功利性爱**：坚决禁止将 18+ 场景作为“恢复星辰之力”、“涨战力”、“提数值”的升级工具；亲密只在情感 / 救赎维度起作用（降星辰异变、稳心神），且因果方向锁死——先有欲望与信任，收益是自然结果，不是索取性的理由。
3.  **禁降智贴倒**：女主不能毫无缘由、毫无情感防线地迎合，必须有日常相处或危机累积的逻辑。
4.  **禁场型混用**：男主不对自己的女主施真实非自愿压迫；黑暗场不得借用本文件的迎合心理与甜蜜余韵收尾。
5.  **禁词汇越界**：严格遵守禁用词表，宁可写实，也不使用低俗、过时、悬浮或机制滤镜类修辞。
