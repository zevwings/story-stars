# 深色情欲带 题材口径

> 本文件是本项目**深色情欲带**（正向线暗色情趣）的机制模块口径，是 `18+HD纯肉.md` B 场框架内的暗色子带。场型分工、尺度红线以 `BIBLE.md` 为准，文风以 `STYLE.md` 为准；通用禁区与禁词继承 `18+HD纯肉.md`。具体写法素材只调用本项目 `intimacy/references/`，调用条件和风险边界读取本项目 `intimacy/mechanisms/`；不读取 story-writer 公共亲密参考或离线 packs。

---

## 1. 定位：它是什么、不是什么

深色情欲带是**正向线（男主 × 女主）在 B 场框架内的暗色子带**——权力不对等的情趣：支配 / 臣服 / 蒙眼 / 束缚 / 绝对主导 / 命令与服从。它的地基与黑暗场相反：

- **黑暗场**：`BIBLE.md` 三轴任一触发——以不可逆后果为目的、任意一方不同意、或当场完全不能判断；文本不认同施暴方（双声部）。
- **深色情欲带**：选择权始终在场，被支配方**主动把控制权交出去**，清醒、可随时收回；文本与双方都站在「这是我们要的」这一侧。

一句话：**黑暗场是「压迫伪装成情趣」，深色带是「情趣借用压迫的形式」**。形式可以像，内核相反。

---

## 2. 与黑暗场的分界线（全页最硬的红线）

同一个动作（按住、命令、蒙眼、施加痛感），是情趣还是压迫，由下面几条**同时**判定——缺一条就滑向黑暗场，必须改写或降档：

| 判定轴 | 深色情欲带（成立） | 黑暗场（禁止落在男主对女主） |
| :--- | :--- | :--- |
| 意愿根 | 当次自愿，推拒来自羞耻 / 别扭 / 情趣拉扯 | 受害方真实不想要，且被强制越过 |
| 可撤回 | 任何时刻能喊停，喊停即止、被尊重 | 喊停无效或无喊停通道 |
| 支配方动机 | 在乎对方、为共同的快感 | 为发泄 / 占有 / 惩罚 / 摧毁 |
| 事后 | 有照护、确认、关系加深 | 有创伤、账单、后果 |
| 文本立场 | 认同双方 | 不认同施暴方（双声部） |

**上位判据是 `BIBLE.md` 三轴（目的 / 意愿 / 判断），本表是深色带的落地检查表**：意愿根与可撤回落在意愿轴，支配方动机落在目的轴，事后与文本立场是**写法要求不是判据**。判归属回三轴，写这一场看本表。

**「意愿根」只问当次，不问动机**：为什么愿意——喜欢、好奇、金钱、利益交换、生计压力、身份或关系落差——都不改变归属；只有**当次不能拒绝**才落黑暗场（`BIBLE.md`「场型归属判据」）。反过来，处境不自由也不自动开放深色带：**自愿交换场不是深色带**——它的内核是交换与失衡，不是权力交换的情趣，走 `BIBLE.md`「结构失衡不抹除」，不套本页的照护动线与臣服美感。

**写作铁律**：深色带里「绝对主导」是被**交付**的权力，不是被**夺取**的权力。被支配方的**主体性一刻都不能消失**——她是「选择臣服」，不是「被消灭意志」。一旦她的意愿在文本里变得不重要，这场就不再是深色带。

## 3. 权力交换的美感内核

深色带写得好不好，不在动作多重，而在**「交付」这件事写得多可信**：

- **支配方的克制**：真正的掌控是「我随时能停，所以我敢给你更多」，不是失控施虐。掌控里要有**在乎**——留意对方的边界、状态、承受度；越是主导，越要写出他在读她。
- **臣服方的主动**：臣服是「我把控制权给你」的主动姿态，不是被动挨受。写她**选择**放手时的紧张、信任、释放——羞耻和快感来自「我居然愿意」，而不是「我被迫」。
- **信任是燃料**：深色带只能建立在关系存量上——蒙眼 / 束缚要求她看不见 / 动不了还信你，这是关系里程碑，不是随机场。深色带优先放在信任已立住的节点。

## 4. 常见情趣类型与各自的写法边界

- **蒙眼 / 感官剥夺**：美感＝剥夺视觉后其他感官被放大、对声音和触碰的依赖、「看不见却信你」的交付。边界＝事前有默契、随时能摘、他持续用声音 / 触碰给她锚点，不制造真实恐惧。
- **束缚 / 拘束**：美感＝失去行动权后的被动承受与信任、挣动与被安抚。边界＝不造成真实伤害 / 窒息，松紧可控，她一句话即解。
- **命令与服从**：美感＝言语支配的张力、服从里的别扭与隐秘快感、「听话」后的奖赏。边界＝命令内容不越她的底线，拒绝一个命令不招致真实惩罚，是游戏不是规训。
- **绝对主导 / 掌控体位**：美感＝力量对比、被完全掌控的安全感与失控感并存。边界＝主导方始终在读她的反应，过载就降速。
- **轻度疼痛感 / 力度**（掐、咬、拍打）：美感＝痛与快的边界模糊、痛感作为强度的调味。边界＝**痛感是调味不是主菜**，不留真实伤害，她的反应是迎合不是逃避。
- **扮演式强制（约定内的「假装不愿意」）**：美感＝在两个人都知道是假的前提下，把「被夺取」演出来；张力来自演与真之间那道**双方都清楚、读者也看得见**的缝。这是全页表面最像黑暗场的一档，**三项硬前置缺一即出界**：
  1. **约定在正文可见**——不必写成谈判流水账，可以落在前一场、前几段的对话里，或事后回指；但读者只看得到表面，**读者看不到约定，就等于没有约定**。
  2. **另设真实停止信号，并在场上兑现过一次**——「不要」「不行」一旦被征用成台词，就不再是停止通道；必须另有一个词或动作是真的，且这一场里真的用过一次、他真的停了。缺这条等于取消停止权。
  3. **事后照护加重**——退出角色的动作必须明写，并有一句「刚才是我们一起要的」的确认。**扮演得越像，退出越要清楚。**

  边界＝不复刻黑暗场的写法：不写真实恐惧、不给她真实的无助、不使用施暴方专用词汇（那是 [黑暗压迫场](./黑暗压迫场.md) §6 的档位），不落身体 / 心理 / 剧情后果（后果是黑暗场的账单，不是情趣的余韵）。§2 五轴一条都不放宽——意愿根仍然是「她想要」，可撤回仍然必须成立。**准入**＝不按 A / B / C 强度档开放，须该角色亲密卡「深色带适配（速查）」节明确授权（§7），卡上没写即未授权。

## 5. 安全信号与照护（写实化，不搬现实术语）

深色带不必出现「安全词」这类现实术语（除非人物设定需要），但**功能等价物必须在场**，写实化落地：

- **停顿信号**：一个两人之间成立的「停」——一个词、一个动作、身体僵住。支配方读到就降速 / 停下，这一读一停本身就是深色带最动人的地方。
- **状态确认**：升级前后有确认（眼神、气音的问、身体的回应），不是自说自话地推进。
- **事后照护（aftercare）**：深色带**尤其**要写事后——从「支配 / 臣服」的角色里退出来，回到平等，确认「你还好、我在、刚才是我们一起要的」。深色带的余波不是普通收束，是**把她接住**；缺了照护，再美的过程也会读出凉意。

## 6. 写法落地：从原则到动作

§2 的分界、§3 的内核与 §5 的照护如果只停在原则，正文里就会写成「她完全信任他」「他很在乎她」这类旁白。以下是它们各自对应的可写动作，按需取用，不必一场用全。

### 6.1 五轴的动作译法

§2 五条判定轴是判据，判据本身不进正文；正文里它们靠动作被看见。

| 判定轴 | 在正文里靠什么动作被看见 |
| :--- | :--- |
| 意愿根 | 她先开口提的、她把东西递过来、推拒之后自己又靠回去——不是他说服了她 |
| 可撤回 | 她真的喊过一次停、他真的停了、然后两个人怎么继续。**这一条最好在场上真的发生一次**，比写十句「随时可以停」有用 |
| 支配方动机 | 他的注意力始终在她的反应上：中途降速、换姿势、把滑下来的东西扶正、问一句 |
| 事后 | 从角色里退出来的那个动作——松手、盖被子、递水、叫名字（不是称呼） |
| 文本立场 | 叙述贴着两个人，不站在旁边欣赏；她的感受与他的快感在同一段里同时在场 |

### 6.2 权力交接的可写动作

- **提出**：谁先说、怎么说、说完之后那几秒的空。
- **协商**：说好哪些可以、哪些今天不行——协商本身是前戏，不是流程。
- **约定信号**：一个词、一个动作、捏两下手。约定过就必须在场上真的可用。
- **第一次试探**：轻的那一下、看她的反应、她点头或者没点。
- **当场收回**：她中途改主意时，写他怎么接住这件事，比写他怎么主导更重要。
- **事后归位**：角色退出、称呼退出、回到平等的第一句话。

### 6.3 主导方的照料动线

**写主导必须连带写照料**，两者是同一条动线：

- **事前**：准备什么、把什么放到伸手够得到的地方、检查、试温。
- **场上**：读她——呼吸变了、身体僵了、声音低下去；据此降速、调整、问一句。
- **收尾**：解除的顺序、先扶起来还是先松开、有没有等她缓过来。
- **事后**：热、水、被子，以及把她接住的那句话。

**「他在读她」不是一句形容，是一连串具体的调整动作**；写不出这些动作，主导就只剩施加。

### 6.4 臣服方主动性的可写动作

臣服是选择，选择必须被看见：

- 她**挑**了哪一项、**拒**了哪一项——拒绝一小项而接受另一项，是主体性最省字的证明。
- 她主动把手腕递过去、转过身、闭上眼——**动作由她完成**。
- 中途改主意：说出来，或者身体先停住；他停下之后，她自己解释或者不解释。
- 事后说出哪一下不舒服、哪一下想再来一次。

一旦她全程只是被动接受，这场就该改写（§2 意愿根、§8 主体性不灭）。

**扮演式强制里的主体性另算**（§4）：她在框架内演抗拒时，表面的抗拒不算被动接受，但主体性必须落在框架之外的三处——**这场是她挑的、她划过哪条不行、她真喊停时他停了**。三处一处都不能缺，缺了就只剩抗拒的表演，与黑暗场无从分辨。

### 6.5 仪式与称呼

- 称呼、规矩与角色**当次成立，不自动延续到下一场**；下一场要重新进入。
- 启用与退出都要有动作：什么时候开始这么叫、结束时谁先叫回名字。
- 具体用什么称呼随人物走，回该角色的亲密卡与当次设计卡，本页不预设到人。

### 6.6 写法禁区

- **禁把支配写成不需要确认**：主导不免除询问；不问的主导是压迫的写法。
- **禁把臣服写成没有主体**：不写「她已经无法思考」「任由他摆布」这类抹掉意愿的句子。
- **禁只写主导的爽**：照料动线缺席的深色带没写完（§5）。
- **禁用旁白直接下结论**：把「她完全信任他」换成她把手腕递过去的那个动作。
- 器物、衣装与空间的具体词条回 `intimacy/references/` 三页，不在本页展开；那三页只给写法词条，不构成人物事实，也不构成调用许可。

## 7. 尺度档位（按角色 + 协作启用）

深色带**不设全书统一硬上限**，档位随**角色特质与当场协作**启用——不同女主适配的深度不同，同一场也可随情节推进换档：

- **A 档（轻）**：命令与服从、蒙眼、轻束缚、掌控体位，情趣为主、几乎无痛感。
- **B 档（中）**：较强束缚、明确的支配-臣服角色扮演、轻度疼痛感（掐咬、力度）作调味。
- **C 档（重）**：羞辱性言语（双方情趣内）、较明显的疼痛感、更彻底的控制。

选档看角色：谁适配到哪一档、启用哪些情趣，随该女主的亲密卡 / 当场 skill 逐步确定（本页不预设到人）。无论哪档，§2 的分界线与 §5 的照护都不可省。

**三档均可进正向线**：C 档不是禁区——羞辱性言语在情趣内且当场结束、疼痛会消退、控制随场解除，按 `BIBLE.md`「伤害＝不可逆」都属可逆，三轴清白即属正向线。**有人本身就是重度取向，这是人物差异不是越界。**开放 C 档同时锁死五条：

1. **停止通道不得取消**：羞辱与彻底控制最容易把「不要」变成台词——停止信号必须始终存在，且当场兑现过一次。
2. **照护随强度上升**：C 档的事后照护要求高于 A / B，不是同一份分量（§5）。
3. **不空降**：C 档不出现在关系未立住时，更不出现在第一次。
4. **逐卡授权，不由 A / B 推出**：与扮演式强制同一逻辑——角色卡「深色带适配（速查）」节未写明即未授权，不得由档位、相邻角色或「这场气氛到了」推出。
5. **强度上限不等于不可逆风险上限**：窒息、颈部受力、药物推动、偷拍与任何留下把柄或永久损伤的做法，**不因 C 档开放而放宽**——它们被禁不是因为「太重」，是因为不可逆（`intimacy/mechanisms/亲密道具调用规范.md` 正向线禁用行）。

> **当前授权状态**：现有角色卡维持原档，**开放 C 档不等于任何人升档**。谁到 C 由各卡单独确认。

**探索广度是另一条轴，不由档位推出**：A / B / C 只回答**强度**（痛感与控制程度），不回答**花样多少**。一个人完全可以玩得很多、很杂、很爱试，而每一样都不重——那属于「探索广度」，与强度无关。

- 广度高＝愿意试的品类多、换得勤、把新玩法当共同游戏、会自己提议下一个。
- **广度不改变强度上限**：不因为「什么都试过」就推出可以上羞辱、明显疼痛或更彻底的控制。想升强度，仍须回本节看该角色的档位。
- 需要时在角色亲密卡「深色带适配（速查）」里单列一行标注；不写即按常规处理，不由档位或相邻角色推出。

**扮演式强制不按本节档位开放**：它（§4）的风险不在强度而在**表达通道被征用**，因此与 A / B / C 无关——A 档角色可以适配，B 档角色也不自动获得。准入只看**该角色亲密卡「深色带适配（速查）」节是否明确授权**；卡上没写即未授权，不得由档位、由相邻角色或由「本场强度不高」推出。

**与规范层档位的关系**：本节 A / B / C 是**情趣强度**档，`intimacy/mechanisms/` 三页规范的基础 / 进阶 / 高门槛是**器物与场域的风险准入**档，两者是不同维度，不做换算。同一场两套各自判定：情趣强度按本节看角色，器物、衣装与空间按规范页取三者最低档；任一套不成立即降档，不以另一套已成立为由放宽。

## 8. 禁区

1. **不滑向黑暗场**：任何时候被支配方变成「真的不想要」，立即出界，必须改写或降档（§2）。
2. **主体性不灭**：被支配方不降智、不失去意愿的分量；臣服是选择，不是意志被抹掉。
3. **照护不可省**：深色带没有事后照护 ＝ 没写完。
4. **信任前置**：高强度深色带不空降在关系未立住时。
5. **不越通用禁区**：年龄、禁功利、禁降智贴倒、词汇边界，全部继承 `18+HD纯肉.md` §7。
6. **扮演式强制不得取消停止通道**：约定内的「假装不愿意」允许，但「不要」被征用成台词后，必须另设真实停止信号并在场上兑现过一次；§4 三项硬前置缺一即出界。

## 9. 写前检查项

- [ ] §2 五条分界线是否全部落在「情趣」侧？有没有一条滑向压迫？
- [ ] 被支配方的主体性、意愿是否全程在场且有分量？
- [ ] 是否写出了支配方「在读她」的在乎，而非单向施加？
- [ ] 停顿信号 / 状态确认是否在关键升级点在场？
- [ ] 事后照护是否落地（退出角色、回到平等、把她接住）？
- [ ] §6 的原则是否都落成了**动作**，而不是停在「她完全信任他」这类旁白结论？
- [ ] 「可撤回」是否在场上真的兑现过一次，而不只是被声明？
- [ ] 本场强度是否匹配该角色适配的档位？情趣强度档与器物 / 场域的规范档是否分别判过，没有互相放宽？
- [ ] 若本场用了扮演式强制：约定在正文里可见吗？另设的停止信号是否真的兑现过一次？退出角色是否明写？
- [ ] 是否继承了 `18+HD纯肉.md` 的通用禁区与禁词？
