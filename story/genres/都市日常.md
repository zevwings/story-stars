# 都市日常题材口径

> 本文件是本项目实际采用的都市日常与职场题材口径正典。它规范了心理咨询工作室执业、社会阶层交往、日常暧昧蓄势及俏皮幽默的对话文风。具体角色档案与章节状态，以 `BIBLE.md`、`characters/` 与 `chapters/` 为准。

## 1. 核心承诺
都市日常的叙事承诺是：**日常生活是有温度、有质感且逻辑真实的；它是人物情感在日常互动中长根的蓄势场，所有的情感救赎与亲密欲望，皆源于日常的信任累积。**

它拒绝悬浮的“豪门套路”与空中楼阁的恋爱，要在写实的柴米油盐、医生执业与职场社交中展现真实的亲密关系推进。

---

## 2. 三大日常场域与写法

### 2.1 心理咨询工作室（咨询师执业与职业素养）
*   **职业写实**：林语心理咨询工作室依法登记，晏林具备相应心理咨询执业资格。访谈过程应体现写实的心理学素养（如言语侧写、表情观察、建立安全感），将超凡梦境／幻境隐藏在知情授权下的现实咨询与助眠引导中。
*   **硬边界**：晏林必须谨守职业道德与私人边界。**绝对不从正在接受咨询的来访者关系中发展恋情**；女主入场一律走契约者圈内线，而非咨询关系线。

### 2.2 都市阶层（市民与家族的现实碰撞）
*   **阶层差写实**：晏林是普通市民白领出身，而核心女主中包含掌握大量星表和资源的“星辰家族”传统权贵。
*   **现实质感**：阶层之间的碰撞要照现实来写。体现在商务合同、资源分配、生活习性、社会声誉及权力规则的客观存在，主角不能用武力或异能轻易抹平现实的阶层壁垒。

### 2.3 学院、疗养与同居同行（人际升温空间）
*   **封闭或半封闭环境**：利用学校的社团掩护、疗养所的私密空间或偶发的短期同居/同行，制造合法的“同处一室”机会。
*   **微动作张力**：通过日常的分担家务、共进早餐、眼神碰撞、微小细节的关心，自然展现情感的升温与依赖。

## 3. 跨人物日常运行协议

跨人物日常不是独立于人物线之外的第四种剧情线，而是人物关系、职业生活与共同秩序反复落地的公共承载层。它负责让人物在危机和功能身份之外继续生活，也负责让不同关系在同一座城市中发生低压交叉。

### 3.1 全书职责

- **关系长根**：关系建立前，日常负责积累信任、暴露偏好与确认边界；关系建立后，允许依靠已有关系存量自然复现，不要求每场都制造新的里程碑。
- **高压回落**：案件、职场冲突、协查或关系争执之后，必须保留身体恢复、现实善后与情绪余波；轻松场不能抹掉责任和后果。
- **独立生活**：人物不能只在自己的案件、职业或亲密功能中出现。每个人都应保留自己的工作、休息、社交、照料责任与拒绝空间。
- **城市实感**：人物需要真实使用工作场、住宅、街道、社区与休息时间；城市不是只供设定说明的背景板。
- **共同秩序验证**：中后期的知情、陪伴、留宿、排期与公开边界，要在日常执行中被验证，不能只停在会议或口头约定。

### 3.2 长期场域

| 场域 | 正式职责 | 硬边界 |
| --- | --- | --- |
| 林语心理咨询工作室 | 承担职业运转、普通生活与异常世界之间的缓冲，以及授权、记录和知情权限的现实分层 | 不发展咨询关系恋情；普通员工、女主和来访者不得越权接触咨询隐私、梦境材料或管理局档案 |
| 员工宿舍与共同居住空间 | 承担晏林、许知夏、许知秋的共同生活，以及私人空间与公共生活权限的逐步协商 | 独立卧室、身体靠近和私人行程分别取得同意；共同居住不等于行程审查、家务义务或关系绑定 |
| 周末与城市生活 | 承担跨人物低压相遇、事件后回落、生活习惯显影及双人／三人组合差异 | 优先使用最小有效组合，不把所有人物固定成大团建；休息日不得破坏工作室与其他职业场的可信度 |
| 宠物、诊所与社区生活 | 承担温婉的职业入口、群像责任观及普通社区连接 | 动物不充当媒人、测谎仪、异常探测器或案件答案；救助必须保留医疗、费用、安置与长期责任 |
| 酒吧与都市夜生活 | 承担成年社交、职业边界、普通越界与隐秘痕迹并存的现实场面 | 酒精不构成同意；苏璃不自动承担调查职责；酒吧不固定为违法场所或案件中心 |
| 家庭、旧友与普通社会关系 | 从职业、伴侣、生活选择与公开程度向人物关系施压 | 普通人只接触已同意公开的外层，不自动知晓星契秘密，也不拥有关系审批权；单次见面不扩成家庭或旧友主线 |

### 3.3 家庭与亲缘回声

家庭与亲缘关系负责检验人物的普通身份、职业生活、身体状态与关系公开程度，也可以作为高压事件后的低烈度生活回落。家人只能接触当事人同意公开的普通生活外层，不因亲缘自动取得星辰契约秘密、管理局协查材料、咨询隐私或伴侣关系的审批权。

家庭关心可以带来职业稳定、作息、健康、婚恋与公开身份压力，但不得替代人物自身的关系选择，也不扩写为隐藏血脉、星契家族或靠巧合触发的案件线。电话问候、回家、饭桌交谈与工作场外侧偶遇均为可按章节需要调用的普通场景；只有造成以后必须继承的人物、关系、知情或公开边界变化时，才提炼为正式剧情节点。

### 3.4 跨人物调用规则

1. 同一日常场可以服务多条人物线，但每名在场人物都必须保留自己的立场、知情范围和选择权。
2. 优先选择能够完成当前任务的最小人物组合；双人、三人和多人场应有不同功能，不以全员到场制造虚假群像感。
3. 场域只提供相遇和生活压力，不吞并人物线：宿舍不替代双胞胎的独立关系，宠物场不替代温婉的职业与关系变化，周末活动不替代任何人的定情或知情节点。
4. 跨人物相遇不自动扩大秘密权限。亲密关系、共同居住或私人熟悉都不能换取咨询隐私、完整协查材料、未登记能力或他人关系信息。
5. 照顾必须保留被照顾者的选择。高压型人物在低压场中可以表达需要、接受照顾或暂时不作判断，但不得被幼态化、生活低能化，也不得由他人替其决定。
6. 已成立关系允许出现不承担新变化的普通生活与亲密复现；靠详略和人物质地区分场景，不把“每场必须升级”变成新的门槛。
7. 普通社会层负责处理骚扰、纠纷、就医症状、经营后果与现实证据；隐秘痕迹只向有权限的人开放。梦境、心理残留和管理局判断不能替代现实证据。
8. 天气只改变路线、时间、可见度、行动成本与情绪密度，不制造同意，也不单独制造超自然异常；具体天气留到章纲和场景施工时确定。
9. 早餐、夜宵、买菜、做饭、分账与口味记忆可以表现照料、权限和关系存量，但不能替代披露、关系确认、冲突解决或同意；照料始终允许拒绝，也不固定为女性后勤职责。

### 3.5 正式化路由

- 跨人物日常的稳定功能、共同边界与调用原则，写在本文件。
- 某类日常在一个阶段内的启动、持续、交叉或收束，写入 `story/arcs/阶段X.md`。
- 日常事件造成以后必须继承的人物、关系、知情或生活权限变化，提炼为 `story/plotlines/` 正式节点。
- 已确认的固定地点、住宿结构、机构职责与社会规则，写入 `worldbuilding/` 对应专题。
- 人物稳定偏好、生活习惯与性格边界，写入 `characters/` 对应人物卡。
- 具体场景、人物组合、对白、天气、路线、次数、章位与未决参数，保留在施工层或章节大纲，不进入正式题材口径。

## 4. 日常的核心功能：亲密蓄势与关系存量

日常不是可有可无的过渡。关系建立前，它负责让亲密拥有可信根基；关系建立后，它也可以作为既有默契、欲望与共同生活的自然释放，不要求每次兑现都重新完成一整套铺垫。

1. **暴露真实需要**：人物在日常咨询、闲聊、同行或共同生活中卸下部分功能身份，显出疲惫、偏好、迟疑、脆弱与拒绝。
2. **提供安全感**：晏林通过克制、守边界的陪伴和现实行动建立信任；亲密与陪伴可以稳定情绪，但不替代医疗、职业责任或当事人的自主选择。
3. **形成专属默契**：不同关系通过各自的生活习惯、对话方式、照顾边界和主动权形成可辨认的质地，不把所有人物写成同一种依赖。
4. **承接存量释放**：关系成立后，普通吃饭、留宿、回程与亲密复现本身即可成立；重点场承担变化，日常场维持关系的可见存在。
5. **行动切换张力**：日常生活中主角可以表现出拖延、怕麻烦、懒散等真实人味；涉及道德边界、安全红线或既定责任时，必须迅速进入收束状态：先处理风险，再安顿人，最后解释。

## 5. 对话与甜宠拉扯规范
*   **生活化对话**：句式要有人味，生活化、轻松俏皮。
*   **拉扯与暧昧**：甜感要靠对话中的“相互调侃、言语试探、潜台词拉扯”来体现，而不是靠大段深情的内心旁白或空洞的爱意告白。
*   **幽默点缀**：在日常互动和修罗场边缘加入适当的轻松调侃，缓解战斗或异变带来的沉重感，保证甜宠的底色。

## 6. 禁区
1.  **禁“无来由倒贴”**：女主必须是有血有肉、有迟疑、有真实性格反应的个体，禁止写成一见面就无底线迎合的完美工具人。
2.  **禁“脑残商战与装逼打脸”**：商界与社交博弈要体现情商与智商。不要出现为了打脸而刻意设计的降智反派。
3.  **禁“脱离现实常识”**：日常中的法律压力、社会舆论监管、房屋租赁、医疗登记等细节必须严密可靠，不可胡编乱造。
4.  **禁旁白说甜**：甜感必须通过动作（如帮她整理碎发、递过一杯温水）和神态微表情（如转过脸时微扬的嘴角）表现，而不是靠旁白解释“他们此时很甜”。
5.  **禁“过度精致与样板房独居”**：主角的日常起居允许保留普通人的懒散与合理杂乱（如：堆放衣物的椅子、没来得及收拾的杯子、拖延处理的琐事），拒绝写成不食人间烟火的悬浮样板房，以强化日常烟火气与生活温度。
6.  **禁“社交猎艳与乘人之危”**：都市夜生活与社交戏是暧昧的重要滋生地。主角在场子里可以会喝、会玩、会接暧昧的话，但严禁利用心理学技巧套路诱导他人，且**绝对禁止**在对方**已经完全不能判断或完全不能选择**时（断片、无反应、被恐吓而无从选择）推进任何实质性暧昧。微醺、情绪波动属催化不属剥夺，本条不因此扩大；真正被禁的是**把这些状态当成取得配合的手段**（`BIBLE.md` 目的轴与判断轴）。

## 7. 后宫恋爱与情感进阶
*   **情感演进的合理渐进**：恋爱关系推进必须走“防备/合作 -> 暴露创伤 -> 日常陪伴与暧昧拉扯 -> 专属信任与同盟”的渐进逻辑。严禁写空降倒贴或一见面就无条件百依百顺的女主。
*   **保留真实人味**：各女主在交往中要有自己的骄傲、吃醋、别扭与理智防御。多伴侣感情线的推进必须符合人物性格逻辑。

---

## 8. 修罗场与协商成阵博弈
*   **隐秘并行与张力错开**：在故事前中期，男主晏林与多位女主的感情线应隐秘并行，避免多线平铺直叙。通过限知视角，体现女主们在尚未发觉他人存在时的独特性。
*   **修罗场爆发与周旋**：关系暴露或重叠时（修罗场阶段），写出女主角们真实的醋意、别扭和心理防线冲突。男主的周旋应体现其心理咨询师的洞察力与高情商，不使用超凡能力进行欺骗或强力压制。
*   **协商成阵的秩序建立**：多伴侣共存的结局并不是某一方的彻底妥协或降智，而是通过在共同渡过生死危机后的理性与感性博弈，共同坐下来建立一个“明确席位、责任边界、危机共担与公开程度”的共存新秩序（契合 BIBLE 结局承诺）。

- [ ] 本场是承担新变化，还是承接既有关系存量？若有持续变化，是否已经路由到对应正式节点？
- [ ] 这一章的日常戏，是否体现了具体的现代职业或都市生活细节（如心理咨询工作室流程、合同、阶层差距）？
- [ ] 男女主角之间的对话，是否通过“相互试探/口头拉扯”体现了暧昧张力？

- [ ] 本章的情感进展，是否朝向结局“协商成阵”的理性自愿同盟方向前进一步？
- [ ] 场景里是否成功排除了油腻感与悬浮感？
