# PL-S-026｜万安文化艺人压迫与咨询显影

## 节点信息

- **节点 ID**：`PL-S-026`
- **线路类型**：副线
- **节点定位**：由程叙的现实危机与咨询入口显影万安文化反复压低求助紧迫感、阻止关键发声并配合职业封锁的艺人压迫模式，再与乔宁、梁予安及何知远的独立案件交叉验证。
- **归属目录**：`story/plotlines/副线/万安文化/`
- **当前状态**：pending

## 前置状态

- **人物前置**：裴兰馨负责万安文化端人员控制，任群是外部危机公关与有限行止执行者；程叙、乔宁、梁予安与何知远均为成年人，且不合并为同一受害经历。
- **关系前置**：程叙与裴兰馨之间存在无法有效拒绝的职权与能力压迫；裴兰馨其余利益型自愿关系必须逐段独立判定，不能由程叙一案反推全部关系。
- **世界观 / 势力前置**：林语心理咨询工作室只承担现实风险评估、咨询、转介与授权协作；医疗、管理局和司法分别承担诊断、异常能力核验与取证追责。
- **线索前置**：合同、项目与公关处置形成现实压迫痕迹；不同艺人重复出现求助紧迫感异常衰减或无法完成某个关键发声行为。

## 必须发生

- 程叙自杀未遂后由合作医院先完成现实医疗处置，稳定并同意后进入林语心理咨询工作室；晏林是本案的咨询师，但工作室不得以一次咨询直接诊断抑制或行止能力伤害。
- 程叙分阶段提供本人经历与可核验材料；梁予安的正规诊断和治疗记录、何知远的历史材料及乔宁主动外送的证据分别进入交叉验证。
- 调查区分裴兰馨压低既有警觉、愤怒、拒绝勇气与求助紧迫感的“心潮渐息”，以及任群针对少数当下具体行为施加的“令行禁止”，不把二者写成永久封口、群体控心或绝对服从。
- 乔宁拒绝身体换资源后被定向提供并遭受侵害，但她存活并亲自完成关键证据转移；具体接收者、裴兰馨与任群分别按自身知情和行为追责。
- 受害者可以选择匿名、暂停陈述、继续治疗或拒绝公开；案件显影不以公开创伤、立即复工或与施害者对质为条件。

## 本线变化

- **人物变化**：程叙从被孤立和失去行动紧迫感的受压者，转为能够在医疗、咨询与司法边界内分阶段作出选择的存活当事人；乔宁的主动证据转移获得正式案件位置。
- **关系变化**：裴兰馨与任群不再能以亲密关系、资源交换、危机公关或艺人表面配合遮蔽权力差与直接责任。
- **知情范围变化**：工作室和调查者由单一个案看到重复方法，但受害者身份、诊断、亲密经历与未授权证言继续受保护。
- **世界观 / 势力变化**：万安文化犯罪链由行业传闻转为可由合同、公关、医疗、通讯与精神干涉痕迹共同验证的责任结构，同时保留普通艺人和合法从业者的非涉案边界。

## 完成判据

- [ ] 正文或定稿摘要确认程叙经医疗处置后自愿进入工作室，并完成现实风险管理与必要转介。
- [ ] 正文或定稿摘要以程叙、梁予安、何知远和乔宁中至少三人的独立材料验证重复压迫方法。
- [ ] 正文或定稿摘要拆分裴兰馨、任群及乔宁案具体接收者的行为与责任，不用单一反派概括整条链。
- [ ] 正文或定稿摘要保留至少一名存活受害者对陈述、公开、治疗与职业路径的自主选择。
- [ ] 正文或定稿摘要没有把抑郁、自伤、自杀风险或死亡单一归因于星辰能力。
