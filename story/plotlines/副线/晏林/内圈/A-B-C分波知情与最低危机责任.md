# PL-S-022｜A/B/C 分波知情与最低危机责任

> 本页只保存阶段四 A/B/C 在当前危机中必须继承的分波告知、参与、退出、陪伴与暂停边界。危机场景、会议节拍、小组亲密流程与后期完整协商制度留施工层。

## 节点信息

- **节点 ID**：`PL-S-022`
- **线路类型**：副线
- **节点定位**：阶段四；管理局或黑暗线危机使 A、B、C 三簇的真实关系与安全风险无法继续分开处理，已确认关系者分波获得影响自身选择的必要事实，并形成只对当前危机有效的最低责任规则。
- **归属目录**：`story/plotlines/副线/晏林/内圈/`
- **当前状态**：pending

## 前置状态

- **人物前置**：苏璃、唐雨薇、苏晚乔、安若曦与江岚均保留自己的职业、关系、风险判断与退出权，不因进入同一危机面而成为统一阵型。
- **关系前置**：`PL-S-019` 已使 A/B 首次正式可见；`PL-I-012` 已使安若曦、江岚与晏林能执行最低三人互动秩序；江岚与晏林的关系定义由 `PL-I-011` 承接。
- **世界观 / 势力前置**：管理局复评、黑市责任链或星辰之印个案已使危险影响已确认关系者的安全与选择；关系知情不自动扩大成完整能力或案件知情。
- **线索前置**：阶段三的失踪、药剂、星表与管理局协查已从单线风险进入多人可被牵连的现实后果。

## 必须发生

- A、B、C 三簇按各自已知事实、安全风险与可承担责任分波获得必要信息，不以一次全员会议拉齐全部秘密。
- 苏璃、唐雨薇、苏晚乔、安若曦与江岚分别知道什么情况必须告知本人、自己可否参与、如何退出，不由其他人代替判断。
- 当前危机至少形成四项可执行边界：风险影响本人时必须告知；参与须单独确认；任何人可退出或暂停；陪伴与独处不因危机被统一分配。
- 晏林承认保护不能替代告知与选择，并在至少一项具体危机安排中实际遵守上述边界。

## 本线变化

- **人物变化**：晏林从分别隐瞒与单方保护，进入必须把影响本人的风险交还本人判断；五人各自建立参与、拒绝与暂停的实际位置。
- **关系变化**：A/B/C 从分簇或局部可见，进入可在当前危机中执行最低共同责任的状态；每段双人关系和已有小组秩序继续独立。
- **知情范围变化**：A、B、C 知道彼此在晏林关系中的真实存在，并按本人危险与责任获得必要事实；不自动知道双子星完整结构、全部案件材料或 D/E 簇的完整关系状态。
- **世界观 / 势力变化**：无新增机构或权限；内圈只形成当前危机的私人责任边界，不取得管理局协查或执法身份。
