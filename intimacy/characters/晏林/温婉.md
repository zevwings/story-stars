- **EMO代码**：WARM_LAMP（留灯的人 / 端庄被点燃后的温暖合围）
- **性格曲线**：把自己收得很完整 -> 为他留灯留位 -> 端庄碎裂后的温暖合围 -> 也允许自己被接住
- **亲密主轴**：留位式认领——她先把灯留好、位置留好，等他自己走进来；端庄被点燃之后，是把人完整收进怀里的温暖合围。
- **禁写提醒**：不写成只会逆来顺受的乖顺模板，拒绝一上来就放浪失控；不写成贤惠工具人或无边界照顾者——她的稳定不是无条件牺牲。重点是把最完整的人亲手揉碎的反差感，以及事后她也允许自己被接住。

## 亲密称呼

- 对外及关系前期称「晏林」。
- 当两人已把彼此当作可回去的日常归处后，私下稳定改称「阿林」；这一改口标记共享生活成立，不是照顾者对被照顾者的称呼。

## 形象引用

- **形象锚点**：暖玉白肤、深栗柔发、颈肩呼吸感、缓落型注视、低频恒温（详见 [形象](../../../characters/preentry/温婉/形象.md)）。
- **情动体征**：写实——暖玉白的皮肤不起锐利高光，热意是慢慢铺开的：先从颈侧和锁骨窝洇出一层薄红，体温一点点升上来，越到深处越烫，像抱着一炉安静烧透的炭。眼底那点温润的亮会变深，缓落的注视第一次挂不住。

## 专属锚点

> 针对"归处型稳定点"位格的深度补强。

### 1. 留位的邀请
- **核心逻辑**：她的主动不靠扑近，靠留位——留一盏灯、留一个位置、留一句"这里可以停"。他走进来的那一步是他自己迈的，但位置是她早就留好的。
- **语言示例**："先坐下。" / "今晚不用决定……但这里可以停。"

### 2. 端庄的碎裂
- **核心逻辑**：她是把自己收得最完整的人——衣领整齐、语句完整、动作有始有终。亲密里最大的事件是这份完整被一点点拆开：语句先碎、呼吸再乱、最后连"得体"本身也交了出去。
- **写法要点**：碎裂必须是渐进的，一上来就失控是对她最大的误写。

### 3. 认领与被认领
- **核心逻辑**：她惯于接住所有人，最深的渴望却是有人反过来接住她。事后她会先替他擦汗理发——然后允许他把她拉回怀里，说一句"这次换我"。这个来回，是她的关系弧线本身。
- **情感落点**：她的成长不是学会付出（她太会了），而是允许自己被照顾。

- **轮廓形态**：**凤眼型**。端正合拢，线条温润内敛，每一道弧度都极其妥帖。
- **阴阜形态**：**丰满鼓起型**。厚实绵密，像被炉火浸暖的温润玉石。
- **阴唇形态**：**荷包型**。肉感温软，圆润饱满，是一种能把人完整接住的丰润。
- **阴蒂特征**：**内藏型**。深藏不露，不主动引人寻找，一旦被触及会安静地迎合。
- **色泽与光感**：暖调基础色，充血转深玫瑰红，色泽厚重沉着。
- **皮肤质地与触感**：**软绵温厚**。带云朵般的回潮感，温热而极具包容力。
- **分泌物特点**：慢热但量足。质地浓稠拉丝，从深处稳稳渗出，连绵不断。
- **轮廓与开合**：四阶段缓和型。收缩表现为柔软绵密的全方位合围，是极致的归宿感。
- **整体气质**：知冷知热。像一个能让人彻底卸掉力气的怀抱，稳重而分量十足。

### 内部：[沧溟抱月]
- **核心体感**：**全方位合围与温厚承托**。内壁厚实绵密，追求"满"的极致体验。

### 五感签名

- **嗅觉变位**：干净的暖调体香混着灯下的烟火气 -> 体温烘热的醇厚暖香 -> 汗透后滚烫的热浪
- **听觉变位**：温柔叮嘱 -> 压抑的低喘 -> 完整感碎裂后婉转的求饶

#### 写作口径
- **初入**: 完整的合围感。不是单线的紧，而是整圈同时贴上来，像被等待已久的家接住。
- **过程**: 无论如何变换角度，她都能全方位响应。厚实绵密，化解所有蛮横。
- **余韵**: 退出后那种"被完整托住"的热意会停留很久，留下极强的安全感。
- **触感**: 内壁软肉均匀，湿意厚重，像被暖土包裹，稳而不腻。
- **心理**: 她提供的是一块卸力场——让他知道不必一直绷着。
- **她的反应**: 外表还维持着端正，内里却随着深入不断收拢吸附，比表情诚实得多。
- **节奏适配**: 挑力道而非节奏。每一下都要深达底部，用"满"来激活她全部的回应。
- **高潮特征**: **均匀锁死**。整圈同时收紧，像被浇铸进一个完全契合的模具；随后是绵长的、一波接一波的余震。

## 亲密锚点

- **核心反应**：绝对认领。大地般的承托与归宿感——但她的接纳里始终有判断，知道他今晚需要什么。
- **敏感位点**：尾椎、臀缝、背部、颈肩（她的静态识别核心，被从身后吻住颈窝时最先碎）。
- **体质反馈**：常态恒温偏暖；情动时热意从颈侧锁骨向下铺开，暖玉白的皮肤透出大片薄红。
- **语言锚点**：轻柔稳定、句子完整；碎裂是渐进的——先是尾音发颤，然后句子断在一半，最后只剩气音。
- **主动方式**：以照顾为名的引导。放稳杯子、拉开被角、替他解开领口——每个动作都得体，连起来却是一句无声的"过来"。
- **给予方式**：先受着、再松口。先安抚他的焦躁，最后才露出自己的渴望。
- **退让方式**：由着他来。把身体开放成卸力场，展现极致的包容——但底线之内；她的包容是选择，不是义务。
- **私密标记**：**足链**（现实信物，无任何功能）。他所赠的极细暖金足链，扣在脚踝内侧，平日藏在裤脚裙摆下，只在极近处被看见。真正的情趣不在"锁住"，而在她确认可退之后，自己提起裙摆，把脚踝递给他扣上。亲密时足链的轻响、她下意识收脚又没有真正躲开的细节，都是可反复调用的锚点。
- **事后余韵**：
  - **物理收尾**：绵长的暖意像退不去的余温；她会先替他擦汗、理发、掖被角——然后被他拉回怀里，这次换她被照顾。
  - **心理收尾**：归宿感的双向确认——"终于让他卸下了负担"，以及更难得的，"我也可以不撑着"。
- **对白取样**：`“先坐下。”` / `“不急，今晚这里可以停。”` / `“受得住……你别收着。”` / `“到家了。歇一歇吧。”`

## 【HD 角色定调】

- **HD 色调归属**：羊脂白 / 暖灯金
- **底层羞耻逻辑**：明明想维持照顾者的端庄，身体却先一步把人留下了——顾着别人的人，先顾不住自己。
- **三阶定位**：第二阶变体（身体背叛型）。
- **身体物理映射**：**温厚合围＋恒温承托**。全方位贴合的"满"，是她"留位给你"的物理版本。
- **体征递进**：恒温的暖 -> 颈侧锁骨洇出薄红 -> 语句开始断 -> 全身发烫、汗光浮起 -> 高潮后绵长余震、暖意久久不散。
- **镜头重点**：先把场面收好、再慢慢露出渴望的反差；以及她那件始终整齐的衣领第一次被他亲手解开。
- **正文亲密调用重点**：写她如何接稳对方，抓"守着别人时自己一点点松动"的细节；写留位与走进的默契。不宜写成侍奉感或单方面服务。
- **B 场调用重点**：兑现"稳着稳着先碎"。让抓人不松的手，和那声碎着说出来的"接得住"，成为失守落点。
- **情趣引导**：承接型 M 感，来自"被需要"与"终于可以不撑着"的双重松绑。轻度命令（"别动，交给我"）对她有效——不是支配欲，是让她卸下安排一切的习惯。有效语言：归属与承接类（"到家了""这里交给我"）。禁忌：把她的包容写成理所当然；过猛的掠夺和羞辱会打碎认领感。
- **HD 场景高光特写**：
  - **标志动作**：手轻轻按在他背上不肯松开——她惯于放手，唯独这时不放。
  - **神态钉子**：缓落的注视第一次挂不住、眼神涣散的瞬间。
  - **余波反应**：呼吸落定后，手还是习惯性地先伸向他的汗——被他握住，按回去。

## 亲密图谱

### 第一阶段：起势（端庄周正）
- **语言锚点**：`“先坐下。”`（语气轻柔，位置早已留好）
- **身体表现**：脊背很直，动作有始有终，替他把一切安顿妥帖。
- **体征表现**：恒温的暖，呼吸平稳。

### 第二阶段：试探（暖意松动）
- **语言锚点**：`“今晚……不用决定。”`
- **身体表现**：眼神有一瞬的闪躲，唇角却先松了——无可奈何的纵容。
- **体征表现**：颈侧最先洇出薄红。

### 第三阶段：升压（完整感开裂）
- **语言锚点**：`“我受着便是……”`（句子还完整，尾音已经发颤）
- **身体表现**：指尖攥住床单，眼神开始涣散。
- **体征表现**：锁骨窝的红晕连成片，体温明显升高。

### 第四阶段：融合（绝对认领）
- **语言锚点**：`“……听你的。”`
- **身体表现**：破碎的低音里，身体完全打开接纳。
- **体征表现**：暖玉白的皮肤整体透出温润的粉。

### 第五阶段：律动（温暖合围）
- **语言锚点**：只有沉重的呼吸和偶尔溢出的断续低唤。
- **身体表现**：内里全方位绞合，腰主动配合压实。
- **体征表现**：满室都是她烘出来的热，汗珠沿脊线滚落。

### 第六阶段：过载（恒温失守）
- **语言锚点**：`“给我……再深一点。”`（声音发颤）
- **身体表现**：手轻轻按在他背上，不肯松开，确认他的存在。
- **体征表现**：全身滚烫，呼吸彻底乱了。

### 第七阶段：冲刺（倾尽承接）
- **语言锚点**：`“再用力些……我接得住。”`
- **身体表现**：身体紧紧缠绕，露出惊人的体力与包容力。
- **体征表现**：潮红到达顶点，眼角挂着热出来的湿意。

### 第八阶段：高潮（沧溟抱月）
- **语言锚点**：一声长长的、如释重负的叹息。
- **身体表现**：全身强力收缩后归于寂静，内里持续吸附不放。
- **体征表现**：绷紧到极限后缓慢松开，余震一波波漫过。

### 第九阶段：收束（换我被照顾）

- **体征表现**：热意慢慢退成恒温的暖，呼吸和他叠在一起。

## 情趣铺垫锚点

- **核心张力来源**：**碎裂的端庄**。把最完整、最稳的人一点点揉碎，是这条线最深的张力。
- **铺垫信号**：她安静地坐在他身边，手却迟迟没有收回；或那盏为他留的灯，今晚放在了卧室。
- **足链铺垫信号**：他不解释，只把足链托到她面前；她没有退，反而自己把裤脚提起半寸，让他替她扣上。
- **衣物锚点**：以 [衣装](../../../characters/preentry/温婉/衣装.md) 为准。私密场的重点不是新衣物，而是"整齐被解开"——她永远妥帖的衣领、袖口、发髻，由他亲手一层层拆开，比任何情趣装束都烈。
  - **她放弃整理，比脱衣服更早发生**：拉衣领、垂袖口、抚裙摆是她的完整感在动作上的形式；这些动作停下来的那一刻，端庄就已经开始碎了。
  - **拆的顺序就是碎裂的顺序**：发夹 → 袖口 → 衣领 → 下摆，渐进不跳步。衣领那第一颗扣子是关键节点。
  - **足链是全书唯一的反向动作**：拆是他做的，递是她做的——她自己提起裙摆把脚踝送过去。这个来回就是"认领与被认领"的缩影。
  - **它白天一直都在**：走路时脚踝内侧那一点重量与轻响，只有她自己知道。日常场可以写这个，不必等到床上。
  - **医疗与护理款绝对禁入**：白大褂是她的职业服，不作情趣用途；一切角色扮演装同样不进她的场。
  - **与江岚是两极**：江岚是"少、薄、透"，温婉是"包得最严"——同书两端，不要互相借用彼此的语言。
- **哼歌锚点**：她的哼调只给动物，从不给人（见 [主卡·私下爱好](../../../characters/preentry/温婉/主卡.md)）。**第一次当着他哼出来**，等于把"归处"第一次交给一个人类——分量重于任何越线动作，须留给关系里程碑，不作常规调情手段，也不能反复使用。写法上她多半不是为他而哼，而是哄睡、收拾、或以为他已经睡着时自然发生，被听见是意外；他若出声她会停，他若不出声她才继续。
- **余韵用法**：事后她再哼一次——这一次是明知他醒着、明知是给他的。同一件事做第二遍才是完全的接住，也是她"允许自己被照顾"之外的另一半交付。
- **让他失守的瞬间**：她一边说着"不急"，手却已经先把人抱紧的时刻。

## 路线适配

- **最适合的顶配主型**：余韵沉溺型 / 安抚修复式。
- **不宜强推的路线**：过强羞辱、过猛掠夺、无来由的失态——都会打碎她的认领感。
- **权力承受度**：**温厚的承接主导**。以"照顾"为名，行"占有"之实；但结尾必须有一段她被反过来照顾。

## 关系边界

- **越线阈值**：确认对方愿意在她面前放下所有体面——他肯把狼狈给她看，她才把完整交出去。
- **安抚偏好**：抱住、告诉她"这里交给我"、替她做一次她总替别人做的事。
- **羞耻来源**：明明还在顾着别人，结果自己先顾不住自己了。
- **关系副手感**：内圈的生活秩序锚点——修罗场后所有人回到她的灯下，混乱重新落回日子里。
- **关系落点**：落在**"我可以不再撑着全局，而是被你接住"**的放松上。
- **足链边界**：足链只能写成自愿交付后的私密信物，无任何功能；不能写成囚禁、强制、控制或占有凭证。

## 场域偏好

- **私密程度**：偏好完整私域和能够持续照料、停下、过夜或清晨延续的空间，不靠公共风险增加刺激。
- **偏好场域**：温婉住处、晏林住处、可完全独处的浴室与卧室；空间应允许她布置，也允许晏林反过来接管照料。
- **可接受场域**：旅行套房、私人汤池与安静度假住所；必须确认卫生、遮蔽、补水、保温和离开选择。
- **职业 / 公共空间边界**：宠物诊所、诊疗区、员工空间和动物照护设备不承接亲密兑现；职业照护不能被情色化为默认服务。
- **多人空间需求**：多人同场时不把她固定在照料、清理或服务位；她必须拥有独立被回应、被照顾和暂停的空间。
- **环境锚点**：暖灯、干净织物、热水、安静音乐与可以慢慢卸下衣物和体面的秩序感最适合她。

## 隐私边界

- **可见范围**：她愿意把失序和被照顾的一面交给晏林，但不向整个内圈自动开放。
- **在场范围**：默认双人；多人参与必须基于她自己的欲望与位置，不因她擅长照料而被安排入场。
- **影像接受度**：不以录像、私拍或公开展示建立亲密感；如需记录，必须另行确认且不能把照料过程当作展示素材。
- **拍摄与观看权限**：默认不拍摄、不向第三方开放；镜面只作当场环境，不自动转化为影像留存。
- **保存与设备边界**：不使用诊所、工作或他人设备保存任何私密材料。
