- **性格曲线**：压火试探 -> 撒欢点火 -> 肆意燃烧 -> 解脱式交付 -> 暖融筑巢
- **亲密主轴**：自由狂欢下的肆意燃烧——她不是被驯服的野火，而是终于遇到敢陪她烧、也接得住她的人。交付是解脱，不是认输；下一轮她还会再抢。
- **禁写提醒**：不写成宠物化的幼兽萌点、被按住的战利品或死气沉沉的实验；拒绝纯压制惩罚。她即便在交付时，也是带着解脱感主动扑上去的；她的主动扑近来自清醒选择，不来自被拴住后的依附。

## 形象引用

- **形象锚点**：红发空气刘海、大而圆的灵动猫眼、小脸柔 V、轻量爆发感（详见 [形象](../../../characters/preentry/唐雨薇/形象.md)）。
- **情动体征**：写实——冷白薄红的皮肤最藏不住火：运动或情动时，耳根、颈侧与胸口很快烧起大片樱红，体温烫得惊人，汗来得快也亮；高马尾散开后红发贴着汗湿的背，像火线终于落了地。眼尾一挑，就是温度上来了。

## 专属锚点

> 针对"压火小野火、破势前锋"位格的深度补强。

### 1. 撒欢式挑衅
- **核心逻辑**：把床当成撒欢的场子——故意挑衅、故意抢高位、故意用那句"就这？"点火，然后在被反制时大笑着交付，享受"终于有人敢接我这把火"的解脱感。
- **写法要点**：她被按住时眼睛要亮得更厉害——笑着认这一轮，下一轮再抢。不写成讨饶。

### 2. 对等的痕迹
- **核心逻辑**：她喜欢在他肩上留下牙印，也会主动要求他在自己身上留红痕——这不是归属的烙印，是"我们打成平手"的对等勋章，一人一枚，谁也不欠谁。
- **语言示例**："这个算你赢的。……这个算我的。"

### 3. 压火后的爆燃
- **核心逻辑**：她的火不是一直烧满的——被压久了会先沉默、先记账；真正的引爆点是被尊重的那一刻："可以帮，但别替我决定"被他真的做到了，她的火才彻底放开。
- **情感落点**：她要的从来不是被管住，而是被尊重之后自己选择扑进去。

### 外部
- **阴毛**：浓密，深红棕，与发色同源，质地带一点天然的卷。
- **轮廓形态**：**心型**。线条紧致且向上微提，透着练家子的利落感。
- **阴阜形态**：**丰满鼓起型**。紧致饱满，不松不坠，带攻击性的美感。
- **阴唇形态**：**明艳合拢型**。色泽明艳肉粉，小阴唇内藏，充分打开时方显湿红。
- **阴蒂特征**：**显露型**。极其敏感，触之即发——她的好奇心远胜羞耻心。
- **色泽与光感**：冷白底色配中间一抹明艳的红，情动后转油亮，伴随全身潮红。
- **皮肤质地与触感**：**紧致回弹**。练家子的强力弹性；常态温热，情动后烫得灼手。
- **分泌物特点**：起得极快、量极大。质地迅速转浓稠拉丝，量感易潮且伴随间歇的涌动。
- **轮廓与开合**：四阶段强反差。收缩表现为高频痉挛与失控的吸吮交替——身体永远比嘴诚实。
- **整体气质**：最具攻击性的美感。坦荡不遮掩，像一团刚点着的野火。

### 内部：[焰心变奏]
- **核心体感**：**灵动弹力与跳跃高温**。内壁随情绪节奏主动变奏——她笑时收紧、喘时放松，身体和情绪直连，完全不可预测。

### 五感签名

- **嗅觉变位**：干净的皂感混着运动后的热气 -> 汗与体温烧出来的辛甜 -> 满室滚烫的热浪
- **听觉变位**：欢快笑闹 -> 挑衅叫嚷 -> 节奏感极强的、笑与喘混在一起的破碎声

#### 写作口径
- **初入**: 雀跃的咬合感。内壁主动往前迎，带着微小的高频颤动，咬住了就不肯松。
- **过程**: 充满变奏的挤压。动作快时缠绕，放慢时磨蹭，热力像火星四溅。
- **余韵**: 身体带着运动后的兴奋与高温，脚趾会因为快乐而蜷起来。
- **触感**: 紧致有弹性，内里的收放跟着她的笑声变——她笑一下，他就知道。
- **心理**: 绝对的自由。所有压制感都被她转化为滚烫的快感——越是接得住她，她烧得越艳。
- **她的反应**: 红发飞扬，眼底闪着狡黠的光；会大声笑出来，或狠狠咬住他的肩。
- **节奏适配**: 灵动多变奏。忽快忽慢，用"乱"打"乱"——越猜不到下一下，她越兴奋。
- **高潮特征**: **变奏骤停后的总爆发**。高频雀跃中猛然收紧；顶点时体液裹着痉挛喷涌而出——她是七个人里唯一会一边大笑一边到达的。事后会兴奋地追问"刚才那是什么原理"。

## 亲密锚点

- **核心反应**：欢快的挑衅、灵动的撒欢、大气的交付。
- **敏感位点**：脚踝、耳朵、腹股沟内侧。
- **体质反馈**：全身泛起樱红，体温烫得灼手，红发汗湿贴背——火线落地的样子。
- **语言锚点**：语速快、动词多、反问多；顽皮时尾音上扬，做了决定干脆截断；被真正烧到时话变短、不再吵。
- **主动方式**：变着花样闹。主动骑乘、主动提议新玩法、拿出新鲜玩意儿要一起研究——开场那句永远是"我自己来"。
- **给予方式**：烧透了就全给。她的交付像火塌进炭堆——彻底、坦荡、不留余烬的算计。
- **退让方式**：被反制按住时眼睛亮得更厉害，笑着认这一轮——"行，这轮算你的"。
- **事后余韵**：
  - **物理收尾**：周身热浪十几分钟不散，四肢像八爪鱼一样缠在他身上，充满运动后的兴奋余温。
  - **心理收尾**：释放后的绝对自由感，一边盘算下次玩什么新花样，一边确认"这个人真的敢陪我疯"。
- **对白取样**：`“我自己来。”` / `“这个也要试试。”` / `“再来！再深一点！”` / `“行，这轮算你的。——下轮我抢回来。”`

- **HD 场景高光特写**：
  - **标志动作**：翻身跨坐上去、按住他肩膀的那下宣战。

## 亲密图谱

### 第一阶段：起势（挑衅宣战）
- **语言锚点**：`“就这？”`（眼尾挑起，宣战开始）
- **身体表现**：猛地扑过去，咬耳朵或抢占高位，眼神亮得惊人。
- **体征表现**：动作轻快，胸口先透出红。

### 第二阶段：试探（研究胡闹）
- **语言锚点**：`“让我试试这个角度……”`
- **身体表现**：满脸通红地研究他的身体，腿主动勾住他的腰。
- **体征表现**：体温显著上升，热气逼人。

### 第三阶段：升压（肆意合鸣）
- **语言锚点**：`“我要飞起来了！”`

- **体征表现**：全身泛起樱红，汗来得又快又亮。

### 第四阶段：融合（自由燃烧）

### 第五阶段：律动（焰心变奏）

### 第六阶段：过载（热浪失守）

## 情趣铺垫锚点

- **核心张力来源**：**坦荡的好奇心**。不遮掩、不怕，大大方方地研究一切新鲜的、被说成禁忌的东西。
- **铺垫信号**：她兴致勃勃地拿出新玩意儿研究，或认真琢磨新姿势时；以及被压了一整天的火在门关上那刻突然点着。
- **衣物锚点**：以 [衣装](../../../characters/preentry/唐雨薇/衣装.md) 为准。她可以吃一点新鲜的现代情趣感（半透长袜、腿环、温感小物，色调偏烈焰红、纯白或深黑），但必须由她自己先拿来研究、试穿、挑衅——"这个也要试试"是唯一的启用方式；拒绝写成被打扮的低姿态。
- **让他失守的瞬间**：她从"实验者"突然被实验反噬、笑着笑着瞬间瘫软的那一刻。

## 路线适配

- **最适合的顶配主型**：对等抢攻互换型 / 热闹共玩型 / 权力翻转型。
- **不宜强推的路线**：单向压制、惩罚驯服、沉重救赎——把她的火写灭了就全完了。
- **权力承受度**：**主导向彻底交托的转化**。交托是解脱，不是认输；每一轮的输赢都要是真的。

## 关系边界

- **越线阈值**：感觉到对方是真切在"贪图"她的热与闹，而不是敷衍或纵容；以及确认他不会替她决定。
- **安抚偏好**：拥抱、长久的温存、顺着头发慢慢摸——疯完之后她意外地黏人。
- **羞耻来源**：自己挑起来的火把自己烧坏了；假装淡定时身体先暴露。
- **关系副手感**：行动场的破势前锋——僵局她先撕口子，他在后面收拾局面；配合默契时一快一稳。
- **关系落点**：落在**"有人愿意陪我一起疯"的自由感与自己选的归属**上——不被关回笼子，而是把笼子门拆了以后，她自己走回来的。

## 场域偏好

- **私密程度**：偏好安全清楚但允许共同实验和真实输赢的私域；新奇来自她主动研究和提出，不来自公共危险。
- **偏好场域**：唐雨薇住处、晏林住处、旅行酒店与设备来源清楚的私人空间，适合实验箱、双遥控和控制权翻转。
- **可接受场域**：私人车库相邻休息空间与停稳车辆，只承接检查、谈话、蓄势和余温，完整兑现转入干净私域。
- **职业 / 公共空间边界**：车行工位、升降设备、维修工具、工业耗材和车辆固定设备一律不作身体场器具。
- **多人空间需求**：三人同场时要有她主动点火、修改规则和真正赢下回合的空间，不能只作苏璃设局后的执行者。
- **环境锚点**：可检查的器具、明确档位、快拆结构、充足照明和不受打扰的实验感最适合她。

## 隐私边界

- **可见范围**：她愿意让晏林看见自己的热、闹与失控，但不接受被当成猎奇实验对象或苏璃的附属。
- **在场范围**：唐雨薇—晏林双人、苏璃—唐雨薇女女双人和三人共同场分别授权；苏璃可以肯定并帮助两人获得独处机会，但不能替唐雨薇邀请、退出或分配身体权限。
- **影像接受度**：接受当次明确商定的少量私密记录，更偏好记录规则、输赢或一句约定，不默认保留完整过程；每次记录都需重新确认。
- **拍摄与观看权限**：默认参与者本人限定；苏璃和晏林之间的录音、影像或道具记录不因三人关系自动共享给她，反之亦然。
- **保存与设备边界**：不使用车行监控、维修设备、工作手机或联网器具保存私密材料。
- **删除与撤回权**：提出新玩法、抢控制器或赢下回合都不构成后续授权；她可随时暂停、改规则或删除记录。
- **身份保护要求**：保护车行职业身份、家族关系与个人实验偏好，不让私密内容成为家族或职业压力接口。
- **绝对禁区**：偷拍视频、工业用品身体化、解除安全限幅、用挑战输赢迫使继续，以及由他人替她决定。

## 玩具锚点

- **SM尺度**：轻度至中度（工具向）。
- **适合玩法**：皮质束带、温感道具、震动小物（先给她研究）。
- **玩法逻辑**：让她先"研究"道具，再反过来用在她身上——从实验者到被实验的反转是核心乐趣。
- **眼罩锚点**：可作为她主动提出的实验。她会先笑着说自己不怕；真被遮住视线后，呼吸和身体反而先乱。
- **禁忌**：拒绝纯压制或惩罚；所有道具必须经过她"这个也要试试"的主动确认——没有确认的支配是踩她的底线。
