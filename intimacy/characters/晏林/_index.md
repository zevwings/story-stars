# 晏林组亲密档案静态设定索引

> 本索引是晏林组亲密戏撰写的逻辑中枢与风格指南，统一采用都市星辰契约口径。

## 🏛️ 全局通用逻辑

- **写实口径**：亲密一律写实＋美感修辞，不用异能滤镜；体质、体征、反应全部落在真实身体上（`STYLE.md` 三基调）。
- **软支撑红线**：亲密 / 共鸣不回星力、不提控制精度、不直接兑现战斗输出、不提数值；只在情感 / 救赎维度起作用；唯一可养的是精神之海承载度（不计五境、幕后机制不进正文）；心神安稳只能是自然结果，不是目的（`BIBLE.md`）。
- **高自由度不等于同质化**：允许顶配强度，也允许更满、更深、更失控，但不能把所有角色都写成同一种臣服模板。
- **先认人，再定路线**：先判断"她是谁、这场关系落在哪"，再选主型与辅型；不要为了某种场面反过来掰弯人物。
- **状态锚点不是阶段表**：可跳读、可抽取、可回摆，不必从"抗拒"一路写到"顺从"。
- **晏林的底色必须在场**：他的主导建立在观察、确认、接住回应之上——确认开路，收束压场；亲近失态是瑕疵不是奖励。不写油滑，不写理所当然，不写对女性痛苦的观赏性迷恋。
- **知情、自愿、可撤回**：全组亲密戏的硬边界。

## 🕯️ 场景调度指南

### 分型调用说明

- **锚点型（安若曦 / 温婉 / 林芷兰）**：侧重"落点"——最后落在哪个确认上；她们的失守是渐进碎裂，不是瞬间崩塌。
- **拉扯型（苏璃 / 唐雨薇 / 苏晚乔）**：侧重"回合"——禁单一低位，写势均力敌后的让步；她们的输赢都要是真的。
- **失守型（江岚）**：侧重"变奏"——静里开闸，越安静场面越深。
- 通用：分型是倾向不是牢笼，可跳读、可回摆；同场多人时先定谁主场、谁承接、谁收束。

### 调用提醒

- 装束锚点与玩具锚点以各角色卡为准；静态衣装以各角色 `衣装.md` 为准，本索引不维护全员装束总表。
- 深色带（正向线暗色情趣）：口径见 `story/genres/深色情欲带.md`；档位与入口以各卡「深色带适配（速查）」节为准。全组不走 C 档羞辱向，疼痛至多 B 档「调味」级；与黑暗场的五条分界线任何一场都不可破。
- 撰写前同时读取本索引与 `intimacy/context/晏林/` 下对应动态卡（随正文进度建档）。

## 📝 写作约束

- **拒绝同质化**：不允许把所有人写成同一种臣服模板。
- **异能滤镜与显影**：兑现场不写发光、能量流转等异能滤镜；唯一例外是**事后段的共鸣显影**——可写双方精神之海缓缓交融（相引）或紧闭刺痛（互斥），只写感受、禁数值。美感仍靠感官、情感与节奏。
- **拒绝功利化**：任何"亲密带来好处"的因果都禁写；先有欲望与情感，才有亲密。（机制层的受限承载磨练是幕后设定，绝不在正文写成亲密获益。）
- **拒绝油滑男主**：晏林的底色是观察与接住；破功要狼狈，主导要先确认。
- **拒绝强行破防**：推拒来自羞耻、别扭或情趣拉扯；真实非自愿只属于黑暗场，不进本组正向线。
