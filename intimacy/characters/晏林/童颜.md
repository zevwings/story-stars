# 💎 童颜 · 不用换的要 · 直说直要

- **主角色卡**：[童颜](../../../characters/preentry/童颜/主卡.md)
- **形象档案**：[童颜｜形象](../../../characters/preentry/童颜/形象.md)
- **衣装档案**：[童颜｜衣装](../../../characters/preentry/童颜/衣装.md)
- **契约档案**：[童颜｜契约](../../../characters/preentry/童颜/契约.md)；记忆权柄不进亲密线，见「隐私边界·绝对禁区」
- **EMO代码**：NO_TRADE（要就只是要 / 不带下一句）
- **性格曲线**：被定价的顺从 -> 确认不用换 -> 直接说要，并且承受得住被拒绝
- **亲密主轴**：取消交换之后的索取——全组其他人在亲密里交出的是控制权，她交出的是**交换本身**。她所有的身体经验都带着账：父亲的赌债、场所的抽成、后来的住处、学费与保护。她的成人欲望第一次成立，不在于她终于敢要，而在于她说完"我想要"之后，**没有再补上那句"那我要做什么"**。
- **禁写提醒**：不写成以身体偿还救援、学费与住处；不因幼态称呼写成失去成年主体性与拒绝权；不把高性欲解释为创伤、被交易经历、报恩或父爱缺失（[主卡](../../../characters/preentry/童颜/主卡.md) 硬边界）。不与白玥共用耗材恐惧的写法（见「专属锚点·5」）。不写成被养大、被收编、被哄好的养成模板。

## 亲密称呼

> 童颜是全组唯一一个称呼本身构成结构的人：同一个词在她这里有两种身体。称呼总表见 [晏林组索引](_index.md)。

| 场合 | 称呼 | 说明 |
| --- | --- | --- |
| 人前 | 哥哥 | 公共场合的默认称呼，不含情趣义 |
| 私下·求安全 | 爸爸 | 起源是安全依恋（`PL-S-055`），非父女关系；晏林不认领父亲、监护人或所有者身份 |
| 私下·情趣 | 爸爸 / 主人 | 由她在清醒、非恐惧状态下主动重新定义并授权（`PL-I-025`）；只在深色带框架内成立，随时可收回 |
| 清醒表达欲望 | 晏林 | 她不依赖幼态称呼也能说出想要与停止，这是 `PL-I-025` 的完成判据之一 |

- **"爸爸"两种用法靠身体分，不靠语气分**：求安全时她是收着的——抓衣角、句子变短、音量降低、站在原地不动；情趣时她是打开的——语速快、会追问、会主动靠上来。同一个词，写反了整段就废。
- **"老公"永不出现**。这不是口味问题：老公是名分与保障，她一旦开始要名分，等于回到"用关系换一个不会被赶走的位置"，主轴当场作废。**"亲爱的"可以偶发**，只作打趣，不登记为固定称呼。
- 权力性称呼不改写日常关系位：叫过主人的第二天，她照样自己去上学、自己管钱、可以拒绝他。

## 形象引用

- **形象锚点**：脸显小、小骨架承接明显丰满的上身、进陌生空间先找门窗与出口、安全之后肩背舒展、笑意来得快而亮（详见 [形象](../../../characters/preentry/童颜/形象.md)）。
- **情动体征**：写实——她的红**不从脸开始**：颈侧与耳根先热，再一路铺过锁骨到胸口上缘。小骨架让呼吸的起伏格外明显，藏不住。真正的信号不是脸红，是**她终于不再去看门**。眼睛是她最大的情绪出口：防备时视线移动很快，真正想要时会长时间直视，不移开。

## 专属锚点

> 针对「被定价过的人第一次不带账地要」这一位格的深度补强。

### 1. 没有下一句

- **核心逻辑**：她的默认反应是"这个要还"。第一次身体经验就是被标价的，之后的住处、学费、房间、保护又全是别人给的。所以她在亲密里最锋利的一下不是主动，而是主动之后**那两秒空白**——她说完"我想要"，习惯性地要接上"那我要做什么"，然后自己听见了，停住。
- **写法要点**：破防写在"没有下一句"，不写成放下心结、不写成被治愈。她仍然清醒，只是账停在半空。
- **语言示例**：`“……那我要做什么？”` -> （自己停住）-> `“……不用做什么，是吧。”`

### 2. 拒绝不等于抛弃

- **核心逻辑**：她最大的软肋是把拒绝误读为抛弃（[主卡](../../../characters/preentry/童颜/主卡.md)）。`PL-I-025` 要求至少一次晏林明确拒绝，而她没有因此失去住所、教育、关系身份或任何支持。
- **这一条是全卡的地基**：没有验证过这件事，她的深色带一档都不能开——对一个把拒绝当遗弃的人，任何支配都不安全。验证过之后，她才第一次拥有"可以被拒绝的关系"，也才谈得上自愿服从。
- **绝对禁区**：**以拒绝、冷落、不理她作为惩罚手段**。这一项别人可以，她不行；用一次，整条线的地基就塌了。

### 3. 称呼是结构，不是昵称

- **核心逻辑**：她的称呼不是亲昵度刻度，是**她当下需要什么**的直接播报。写她的对白，先确定她此刻要的是安全、是谈心，还是想要——称呼跟着这个走，不跟着感情浓度走。
- **写法要点**：她自己也在学着区分这三件事（`PL-S-055` 的必须发生）。她会用错，用错之后会自己发现，这是可写的。

### 4. 没有墙的地方

- **核心逻辑**：她进任何空间都先找门窗和出口（[形象](../../../characters/preentry/童颜/形象.md)）。而独立之后，她开始主动带晏林去屋顶、树林、没有人的野外——从"必须知道怎么逃"到"敢在没有墙的地方待着"。
- **刺激来源写准**：她要的是**没有屋顶也没有人来抓她**，不是被人看见。这条线是她的自由证据，不是暴露癖；把它写成"想被撞见"，人物立刻错位。
- **他不能用能力保证安全**：星辰契约能力不得用于制造私密条件（[亲密场域调用规范](../../mechanisms/亲密场域调用规范.md) §六），幻境遮蔽在这里是禁手。每一次的安全都必须靠现实勘察换来——这正是这条线真实的张力来源。

### 5. 与白玥分清（同一条黑产层出来的两个人）

| | 白玥 | 童颜 |
| --- | --- | --- |
| 位置 | 主动逐利的经营者，有真实加害责任 | 被父亲以赌债送进去的被控制者 |
| 恐惧 | 失去利用价值后被切割 | 要得太多而被嫌、被赶走 |
| 最怕被看见 | 自己的反应**显得熟练** | 自己**要得太多** |
| 交付方式 | 收起全部工具、不留后手 | 直给，并且反复确认对方收到了 |
| 失控信号 | 语速变慢、尾音收紧 | **语速变快、话变多、一直问** |
| 会所与酒店 | 永久禁区，情色化即写回耗材位 | 会所永久禁区，**酒店必须被重写**（见「场域偏好」） |
| 被发现风险 | 绝不接受作为刺激来源 | **全组唯一接受**，且条件极严 |

两人的"怕被丢掉"不共用一套写法：白玥的孤是算过之后的主动选择，童颜的黏是本性不是创伤。

### 6. 与其余各线的分界

- **与苏晚乔**：晚乔是"我的热烈值得被郑重接住"，要的是**郑重**；童颜要的是**不必偿还**。两人都直接，落点完全不同。
- **与温婉**：温婉提供稳定空间并留位；童颜是先需要安全、随后反过来提供高密度情绪反馈的人（[形象](../../../characters/preentry/童颜/形象.md) 差异化定位）。
- **与许知夏 / 许知秋**：四居室的同住者。**同住不构成同场授权**（`PL-S-059`、[场域规范](../../mechanisms/亲密场域调用规范.md) §五）；住房福利不因她加入缩水，房租不构成任何人的义务。

## 私密构造

### 外部

- **阴毛**：自然浅淡，平时就会修剪得简单干净；关系稳定后，她会直接问晏林喜欢怎样，再按两人都舒服的方式细致整理。她确实想把自己准备得很好、给他最好的感受，但这份用心出自喜欢与主动给予，不是偿还，也不构成后续亲密的默认许可。
- **轮廓形态**：**小巧收合型**。整体偏小、线条干净，与她的小骨架同构。
- **阴阜形态**：**柔软微隆型**。有一点软肉的厚度，触感偏暖，不是紧绷的薄型。
- **阴唇形态**：**薄合型**。平时合得整齐，小阴唇几乎不外露；充血极快，一转色就完全显形。
- **阴蒂特征**：**半显型**。不深藏，反馈来得又快又直接，几乎没有延迟。
- **色泽与光感**：白底转粉极快，中间几乎没有过渡——和她的笑意一样，来得快而亮。
- **皮肤质地与触感**：**细软微暖**。常态温度偏高，皮肤薄，痕迹留得清楚也退得快。与白玥的薄凉迟温完全相反。
- **分泌物特点**：**起速极快、量足、持续**。几乎没有酝酿期，且不会中途干涩——这是她高需求的身体版本，不是被开发出来的。
- **轮廓与开合**：四阶段反差不大，但**恢复极快**。她的特异不在张力，在**可连续**。
- **整体气质**：直接、暖、门槛低——但**门是她自己开的**。低门槛不等于无条件，这一句必须同时在场。

### 内部：[骤暖连潮]

- **核心体感**：**起速极快与连续性**。不是高门槛，不是迟滞，是几乎没有前戏门槛，而且一次不够。

#### 写作口径

- **初入**: 温软湿润，几乎没有阻力，但入口偏小、包裹得紧——小骨架的物理版，紧不是抗拒，是尺寸。
- **过程**: **高频回应**。他做什么，她立刻给回什么，快到近乎坦白；她的身体不留时间差。
- **余韵**: 不退。热度停在原处，收缩仍在继续，她通常在余韵还没走完时就已经在要下一次。
- **触感**: 细软、偏暖、湿意充足；浅处反馈最密，深处是软而不散的贴合。
- **心理**: 被立刻回应的爽感，以及"她根本没在藏"的坦白感——不是被吞、不是被审视，是被**接得满满当当**。
- **她的反应**: 话多。她会一直说、一直问、一直要确认，这是她失控的方式，不是清醒的证据。
- **节奏适配**: 可快、可连、可短兵相接。她不需要长铺垫，但**每一次开始仍要重新问一句**——低门槛不等于默认同意。
- **高潮特征**: **连潮**。不是一次性塌陷，是一波接一波；中间她还会清醒地说"还要"，这一句比高潮本身更像她。

## 亲密锚点

- **核心反应**：**即时回声**。给什么她立刻还回来，而且说出来。
- **敏感位点**：后颈（她低头时露出的那截，早期是她的防备位）、腰侧最细处、胸口上缘（她习惯轻收肩去遮的地方）、内壁浅处（起速最快的一段）。
- **体质反馈**：常态体温偏高，手脚容易凉；皮肤薄，红痕清楚但退得快；夜里体温更高，抱上来是烫的。
- **语言锚点**：平时语速快、问题多；害怕时句子缩短、音量降低；**真正想要时反而不绕弯，直接说**。失控时不是失语，是**话更多**——一直说、一直问、一直要确认。（与白玥的"语速反常变慢"是全组最明确的一组反向对照。）
- **主动方式**：**敲门并说明来意**。她已经从抱着枕头进来，成长为敲门、等回应、说清楚自己为什么来（`PL-S-055`）。她的主动永远带一句来意声明，从不含糊其辞。
- **给予方式**：**反馈本身就是她的交付**。她把对方做的每一件事说出来、确认收到——这是她"记得别人随口提过的小事"的亲密版本。她给的不是身体，是**回声**。
- **准备方式**：她会坦白问晏林喜欢什么、什么会更舒服，也会自己学习不同姿势与配合方式，再把愿意尝试的逐项和他确认。她不只问他的偏好，也会直接说自己喜欢什么、不喜欢什么，试过以后继续给出反馈。她想把两个人的体验做到最好，重点是**主动了解、认真练习、现场继续问，也能明确拒绝**，不写成单向迎合或表现考核，更不能滑回“做得够好才不会被丢掉”的交换逻辑。
- **退让方式**：她会先说"没关系"，身体却停在原地不走。**"没关系"要被当成没答应处理**——这是全卡最重要的一枚写法钉子。
- **事后余韵**：
  - **物理收尾**：不整理、不遮，直接贴上来；肩背完全打开。最高级的信任信号是**她能背对着门睡着**。
  - **心理收尾**：她会问"你会不会觉得我太多了"。她怕的不是被看见欲望，是**要得太多而被嫌**。
- **对白取样**：`“我不是害怕，我这次是真的想要。”` / `“……那我要做什么？”`（自己停住）/ `“还要。”` / `“你会不会觉得我太多了？”` / `“你说过的，还算吗？”`
- **正文亲密偏好**：强度优先落在**感官与即时反馈**，其次是事后余波。不宜在前段堆心理博弈——她没有博弈，她的复杂全在事后那一句。

## 【HD 角色定调】

- **HD 色调归属**：暖白 / 浅紫 / 留灯的暖黄
- **底层羞耻逻辑**：不是怕被看见欲望，是**怕自己要得太多**——多到对方嫌烦、多到还不完、最后不要她了。
- **三阶定位**：第二阶变体（即时回声型）。
- **身体物理映射**：**入口小、起速快、包裹紧、可连续**——她的身体是她性格的直译：门槛低、反馈快、一次不够。
- **权力格局演化**：她从"整个体系里没有选择权"到"什么都能拒绝"。深色带里的服从是她**第一次自己挑出来交出去**的东西，与被父亲和场所剥夺的失控方向完全相反——这是她能玩到 B 档的唯一理由，也是她随时能收回的理由。
- **体征递进**：颈侧耳根先热 -> 红铺过锁骨到胸口 -> 呼吸乱、话变多 -> 不再看门 -> 事后背对门睡着。
- **写实溢出**：她话多，所以这场戏的声音先于身体铺满房间；事后房间里最明显的痕迹是她一路脱在地上没收的衣服，和她第二天早上自己捡起来的样子。
- **镜头重点**：她说完"我想要"之后，**没有再补下一句**的那两秒。
- **正文亲密调用重点**：写她如何直接说、直接要、直接给回声，而不是写她如何被引导。不宜写成被哄开、被养熟、被治好。
- **B 场调用重点**：兑现"低门槛不等于无条件"——她开得快，但每一次仍是她自己开的门；以及"连潮"的可持续性，让她在中间清醒地说出"还要"。
- **情趣引导**：她吃**明确的要与明确的留**。有效语言：具体而不带条件的索要、明确留人、明确说"你不用做什么"。禁忌：把她的经历当情趣素材、拿住处学费保护开任何形式的玩笑、"你欠我的"这类话——即使在情趣框架内也禁，它精准踩她的账本。
- **HD 场景高光特写**：
  - **标志动作**：手去够对方的手腕或衣角——她紧张时抓衣角，安心时抓人。
  - **神态钉子**：不再去看门的那一眼。
  - **余波反应**：话多之后忽然安静下来，然后问"太多了吗"。

## 亲密图谱

> 基准态：`PL-I-024` 之后的稳定关系。第一次交易场（`PL-I-023`）只作前史，不用本图谱。

### 第一阶段：起势（敲门与来意）
- **语言锚点**：`“我可以进来吗？”` 停一拍，`“……我不是来睡觉的。”`
- **身体表现**：敲门、等回应，进来之后先站着把话说完，不直接扑。
- **体征表现**：语速比平时更快，手在衣角上。

### 第二阶段：试探（先给再要）
- **语言锚点**：`“你上次说的那个……我记着呢。”`
- **身体表现**：主动靠近，转一圈让他看今天换的衣服，等一句回应才继续。
- **体征表现**：耳根先热起来，她自己没察觉。

### 第三阶段：升压（直说）
- **语言锚点**：`“我不是害怕，我这次是真的想要。”`
- **身体表现**：肩背完全打开，长时间直视不移开视线。
- **体征表现**：红从颈侧铺到锁骨，呼吸的起伏开始藏不住。

### 第四阶段：融合（低门槛的门）
- **语言锚点**：`“……嗯，可以。”`（她每一次仍会明确应一声）
- **身体表现**：几乎没有适应期，入口偏小但迎得完整。
- **体征表现**：湿意来得极快，体温整体升上去。

### 第五阶段：律动（回声）
- **语言锚点**：`“这样……对，这样。”` / `“你别停。”`
- **身体表现**：反馈即时，他做什么她立刻还回什么，腰主动跟上节奏。
- **体征表现**：话开始变多，句子变碎但没停。

### 第六阶段：过载（话更多）
- **语言锚点**：`“你在吗……你还在吧。”`（不是怕，是要确认）
- **身体表现**：手抓住他的手腕不放，视线始终锁在他脸上。
- **体征表现**：语速反常地更快，红潮爬满胸口上缘。

### 第七阶段：冲刺（不再看门）
- **语言锚点**：`“别走开——”`
- **身体表现**：整个人完全打开，第一次没有分神去确认门的位置。
- **体征表现**：全身绷到顶点，呼吸完全乱掉。

### 第八阶段：高潮（连潮）
- **语言锚点**：高声之后是密集的气音，中间清醒地插进一句 `“还要。”`
- **身体表现**：一波接一波，不是一次性塌陷；恢复极快。
- **体征表现**：皮肤整片发烫，红退得比来时慢。

### 第九阶段：收束（太多了吗）
- **语言锚点**：`“你会不会觉得我太多了？”`
- **身体表现**：不整理不遮，直接贴上来；最后能背对着门睡着。
- **体征表现**：体温缓慢回落，话忽然停了，呼吸落地。

## 情趣铺垫锚点

- **核心张力来源**：**她先说**。她不设铺垫也不设台阶，她直接讲——难的从来不是开口，是开口之后不补那句"那我要做什么"。
- **铺垫信号**：她换好衣服走到他面前转一圈问好不好看；或者她开始复述他很久以前随口说过的一句话；也可能把刚学到的姿势直接说出来，问他喜不喜欢，同时告诉他自己想不想试。
- **衣物锚点**：以 [衣装](../../../characters/preentry/童颜/衣装.md) 为准，直接对接该页「晏林身边 / 主动展示」一节——黑色或浅紫的合身上衣、高腰线、露出完整肩颈线，动作重点是"给他看"，不是等他评估她值不值得留下。
  - **展示是她的开关，不是讨好**：换装必须出自她自己的选择，且**不自动构成进一步亲密的同意**（衣装页硬边界）。
  - **校服与 JK 私服绝对不进亲密场**：校服只在真实校园场景中调用，不挪作私密装扮或关系情趣（衣装页硬边界）。这是她的显小面容与亲密场之间必须守住的隔离带，无例外。
  - **不为这场买成套的东西**：给她买一套等于又欠一笔。她自己挑的、平时也穿的，才不用还。
  - **与凝绯式的仪式展示分清**：她不需要被注视才能启动，注视对她是加分不是前提；她的展示是"我今天想给你看"，不是"你看着我我才敢动"。
- **让他失守的瞬间**：她清醒、直视、不绕弯地说出"我这次是真的想要"的那一下。
- **完整亲密前的节奏**：短。她不需要长铺垫，但**每一次仍要重新确认一句**；把低门槛写成默认许可，是这条线最容易犯的错。

## 路线适配

- **最适合的顶配主型**：直给索取型 / 高频承接型 / 明确留人型。
- **可接受但非默认的路线**：她主导——可以写，但那不是她的成长方向。她的成长方向是**承受拒绝**，不是取得控制。
- **不宜强推的路线**：救赎收编、养成系、宠物化、幼态撒娇模板、以拒绝作惩罚——前四项直接抹掉她的成年主体性，最后一项踩她的绝对禁区。
- **权力承受度**：**可收回的自愿服从**。她能到 B 档，前提是"拒绝不等于抛弃"已经被现实验证过；服从是她挑出来交出去的，不是她的默认位置。
- **多人同场动态**：默认不参与。四居室同住不构成同场授权，任何同场须逐次单独确认，不由居住关系推定。

## 关系边界

- **越线阈值**：三件事同时成立——她已具备住所、教育与现实离开能力（`PL-S-054`、`PL-S-059`）；第一次交易的账已经被清算过一次（`PL-I-024`）；她能在不损失任何现实支持的前提下说"不"（`PL-I-025`）。缺任何一条，她的靠近都还带着账。
- **安抚偏好**：明确的话、持续的接触、允许她待着别走。不需要哄——她要的是"你没有太多"，不是"乖"。
- **羞耻来源**：要得太多而被嫌；以及被当成小孩哄。后一项常被误当成她的萌点，其实是她的雷区。
- **关系副手感**：她给的是**高密度的情绪反馈**——她记得所有细节，会说出来，会确认。她的记忆权柄在亲密关系里**完全不出场**（`PL-I-025`），能力协助不是她提供的东西。
- **关系落点**：落在 **"我要，并且我不欠你"** 的确认上——不是被拯救，不是被养起来，是她自己要，且要完之后什么也不用还。

## 场域偏好

- **私密程度**：以完全私密为主；**全组唯一接受"可能被发现"作为刺激来源的人**，但成立条件极严（见下）。与白玥的"绝不接受被发现风险"构成明确反向。
- **偏好场域**：
  - **她自己的房间**（四居室，她拥有锁门、邀请、拒绝进入与以后搬离的权利，`PL-S-059`）——她这辈子第一扇自己开的门，是她的默认主场。
  - 晏林房间；共同居所内的其他空间须先确认同住者与隔音。
- **酒店：必须被重写的场**：`PL-I-024` 的接口明确以酒店交易场形成镜像。同一类空间，一次是被安排的交易，一次是她自己订、随时能走。这是她独有的场域逻辑——**别人的酒店是二级临时空间，她的酒店是一次改写**。
- **野外与无人开阔地**（屋顶、树林、无人的野外）：核心刺激是**没有屋顶、也没有人来抓她**，不是被人看见。硬前置全部成立才可开场：
  - 事前实地确认无人、无监控、无必经路径与作业时段；
  - 退出方案在位——能立刻穿好、有交通、有保暖、有清洁；
  - 不留任何痕迹，不占用他人物品与场地；
  - **一旦出现任何第三方，本场立即终止**，不换地继续；
  - **不得以幻境或任何星辰契约能力制造私密条件**（[场域规范](../../mechanisms/亲密场域调用规范.md) §六）——安全只能靠现实勘察换来。
  - 档位对齐 [场域规范](../../mechanisms/亲密场域调用规范.md) §二五级：只到基础档与暖色带，不做需要解除时间的束缚与衣装。
- **职业 / 公共空间边界**：**学校、校园周边与教育机构一律禁区**——她的复学身份不进亲密场，一次都不行。**会所、KTV、私宴与场外转接环节永久禁区**：那是她被定价的地方，情色化即等于把她写回被出售的位置（与白玥同禁，理由不同：白玥是案件现场，童颜是她被标价的现场）。管理局与案件相关空间同样不承接。
- **多人空间需求**：不适用（默认双人）。
- **环境锚点**：能锁的门、留着的一盏灯（她的核心意象）、看得见天。全黑对她无效——她需要看得见对方还在。

## 隐私边界

- **可见范围**：她的失守状态只对晏林开放。**会所经历、案件当事人身份与在读学生身份三者必须彻底隔开**，不允许任何人把三者放在一起看。
- **在场范围**：默认仅她与晏林。四居室同住者知情不等于同场许可。
- **影像接受度**：**不拍摄、不留存、不回看**。她的复学身份与案件当事人身份需要保护，本线不调用或补写任何既往私密影像史。
- **拍摄与观看权限**：本线不以记录建立专属性。
- **保存与设备边界**：不使用可能同步、备份或被调取的设备；与案件、证据、学校相关的设备绝对隔离于私密内容之外。
- **删除与撤回权**：她可在任何阶段停止、离开或要求清除。她的主动、她的高需求、她的过去，都不构成任何形式的永久授权推定。
- **身份保护要求**：保护她的现实身份、复学身份与案件当事人位置；不以关系公开或影像证明任何东西。
- **绝对禁区**：把她的过去当情趣素材或调侃；以住处、学费、保护或案件安排交换亲密（含玩笑形式）；任何"你欠我 / 你要还"的语言；**以拒绝、冷落或不理她作为惩罚**；**动用记忆权柄读取偏好、删除冲突或代替协商**（`PL-I-025`）。

## 玩具锚点

- **SM定义**：核心不是痛感，是**她第一次自己挑出来交出去的控制权**。对一个整个少年时代都没有选择权的人，"我选择听你的"和"我没得选"是方向完全相反的两件事——这条线全部的意义在这个方向差上。
- **SM尺度**：**B（中）**。较强束缚、明确的支配—服从框架（含称呼）、轻度疼痛作调味；不上羞辱向。
- **适合玩法**：称呼框架内的支配与服从、姿势指令、明确的许可与不许可、轻束缚、延迟与克制。
- **玩法逻辑**：每一项的前提都是同一句——**她随时可以收回，收回之后什么都不会失去**。她的服从之所以有分量，是因为她清楚地知道自己可以不服从。
- **禁忌**：**以拒绝、冷落、不理她作为惩罚手段**（她的绝对禁区，别人可以她不行）；羞辱降格；"你欠我 / 该还了"这类语言；窒息与意识风险；无法快拆的束缚；把幼态外形、学生身份或校服写进玩法。
- **扮演式强制**：**禁用**。她有真实被控制史，这一档是长期禁区，不因 B 档或相邻角色推出。

## 深色带适配（速查）

> 提炼自本卡「玩具锚点 / 情趣引导 / 权力承受度」，供 skill 取档；详细描写仍以上述各节为准。口径见 `story/genres/深色情欲带.md`。本书正向线不走 C 档羞辱向，疼痛至多 B 档「调味」级。

- **档位**：**B（中）**· 称呼框架内的支配与服从，可收回。
- **入口情趣**：她主动重定义的称呼（爸爸 / 主人）、姿势指令、明确的许可与不许可。束缚与延迟属于框架立稳之后的加项，不作入口。
- **内核 · 触发**：**自愿的服从是她的自由证明**——她吃"我选择听你的"；触发靠明确的指令搭配同样明确的可拒绝空间，两者必须同时在场。
- **禁忌**：以拒绝为惩罚 / 羞辱降格 / 欠债语言 / 幼态化与校服玩法 / 把她的过去当素材 / 扮演式强制。
- **特异点**：**她的深色带入口是一个词，不是一根绳子**。而全书对她最重的一句不是任何指令，是"那你自己待着吧"——对全员最轻，对她最狠。
- **探索广度**：**高**。她愿意试的品类多、换新频率高，也会主动提议换地方、换玩法。广度不改变强度上限——不因"什么都愿意试"推出可上羞辱或明显疼痛（口径页 §7）。
