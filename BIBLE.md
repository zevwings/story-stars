## 一、一句话定位与定调

- **定位**：完全架空的**星契都市异能后宫长篇**——现实质感 ＋ 架空专名。
- **书的定调（篇幅与权重配比）**：**亲密是主调 ＞ 日常为次 ＞ 修行最辅**；主调内部，**情欲与身体兑现（正向 HD）是主色带，也是全书最高频、最高优先的场型，密度按情节自然铺满、不设硬上限**。
  - **亲密（主）**：关系推进 / 欲望兑现 / 修罗场 / 协商成阵——主干与爽点主体。亲密不是单一浓度，而是一条由暖到深的色带谱（见「四、题材口径与尺度红线」）。
  - **日常（次）**：都市生活 / 职场 / 学院 / 疗养 / 关系日常——占篇幅大，是**亲密的主要蓄势场**，关系在日常里长根。
  - **修行（最辅）**：战斗 / 修炼最点缀，只为制造「危机 → 救援 → 关系动能」，占比最小；星契不提供肉搏强化，近身搏斗只属于现实物理层；精神系收敛，每次出手都要为关系服务。
  - 关键前提：**铺垫为兑现蓄势，不是兑现的门槛数量上限**——日常与危机把关系养扎实，是为了让高频兑现有根、不失重成 PWP；关系一旦立住，允许短铺垫、高频兑现，靠关系存量供能，不必每场重铺。
- **核心叙事承诺**：都市、星契、势力斗争与灾害，最终都服务于人物间的**亲密关系推进、欲望兑现、关系后果与共同秩序建立**。

## 四、题材口径与尺度红线

- **情欲主色 ＋ 色带谱系**：全书亲密不是单一浓度，而是一条由暖到深的连续色带——**暖色带**（甜宠 / 逗弄 / 暧昧拉扯 / 亲密前奏，是情欲的过渡与缓冲）、**主色带**（正向 HD / B 场身体兑现，全书最高频、写到底、有美感）、**深色带**（正向线暗色情欲：支配 / 臣服 / 蒙眼 / 绝对主导 / 暗色情趣，双方成立、清醒可拒绝——那是情趣不是压迫）、**黑暗场**（反派压迫，走双声部铁律）。甜宠从旧「主旋律」降为暖色过渡，情欲兑现是新主色。
- **正向线为主体**：A 场（高暧昧 / 软切黑 / 正文亲密）＋ B 场（正向 HD）；**B 场是主色带与兑现主体，追求更多、更连续、且有美感的性爱**。A 场软切黑保留为连续场里的呼吸留白与过场，不是偷懒默认。
- **黑暗场为常规色带（占比提升）**：黑暗 HD / 压迫场不再收敛为罕见特例，可作为常规张力色带之一按需使用；但仍**仅落在对敌 / 反派 / 黑暗星辰契约者**，走双声部铁律，不与正向线的男主 × 女主兑现平铺混写。
- **唯一硬底线：参与者年满 18 岁**（正邪场同等适用，不得借黑暗场规避）。
- **线 × 场**：
  - **正向主线（男主 × 女主，最高频主色）** → A / B 场 ＋ 深色情欲带。
  - **黑暗线（对敌 / 反派 / 黑暗星辰契约者）** → 黑暗场，走双声部铁律，仅对敌、不落男主对女主；多视角下默认走男主见证框、施暴方视角禁用（见 `STYLE.md`）。
  - **少数正向色带（服务世界 / 人物 / 主题，不与主色抢频率、不写成 PWP）**：①**阵营内女女亲密**（女主 × 女主，同守知情 / 自愿 / 可撤回；男主在场可限知，不在场靠多视角承载）；②**配角独立情欲**（非后宫女性之间或配角关系，现实成人关系质感）。
  - **明文禁区**：**女主 × 其他男性的情色兑现＝禁**，锁死后宫契约；女主与男主之外的男性只可有非情欲的现实关系与前史，不写身体兑现。
  - 上述非男主场是**多视角限知**的主要适用对象；主色默认仍贴身男主（见 `STYLE.md` 视角）。
  - **男主不对自己女主施真实非自愿压迫**——深色情欲带的支配与暗色只在双方成立、清醒可拒绝的情趣范围内，真实的选择权失衡只属于黑暗场。
- **亲密不功利化（软支撑四红线）**：亲密 / 共鸣**不恢复星辰之力、不涨战力、不提数值**；只在**情感 / 救赎**维度起作用（降星辰异变、稳心神）；**因果方向锁死**——先有欲望 / 情感 / 信任才有亲密，心安与星辰异变被压住是自然结果，不是为收益去索取性。（「亲密即战力」已废，改「亲密驱动」。）
- **双声部铁律（黑暗场专用）**：快感是施暴方的、账单是受害方的；文本不认同施暴方；受害方痛苦与施暴方快感同重量在场；必落后果，不用甜蜜余韵收尾；不把身体反应写成愿意、求生配合写成迎合。
- **亲密美感基调**：写实 ＋ 美感修辞、纯写实不用异能滤镜、张弛结合（详见 `STYLE.md`「亲密 / 高张力场景」）。

## 五、核心机制红线（浓缩，细节见 `worldbuilding/`）

- **异能来源**：真实星空——清洁能源时代、无光污染的清晰星空是世界常态；望星结契 → **一星一契**，结契须目视星空；唯一多契特例是极罕见的**双子星契**——双子星只作为整体回应同一人，一次回应两契同立，不构成人为集星通道（见 `worldbuilding/systems/星辰契约.md`）。**结契主动权在星辰一侧**：星表是事后登记与方向级情报工具，不构成结契通道，机缘不可工程化（见 `worldbuilding/systems/星辰契约.md`）。星契存在已久，**起源模糊化**：不设正典起源史（见 `worldbuilding/systems/时代背景.md`）。
- **星辰之力**：星契者通过星契通道调用的中性干净力量；有限的是精神之海承载与星辰控制，不设体内蓝条。力量无善恶，用法分善恶。
- **星辰异变 ＝ 同源异化**：力量自身的镜像代价，症状与契主能力同构、概率发生；**不设外部负面力量实体**。
- **星辰权柄**：星辰权柄回答一颗星能够支配什么、按什么核心规则起效。**绝大多数星辰只有一个权柄，极少数特殊星辰可能同时具有多个权柄**；多权柄是星辰自身的先天结构，不可由训练、药剂、设备、家族或外力追加；各权柄之间必须**相似（同域互补）或相反（镜像对偶）**，互不相干的权柄不可能同星（见 `worldbuilding/systems/星辰权柄.md`）。具体能力数量不封顶，但都必须由契约星辰实际持有的权柄起效并遵守共用边界。观测 / 梦境 / 记忆 / 欲望 / 截断 / 逆流 / 御物 / 预兆等只描述能力效果，不构成可训练的公共能力。
- **无综合修炼境界**：精神之海按承载深广分为泉境、湖境、川境、海境、浩瀚境五档；精神之海五境评级只描述承载状态，不代表综合战力，也不是修炼阶段，不存在仪式性的「破境」。实际强弱仍由精神之海当前状态、星辰控制力、星辰权柄、具体能力及连接条件共同决定；不以能力数量建立公开战力等级。
- **黑暗线**：精神侵蚀（消耗自身星辰之力向外施展——造梦 / 篡改记忆 / 幻境欺骗 / 放大欲望 / 扰动人格 / 污染精神之海 / 种印）、星辰之印（星契的黑暗镜像；种印与解印都要求施术者的**星辰权柄及对应具体能力**能够干涉精神连接、控制结构或人格边界）；精神连接一律双向、可被逆用（通道反侵入 / 反向种印）。
- **战斗体系**：**星契不提供肉搏强化，不存在强化肉体力量 / 速度的星辰**；三层语法（感知 / 意志 / 物理），御物可以隔空操纵既有非生命物体，但不能直接操纵活体、强化肉体或改变物质性质，造成的伤害仍与现实武器和现实肉体对抗一并按物理层结算；动作权限由精神之海、星辰控制、星辰权柄、具体能力与连接条件共同决定。肉体兴奋药剂只短时刺激身体机能并带来健康危害，不介入精神层战斗；星辰异变药剂只干预同源异化进程，不恢复或增幅星辰之力，也不赋予权柄。
- **星契代价**：星辰异变（同源异化）是唯一的星契代价轴，不另设其他代价类目。

## 六、核心人物与力量格局

- **人物结构**：晏林是唯一男主；女主为 7 人核心＋若干阶段性重要角色。许知夏、许知秋是非核心长期伴侣，进入最终关系协商，但不计入七位核心女主。七位核心女主按登场批次错开、精排戏份，不平均用力。人物入口见 [人物索引](characters/_index.md)。
- **男主独特性**：晏林是全书唯一的双子星契主，也是感知／辅助流、精神反制特化的关系与行动核心。官方只将他登记为普通单星梦境契约者，真实双子星结构与其余三项权柄仍被隐瞒。人物、职业与前史见 [晏林主卡](characters/major/晏林/主卡.md)，星契细则见 [晏林星契](characters/major/晏林/星契.md)。
- **关系红线**：晏林不用能力非法操控自己人的意识；所有能力协助与亲密关系均服从知情、自愿、可撤回。
- **社会力量结构**：星辰管理局是唯一单列的星契官方机构，负责登记、监管、协查、保密与危害处置；星辰家族是传统权贵类型；黑市是地下交易生态；黑暗契约者是个人或小团体的行为类型。普通城市政府与公共治理只作为社会背景。具体入口见 [势力索引](worldbuilding/factions/_index.md)。

## 七、世界边界

- **影射式架空 ＋ 现实质感**：真实层（都市生活、职场、医疗、娱乐、商业结构、阶层、情感逻辑——照现实写）；架空层的地理与行政格局强映射现实中国，城市与地区专名全部重命名，国家、机构、历史事件、人物、民族宗教与异能社会全部架空重建（见 `worldbuilding/regions/`）。
- **当前地域范围**：本书暂时只设计少量国内城市与场域，不展开全球、海外或跨国星契体系；后续剧情确需扩大范围时，再另开对应世界观文件。
- **隐秘世界**：普通人很少知道星辰之力；星契社会是嵌在普通都市内部的隐秘圈层，管理局负责事件外泄管控（见 `worldbuilding/society/`）。
- **架空纪年**：不用真实公元；整体为制度成熟的远未来质感，正文不标注具体年份数字（见 `worldbuilding/systems/时代背景.md`）。
- 架空专名是黑暗线（非法实验 / 精神污染 / 真实非自愿）的**安全容器**——越黑暗越需要它；黑暗戏尽量落在纯虚构地点，不落在可识别的具体真实地区。

## 八、结构与结局

- **后宫结构**：前期隐秘并行 → 中期关系暴露（修罗场）→ 后期协商成阵（长期伴侣共同协商边界、公开程度与危机责任）。前期按五簇运行：A 苏璃—唐语薇、B 苏晚乔、C 安若曦—江岚、D 温婉、E 许知夏—许知秋；阶段一至二以簇内推进、簇间互盲为主，阶段三启动第一批跨簇可见化，阶段四集中处理大范围暴露与共同责任，阶段五完成协商成阵。协商成阵不抹平五簇，不要求所有伴侣共同居住、共享职业或依附晏林；每个人保有独立生活、职业发展、拒绝权与重新协商权，核心与非核心关系也不因此被抹平。
- **修罗场收束口径**：修罗场控制总量，不把连续冷战、误会拉扯和反复拆散当主线。人物可以吵、闹、争、吃醋、要求回答或短暂停顿，但已成立关系的叙事承诺是**争而不散、闹而仍在场**；不以说不开、负气失联、长期离开或反复分手拖延推进。必要事实与意愿说清后，冲突可以继续落到床笫之间争夺、验证和重新排列位置，优先写成彼此仍选择一起的双人或多人亲密。亲密承接冲突但不抹账，不替代必要告知，也不取消任何人的拒绝、暂停、退出和重新协商权。
- **结局承诺**：男主与女主阵营夺回星契解释权，建立以**知情、自愿、可撤回**为基础的新秩序。
