# 童颜｜衣装

> 本页只负责童颜的场景化衣装、配饰、妆造与气息。身份、关系、年龄边界与写作禁区以 [主卡](主卡.md) 为准；面容、身形、肤色与气质调用见 [形象](形象.md)。

## 总体衣装口径

童颜的衣装变化服务于“被别人安排”到“能够自己选择”的成长，稳定结构是**场域间的黑白转换**：她一个人在学校、街上和公共环境里时，多用黑色、深藏青、炭灰收住存在感；进入与晏林相处的私人关系场域后，整套衣装转为白、浅粉、浅紫与浅蓝，主动显出青春、清甜和活力。深浅不是同一套衣服的外层与内层，也不要求把浅色藏在深色下面，而是她面对外界与面对晏林时两套不同的选择。

她有一条明确的场景分轨：在学校与公共环境里，她更愿意用深色外套、合身但不贴身的上衣收住身体存在感；与晏林相处时，她偏爱衬衣、短裙、针织睡裙等青春轻快、便于活动也便于随时进入亲密的衣装，把“愿意让他看见最好的一面”变成主动取悦与成人欲望的表达。她会认真记住晏林喜欢的颜色、轮廓和展示方式，也会因为成功取悦他而高兴；这份取悦必须出自她自己的选择，不是换取容身或偿还照顾，也不自动构成进一步亲密的同意。

衣装不把学生身份、伤痕、恐惧或讨好倾向转化为情色符号。所谓“藏住”只指降低第一眼的体量判断，不把丰满胸型写成可以凭衣料完全消失。

## 审美底色与场域转换

- **独自在外**：黑色、深藏青与炭灰构成学校、街道和公共场景的整体底色，负责收束、保护与降低注视。
- **晏林身边**：白、浅粉、浅紫、浅蓝构成约会、居家与亲密场景的整套衣装主体，负责显出她的青春、清甜、活力和主动取悦。
- **浅色分工**：白色最干净直接；浅粉偏甜和玩心；浅紫是她在晏林身边的默认亲密主色；浅蓝更松弛、清爽，适合约会与居家。
- **转换方式**：不靠脱掉一层衣服完成，而发生在出门前的主动选择、约会前的换装和回到晏林身边后的整体变亮。她没有换一套人格，只是不再需要用深色降低存在感。
- **材质**：哑光、中等厚度、垂坠但不过分贴身的布料；日常偏柔软棉料、卫衣、针织与校服用毛混纺。
- **禁用**：廉价亮片、漆皮、荧光色与堆叠荷叶边；不把 KTV 工作装伪装成青春衣装，也不让早期衣装超出她当时的经济与生活条件。

## 妆造与配饰

- **发型**：深色长发是常态。初期简单披直或低束，临时安置时允许自然散乱；复学后以低马尾、半束发和简单编发为主。
- **妆感**：初见时保留场所统一处理过的浅妆，妆面干净却缺少个人选择；离开后多为素颜或极淡底妆，复学状态只整理眉形与唇色。
- **主锚点**：深蓝或奶白的小发夹，款式可以更换，只负责收住碎发，不承担赠物或关系信物功能。
- **专属香氛 / 气息**：干净洗衣液、晒过的棉布与很淡的皂香；不常用昂贵香水。
- **配饰边界**：早期几乎没有贵重配饰；学校与日常场景保持简单实用。常驻物件不得未经剧情确认写成晏林赠物或旧关系证明。

## 场景调用

### 初见 / 受控外出

- 一条刚拆封的深藏青连衣裙，视觉上接近黑色。高圆领、短宽袖、轻微 A 字或不收胸的直身剪裁，裙长在膝上但不过分短。
- 面料哑光且有一定厚度，胸前不做褶皱、亮片或贴身分割；配合覆盖完整、具有承托和轻度收拢作用的内衣，只减少突兀感，不制造“不可能看出胸型”的效果。
- 需要进一步回避注视时，外搭一件薄黑针织开衫。新衣仍留着包装折痕，浅色平底鞋也略磨脚，说明这套衣服是临时安排，不代表她自己的审美。

### 临时安置 / 居家

- 初期以宽松卫衣、旧 T 恤、柔软长裤和棉质睡衣为主，部分衣物可以是临时添置的基础款，但不写成奢侈采购。
- 她先选择能包住肩背和胸前的深色上衣，真正获得安全后才会换成奶白、雾蓝等更轻的颜色。
- 居家松弛感来自赤足踩拖鞋、袖口盖住半只手和洗净后自然披散的头发，不靠暴露身体表达亲近。

### 复学 / 身份装

- 复学后进入实行制服制的学校。校服首先是她重新取得学生身份与稳定日程的现实标记；离开校园后是否继续穿着、是否发生关系转场，交给具体场景判断。
- **春秋正式制服**：深藏青西装外套、白衬衫、灰蓝格纹百褶裙或同色直筒长裤，配深色短袜与黑色乐福鞋。裙长适合正常校园活动，不以缩短裙长突出身体。
- **夏季制服**：白色短袖衬衫或针织 Polo 衫，配灰蓝格纹裙或深藏青长裤；面料透气但不透肤，胸前保留足够余量和胸省，活动时不绷扣。
- **冬季制服**：春秋制服外叠灰色或奶白针织背心、深色长裤与过膝羊毛大衣；寒冷天气增加围巾和保暖袜，不为保持轮廓牺牲保暖。
- **运动制服**：深藏青与灰白拼色运动外套、直筒运动长裤、纯色短袖训练衫和运动鞋。版型方便跑动、整理教室和参加集体活动，不临时收腰或改短。
- 衬衫需要胸省或少量弹性，外套按肩宽和胸围分别调整，避免胸前绷扣，也不以夸张大码把整个人吞没。
- 她在学校通常把外套或针织背心穿完整，颜色以深藏青、黑灰为主，肩背自然收住，不主动强调胸腰比例。这里的“藏”是降低第一眼注意力，让她能把精力留给课堂与同学关系。
- 校服在校园里首先承担复学、学习和稳定日程的现实功能；离开学校后的具体穿着、换装与亲密转场按当时关系、场域和人物主动性判断，不预设一条覆盖所有场景的绝对禁令。

### 日常 / JK 风私服

- 在生活稳定、能够自主购买后，她会尝试海军领上衣、常规长度百褶裙、针织开衫、牛仔裤与帆布鞋等年轻私服。
- 色彩以黑色、深藏青、浅紫、雾蓝和奶白为主；这类搭配借用清爽学院感，但不照搬她正在就读学校的完整制服。
- 早期她会记住别人夸过的颜色并反复穿，也会先问“这样是不是更好看”；成长后应逐渐出现不等评价、直接选择自己喜欢款式的变化。
- 与晏林约会时，她会有意选择白、浅粉、浅紫或浅蓝的合身衬衣、短款针织开衫、高腰百褶短裙和长筒袜，让整套颜色从在外时的深色转为明亮。青春活力、小骨架与成年曲线都由她主动展示给他。她知道这是取悦，也享受他的明确反应，不必把这种主动解释成讨好或偿还。

### 晏林身边 / 主动展示与快速亲密

- 她最常用的组合是白、浅粉、浅紫或浅蓝的合身衬衣、短款针织上衣配同一浅色谱或柔和灰蓝的高腰短裙。上身扣件能够迅速解开，贴身层便于褪开；短裙可以直接提到腰间，不必先完成一套漫长的脱衣流程。
- 她会有意识地选择方便随时开始、暂停和重新穿好的结构。这里的“方便”服务于她主动要、主动靠近的节奏，不构成默认许可；每次开始仍由当场表达与回应确认。
- 下装除高腰短裙外，也可以配浅灰蓝高腰直筒裤、浅色 A 字半身裙或垂坠长裙。高腰线负责接住丰满上身，让整体看起来轻巧、匀称，而不是只突出单一部位。
- 白、浅粉、浅紫与浅蓝放大她的清甜、活力和亲近感。她会在换好整套浅色衣服后主动走到晏林面前、转一圈或直接问他好不好看。动作重点是“我专门换成这样给你看”，不是解开深色外层，也不是等他评估她值不值得留下。
- 她并不把普通约会装与亲密战袍截然分开：同一件衬衣、短裙或针织上衣可以从逛街、约会自然过渡到亲密；区别在她何时主动展示、解开和说出自己想要。

### 亲密战袍 / 主动取悦

- 她明确拥有专门为亲密准备的战袍，统一采用白、浅粉、浅紫、浅蓝的青春制服感色谱：合身衬衣、短款针织、高腰百褶短裙、长筒袜与高承托贴身层可以整套保持浅色。它与她独自在外时的深色衣装形成场域反差，不需要再套一层黑色证明转换。
- 浅紫是默认主战袍，白色用于最干净直接的反差，浅粉用于她心情明亮、明显想逗晏林的时候，浅蓝用于约会回家或居家松弛的快速亲密。四种浅色可以换，稳定点是“到了晏林身边就亮起来”。
- 深色服从向可以作为少数特殊战袍，但不是日常转换的内层或默认副型。即使偶尔使用黑色或黑紫，也应保留她甜、亮、主动取悦的气质，不把她变成另一种成熟冷艳角色。
- 战袍追求的是青春、活力、取悦和快速进入亲密，不把她硬塑造成成熟艳女。剪裁必须真正适配小骨架与丰满胸型，允许充分承托和展示，不用错误尺码、绷扣或呼吸受限制造效果。
- 她会自己挑选、购买、试穿和搭配，也会记住晏林对颜色、袜型、领口和裙长的具体反应。稳定关系中，晏林可以赠送或共同挑选，但礼物不附带穿着义务。
- 穿好后主动走给他看、转一圈、问他是否喜欢，是战袍的一部分。她享受取悦他，也享受确认自己能够让他失守；这种满足不与住处、学费、保护或关系保障交换。
- 衣装结构优先支持快速打开与快速恢复：衬衣可以逐扣或直接解开关键扣位，内层可以向下褪开，短裙可以提到腰间；需要离开、有人接近或她改变主意时，也能立刻整理和穿回。
- 居家主型是白、浅粉、浅紫或浅蓝的贴身柔软针织睡裙、短款针织家居裙或细肩带睡裙，整套直接保持浅色。针织睡裙可以顺着身体上提，肩带与领口也便于移动；重点是她回到晏林身边后自然变亮、自然放松，又随时可能主动靠近，而不是先用深色包住再举行脱层仪式。
- 更深的服从向战袍只在关系地基成立后调用。装饰性系带可以先于真实限制结构，所有扣件与颈饰都须能由她自行解除；称呼和明确指令是入口，衣装负责放大她主动交出控制权的选择。

### 学习 / 图书馆

- 黑灰圆领针织衫、白衬衫、宽松直筒裤或常规长裙，外搭深藏青开衫；书包和帆布鞋以耐用为先。
- 衣装保持安静、耐坐和方便整理资料，不让“重新上学”只剩校服造型。

### 周末 / 出游

- 浅紫针织上衣、奶白短外套、牛仔裤或过膝裙，配帆布鞋和轻便斜挎包。天气热时可换棉质衬衫裙，冷天增加黑色短羽绒服或牛角扣大衣。
- 周末颜色比学校更轻，发型也可以从低马尾换成半束发或简单编发，表现她开始拥有不为生存服务的时间。

### 搬家 / 整理房间

- 洗旧的黑色或灰色 T 恤、宽松工装裤、运动鞋，长发扎高，袖口随手卷起。
- 这一场景不追求漂亮完成度，身体观感来自真实劳动中的动作与两人共同建立新生活，而不是临时换上精致造型。

### 正式场合

- 选择黑色、深藏青或灰紫色的简洁连衣裙、柔软短西装与直筒长裤。正式感来自纵向线条、合适肩线、稳定承托和哑光材质，不来自低领或堆叠珠饰。
- 剪裁需要容纳丰满胸型并保留腰线，但不强行把她塑造成成熟社交名媛；发型收整，配饰最多保留一件小耳钉或发夹。

### 危机 / 行动场

- 连帽卫衣或短外套配长裤、运动鞋，头发扎紧，衣袋能够放手机、证件与钥匙。
- 重点是快速移动、保暖和不妨碍动作；不临时出现紧身战衣、防护制服或与都市生活脱节的特殊服制。

## 衣装成长回路

1. **被安排**：衣服是场所替她选择的包装，新而不属于她。
2. **先藏起来**：她用深色、宽松与完整覆盖减少被注视，也减少“欠下照顾”的不安。
3. **恢复身份**：四季校服与稳定日常让衣装重新承担上学、出门和生活功能。
4. **主动选择**：她不再只重复别人夸过的样子，开始主动建立“在外深色、晏林身边浅色”的两套衣橱，自己决定今天想以哪一种浅色见他。
5. **主动展示**：她能区分“被公共目光打量”与“主动让喜欢的人看见”，约会前或回家后换成整套浅色衣装，成为她在晏林身边最稳定的取悦动作。

## 禁写误区

- 具体调用时根据她已经成年、当时场域、关系阶段和主动表达判断，不用一条固定服制规则替代场景判断。
- 不把伤痕、受控经历、讨好与顺从写成衣装魅力来源。
- 不用“宽松衣服完全看不出胸型”制造失真反转；只能降低视觉强调。
- 不让她永远只穿粉色、蕾丝和荷叶边，也不把清甜等同于儿童化。
- 不用奢侈品牌和赠物替代生活重建，更不让衣装承担偿还救助或证明关系的功能；这不妨碍她自己购买成套战袍，或在稳定关系中接受不附带义务的衣装礼物。
