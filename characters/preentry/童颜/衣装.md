# 童颜｜衣装

> 本页只负责童颜的场景化衣装、配饰、妆造与气息。身份、关系、年龄边界与写作禁区以 [主卡](主卡.md) 为准；面容、身形、肤色与气质调用见 [形象](形象.md)。

## 总体衣装口径

童颜的衣装变化服务于“被别人安排”到“能够自己选择”的成长。初期以黑色、深藏青和炭灰降低存在感，稳定后才逐渐加入浅紫、雾蓝与奶白；辨识度来自小骨架与剪裁之间的自然曲线，而不是单纯增加露肤。

她有一条明确的场景分轨：在学校与公共环境里，她更愿意用深色外套、合身但不贴身的上衣收住身体存在感；回到晏林身边，她会主动换成更显肩颈、腰线与胸腰比例的黑色或浅紫色衣装，把“愿意让他看见最好的一面”变成安全感和成人吸引的表达。这个变化必须出自她自己的选择，不是讨好换取容身，也不自动构成进一步亲密的同意。

衣装不把学生身份、伤痕、恐惧或讨好倾向转化为情色符号。所谓“藏住”只指降低第一眼的体量判断，不把丰满胸型写成可以凭衣料完全消失。

## 审美底色

- **主色**：黑色、深藏青与炭灰构成公共场景底色；浅紫、雾蓝与奶白构成安全关系中的提亮色。黑色负责收束轮廓，浅紫负责显出她清甜、柔软而已经成年的一面。
- **材质**：哑光、中等厚度、垂坠但不过分贴身的布料；日常偏柔软棉料、卫衣、针织与校服用毛混纺。
- **禁用**：廉价亮片、漆皮、荧光色、过薄贴身面料、极端短裙与堆叠荷叶边；不把 KTV 工作装仿成校服或 JK，也不让早期衣装超出她当时的经济与生活条件。

## 妆造与配饰

- **发型**：深色长发是常态。初期简单披直或低束，临时安置时允许自然散乱；复学后以低马尾、半束发和简单编发为主。
- **妆感**：初见时保留场所统一处理过的浅妆，妆面干净却缺少个人选择；离开后多为素颜或极淡底妆，复学状态只整理眉形与唇色。
- **主锚点**：深蓝或奶白的小发夹，款式可以更换，只负责收住碎发，不承担赠物或关系信物功能。
- **专属香氛 / 气息**：干净洗衣液、晒过的棉布与很淡的皂香；不常用昂贵香水。
- **配饰边界**：早期几乎没有贵重配饰；学校与日常场景保持简单实用。常驻物件不得未经剧情确认写成晏林赠物或旧关系证明。

## 场景调用

### 初见 / 受控外出

- 一条刚拆封的深藏青连衣裙，视觉上接近黑色。高圆领、短宽袖、轻微 A 字或不收胸的直身剪裁，裙长在膝上但不过分短。
- 面料哑光且有一定厚度，胸前不做褶皱、亮片或贴身分割；配合覆盖完整、具有承托和轻度收拢作用的内衣，只减少突兀感，不制造“不可能看出胸型”的效果。
- 需要进一步回避注视时，外搭一件薄黑针织开衫。新衣仍留着包装折痕，浅色平底鞋也略磨脚，说明这套衣服是临时安排，不代表她自己的审美。

### 临时安置 / 居家

- 初期以宽松卫衣、旧 T 恤、柔软长裤和棉质睡衣为主，部分衣物可以是临时添置的基础款，但不写成奢侈采购。
- 她先选择能包住肩背和胸前的深色上衣，真正获得安全后才会换成奶白、雾蓝等更轻的颜色。
- 居家松弛感来自赤足踩拖鞋、袖口盖住半只手和洗净后自然披散的头发，不靠暴露身体表达亲近。

### 复学 / 身份装

- 新校服采用深藏青西装外套、白衬衫、灰蓝格纹百褶裙、深色袜与黑色乐福鞋；裙长符合正常校园活动尺度，冷天增加灰色或奶白针织背心、长裤与大衣。
- 衬衫需要胸省或少量弹性，外套按肩宽和胸围分别调整，避免胸前绷扣，也不以夸张大码把整个人吞没。
- 她在学校通常把外套或针织背心穿完整，颜色以深藏青、黑灰为主，肩背自然收住，不主动强调胸腰比例。这里的“藏”是降低第一眼注意力，让她能把精力留给课堂与同学关系。
- 校服代表她重新取得学生身份和稳定日程，只在真实校园场景中调用，不挪作私密装扮或关系情趣。

### 日常 / JK 风私服

- 在生活稳定、能够自主购买后，她会尝试海军领上衣、常规长度百褶裙、针织开衫、牛仔裤与帆布鞋等年轻私服。
- 色彩以黑色、深藏青、浅紫、雾蓝和奶白为主；这类搭配借用清爽学院感，但不照搬她正在就读学校的完整制服。
- 早期她会记住别人夸过的颜色并反复穿，也会先问“这样是不是更好看”；成长后应逐渐出现不等评价、直接选择自己喜欢款式的变化。

### 晏林身边 / 主动展示

- 她会主动挑选比学校穿搭更合身的黑色或浅紫色上衣：小方领、柔软针织、短款开衫或自然收腰剪裁，露出完整肩颈线并明确胸腰比例，但不靠深低领和尺寸过小制造挤压。
- 下装可以配高腰直筒裤、深色 A 字半身裙或垂坠长裙。高腰线负责接住丰满上身，让整体看起来轻巧、匀称，而不是只突出单一部位。
- 黑色呈现安静、成熟和轮廓感；浅紫色放大她的清甜与亲近感。她会在换好衣服后主动走到晏林面前、转一圈或直接问他好不好看，动作重点是“给他看”，不是等他评估她值不值得留下。
- 这一类型属于普通约会与私人日常衣装，不使用学校校徽、制式领结或完整校服部件。进入更高尺度的亲密衣装时，仍须另行建立私密补充项。

### 学习 / 图书馆

- 黑灰圆领针织衫、白衬衫、宽松直筒裤或常规长裙，外搭深藏青开衫；书包和帆布鞋以耐用为先。
- 衣装保持安静、耐坐和方便整理资料，不让“重新上学”只剩校服造型。

### 周末 / 出游

- 浅紫针织上衣、奶白短外套、牛仔裤或过膝裙，配帆布鞋和轻便斜挎包。天气热时可换棉质衬衫裙，冷天增加黑色短羽绒服或牛角扣大衣。
- 周末颜色比学校更轻，发型也可以从低马尾换成半束发或简单编发，表现她开始拥有不为生存服务的时间。

### 搬家 / 整理房间

- 洗旧的黑色或灰色 T 恤、宽松工装裤、运动鞋，长发扎高，袖口随手卷起。
- 这一场景不追求漂亮完成度，身体观感来自真实劳动中的动作与两人共同建立新生活，而不是临时换上精致造型。

### 正式场合

- 选择黑色、深藏青或灰紫色的简洁连衣裙、柔软短西装与直筒长裤。正式感来自纵向线条、合适肩线、稳定承托和哑光材质，不来自低领或堆叠珠饰。
- 剪裁需要容纳丰满胸型并保留腰线，但不强行把她塑造成成熟社交名媛；发型收整，配饰最多保留一件小耳钉或发夹。

### 危机 / 行动场

- 连帽卫衣或短外套配长裤、运动鞋，头发扎紧，衣袋能够放手机、证件与钥匙。
- 重点是快速移动、保暖和不妨碍动作；不临时出现紧身战衣、防护制服或与都市生活脱节的特殊服制。

## 衣装成长回路

1. **被安排**：衣服是场所替她选择的包装，新而不属于她。
2. **先藏起来**：她用深色、宽松与完整覆盖减少被注视，也减少“欠下照顾”的不安。
3. **恢复身份**：合身校服与稳定日常让衣装重新承担上学、出门和生活功能。
4. **主动选择**：她不再只重复别人夸过的样子，开始保留自己喜欢的黑色、浅紫、裙裤和发饰。
5. **主动展示**：她能区分“被公共目光打量”与“主动让喜欢的人看见”，在晏林身边展示身材不再等同于讨好或偿还。

## 禁写误区

- 不把校服、JK 风私服或显小面容组合成未成年情色暗示。
- 不把学校里的遮蔽与晏林身边的展示写成随时一键切换的表演；换装应有回家、约会或私人相处等合理场景。
- 不把伤痕、受控经历、讨好与顺从写成衣装魅力来源。
- 不用“宽松衣服完全看不出胸型”制造失真反转；只能降低视觉强调。
- 不让她永远只穿粉色、蕾丝和荷叶边，也不把清甜等同于儿童化。
- 不用奢侈品牌和成套赠物替代生活重建，更不让衣装承担偿还救助或证明关系的功能。
