# 精神侵蚀

> 精神侵蚀是契主经精神连接向他人施加的黑暗行为体系。星辰异变是同源异化，星辰之印是种印完成后的精神约束结构；其中未经知情同意的强种与控制性使用属于黑暗行为。

## 一、定义

精神侵蚀以入侵、污染或控制为目的，剥夺目标的知情、自愿与退出权。它不是独立的黑暗能量，星辰之力本身仍然中性。

## 二、建立条件

精神侵蚀必须以精神连接为通道。欲望放大、非法激励、精神抑制、精神循环、信息回响、响应钝化与关系疏离等干涉可以在锁定目标并形成相应连接后开始；进入梦境、污染精神之海与种印需要更深的精神通道。

- 施术者的星辰权柄必须覆盖对应支配范围，并已形成相应具体能力：造梦入侵需要能够干涉梦境，记忆篡改需要能够改写具体记忆，幻境欺骗需要能够覆盖或替换当下感知，欲望放大需要能够干涉欲望；非法激励、精神抑制、精神循环、信息回响、响应钝化与关系疏离分别需要能够干涉既有目标动力、已有精神反应强度、精神过程重复、既有信息复现、刺激响应阈值与关系响应；人格扰动需要能够改写人格结构，精神污染需要能够侵入并破坏精神之海，种印需要能够干涉精神连接、控制结构或人格边界。效果名称相同，不代表不同契主拥有相同权柄或相同成立条件。
- 精神侵蚀会使施术者暴露可被逆向利用的通道。
- 具体连接深度见《精神之海》和《战斗规则》。

## 三、类型

| 类型 | 作用 | 主要后果 |
| --- | --- | --- |
| 造梦入侵 | 进入、构筑或扰动梦境 | 感知、判断与自我边界受干涉 |
| 记忆篡改 | 遮蔽、删除、重组或植入具体记忆 | 经历记录与事实判断被改写 |
| 幻境欺骗 | 覆盖或替换目标清醒状态下的当下感知 | 对现实环境、对象与危险作出错误判断 |
| 欲望放大 | 强行放大已有欲望 | 欲望对判断与行为的支配上升 |
| 奖励诱导 | 改变特定刺激的奖励循环与满足阈值 | 形成依赖、戒断、耐受与刺激升级 |
| 精神循环 | 强迫已有精神活动反复启动 | 念头、选择或行为难以自主退出 |
| 信息回响 | 使已有精神信息脱离主动调用而重复浮现 | 记忆、声音、画面、梦境或情绪反复侵入当前意识 |
| 响应钝化 | 提高精神结构对特定刺激作出反应的阈值 | 情感、危险、奖赏、痛苦或关系响应迟缓衰减 |
| 关系疏离 | 削弱对他人、群体或关系的归属认定与信任响应 | 关系陌生化、信任瓦解与亲近排斥 |
| 人格扰动 | 改写人格结构、自我认知或价值判断 | 身份认同、判断依据或行为选择受扰动 |
| 精神污染 | 在深层侵入后污染并破坏目标的精神之海 | 内部结构持续失稳、扩散与崩损 |
| 种印 | 植入约束结构 | 形成星辰之印；未经同意的强种属于黑暗控制 |

具体能否使用，受星辰权柄、具体能力、精神之海、星辰控制与连接条件约束。

## 四、精神污染

精神污染必须由覆盖精神侵入与结构破坏的星辰权柄及对应具体能力起效。它不是向目标注入独立黑暗能量，而是侵入力量在目标精神之海内部造成的持续失序与结构损伤；其成立条件、扩散范围与停止方式见《星辰权柄》及《精神之海》。“掠夺星辰之力”不属于现行能力类型；催化星辰异变由专门药剂实现，不属于精神污染或星辰契约能力。

覆盖净化方向的星辰权柄可以识别、抑制、剥离并排出仍在活动的污染，覆盖修复方向的星辰权柄可以处理污染已经造成的结构损伤。净化不自动完成修复，修复也不能在污染源仍然存在时永久维持结果。

被剥离的污染不是可储存、转移或再次利用的独立物质；它脱离目标精神结构并失去原侵蚀通道与施术结构维持后，会崩散为无结构的力量余波并自然消退。净化者会因深层连接暴露在污染结构与原通道面前，可能承受精神过载或通道反向侵入，但不会自动吸收或替目标储存污染。

## 五、施术代价与反向风险

精神侵蚀消耗施术者自身的星辰之力向外施展。施术者还需要承担自身精神过载、通道暴露与星辰异变加深风险。精神侵蚀本身不会自动造成永久能力损失。

精神连接是双向通道，可能出现通道反侵入与反向种印。具体结算受双方精神边界、控制精度、连接状态与第三方干预影响，不设置固定公式。

## 六、边界

- 精神侵蚀是行为总类，不是星辰异变的另一名称。
- 不设「反噬」这一正式术语；连接被逆用一律用通道反侵入、反向种印等直述。
- 种印是建立星辰之印的行为，完成后的控制结构见《星辰之印》。
- 精神侵蚀不承载黑暗场的叙事立场与亲密专项规则。
- 黑暗 / 强制亲密不成立共鸣：经精神连接对精神之海的侵害走本体系（非共鸣），受害方本能闭海，任何一方都得不到共鸣式的承载磨练收益。
