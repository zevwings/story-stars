# 星辰控制

> 星辰控制是契主调用、塑形、维持、回收星辰之力的能力，不设正式等级。精神之海五境评级不评价星辰控制。

## 一、控制对象

星辰契约提供调用星辰之力与星辰权柄的资格和通道。星辰控制决定契主能否将进入精神之海的力量准确塑形成具体能力，并在完成后安全回收。

## 二、控制力维度

- **精度**：作用对象、范围与心理缺口是否准确；
- **复杂度**：能否同时维持多个目标、结构与连接；
- **稳定性**：受到干扰、情绪波动或反制时能否保持动作；
- **持续性**：能否维持效果并控制累积负荷；
- **回收力**：能否关闭通道、收回力量并重建边界。

这些维度通过行动表现，不对外设置统一分数或等级。

## 三、强弱关系

精神之海承载度决定“能承受多少”，星辰控制决定“能控制得多好”。承载度高而控制粗糙，容易造成误伤与失控；控制精确而承载有限，则适合小范围、短时、隐蔽的动作。

## 四、使用与训练

契主对权柄的日常使用、职业应用、协查、救援与实战，本身就是星辰控制的主要训练方式，不要求另设脱离实际用途的统一修炼流程。只要确实调用权柄并完成相应动作，就会反复经历：

```text
定锚 → 感应回应 → 建立入口 → 承载塑形 → 执行 → 回收 → 重建边界
```

使用与训练不是积累经验提升综合境界，也不能以五境评级代替控制水平。其成果表现为契主对自身星辰权柄的理解与塑形，以及精度、复杂度、稳定性、连接持续时间、回收力或方向专精的改善，不要求所有维度同步增长。反复使用也可以磨练精神之海的承载度与结构广度，但具体成长与过载边界由《精神之海》定义，不能把控制进步与承载扩张视为同一件事。

使用与训练可以发展同一权柄下的新能力表达，但不能开启、复制或拼接另一项权柄。深度与上限仍受星辰权柄、契主适配和精神结构限制（见《星辰权柄》）。

## 五、失控

控制失败可能导致动作偏移、通道暴露、结构崩解、精神过载或反向入侵。持续过度控制会提高星辰异变风险，但控制力下降本身不等于星辰异变。
