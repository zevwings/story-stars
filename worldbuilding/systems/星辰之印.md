# 星辰之印

> 本文是星辰之印的本质、建立条件、控制边界、反向种印与解印边界的正式世界观真源。种印所属的精神侵蚀行为体系见《精神侵蚀》；精神连接的通用规则见《精神之海》和《战斗规则》。

## 一、定义

星辰之印是通过种印建立的**强制、单向控制结构**。它是精神侵蚀行为的结果，不是可独立携带、交易或装备的实体器物。

星辰之印以持续占有、支配或干预目标为目的，剥夺目标对连接的知情、自愿与退出权。它因此构成星契的黑暗镜像，但不与人与星辰之间的原始星契混为同一种结构。

## 二、与星契的边界

| 星契 | 星辰之印 |
| --- | --- |
| 由星辰回应与契主承接建立 | 由施术者对目标强制种入 |
| 连接契主与真实星辰 | 连接施术者与被控制的目标 |
| 为契主提供调用自身契约星辰的通道 | 为施术者提供持续干预目标的控制通道 |
| 不以第三方占有契主为目的 | 以强制、占有与支配为目的 |

星辰之印不自动替换目标与其契约星辰之间的原始星契。

星辰之印不能劫持、夺取或强制解除目标的原始星契。种印可以干扰契主的当前状态与调用，但不改变星契归属。

## 三、建立条件

星辰之印由「种印」建立。种印属于高阶精神侵蚀，至少需要同时满足以下条件：

- 施术者已建立足以进入目标精神结构的通道级连接。
- 施术者的星辰权柄能够干涉精神连接、控制结构或人格边界，并已形成种印所需的具体能力（见《星辰权柄》）。
- 施术者具备足以干涉他人操控结构的精神之海承载与星辰控制能力。
- 施术者持续消耗自身星辰之力，将控制结构植入并稳定在目标精神体系中。

满足上述条件只代表施术者具备种印资格，不代表种印必然成功。目标的精神边界、控制稳定性、连接状态与第三方干预都可能影响结果。

星辰之印一旦固化，不具有固定时限，也不会随时间自然消退。它会作为控制结构烙印在目标的精神之海中，直到被有效解印。

## 四、控制边界

星辰之印可以为施术者建立持续干预目标的入口，使目标的自我边界、判断、行为或星契状态受到控制影响。

本页不将「受到控制影响」直接扩张为绝对服从、人格完全改写或无限距离操纵。

## 五、反向种印

种印必须通过精神连接进入目标。连接是双向通道，施术者植入控制结构的同时，也会向目标与可能干预的第三方暴露反向路径。

当这条通道被逆向利用，星辰之印可能反向植入原施术者，使原施术者反而成为被控制方。这一结果称为**反向种印**。

种印失败或通道被逆用时，施术者可以承受精神反冲、通道反侵入或反向种印。这些是口语中可称为「反噬」的具体后果，正式机制仍按对应结果直述。

反向种印不是「目标意志更强就必然发生」的自动机制。具体触发受双方精神边界、控制精度、通道状态与第三方干预影响，目前不设置固定公式。

## 六、解印

解印是识别星辰之印、进入其控制通道，并将控制结构从目标精神体系中切断或解除的操作。

截断可以暂时中止星辰之印继续传递控制与力量，逆流可以在控制作用仍沿原通道传递、尚未完成结算时尝试使其返回来源，但二者都不自动拆除已经固化的印。真正解印属于高强度的控制结构识别与拆除操作，必须由能够干涉并拆解对应控制结构的星辰权柄及具体能力完成；具备截断或逆流能力不等于能够解除星辰之印。是否能够成功解印，仍受双方星辰权柄的作用规则、种印者与解印者的控制水平、印的稳定程度、目标状态与当前连接条件影响。

解印失败可能造成精神反冲、原有控制通道加深、星辰之印进一步固化，或直接损伤目标已经开放的精神结构。最严重时，拆除操作可能破坏精神之海的核心锚点，引发结构性崩溃，使目标永久失去部分或全部星辰之力调用能力，并留下精神边界缺损、能力方向永久失能、残余控制入口，以及记忆或人格锚点损伤。

解印失败不会解除、转移或改变目标的原始星契。即使目标因精神之海崩溃而永久无法承载或调用星辰之力，契约归属仍然存在，对应星辰仍视为已有契主，直至契主死亡。

## 七、设定边界

- 星辰之印不具有外在形态，而是烙印在目标的精神之海中。
- 星辰之印不会自然消退，也不能劫持、夺取或强制解除原始星契。
- 解印失败可以毁伤原始星契的承载端，但不能毁伤或解除原始星契本身。
- 任何角色都不默认具有解除所有星辰之印的绝对权限；具体能力回到角色卡。

未经目标明确知情同意的种印、解印与控制性精神连接，均属于非法精神控制。经知情同意、登记授权且可随时撤回的临时精神连接，可以在合法协作或治疗范围内存在，但不得借授权扩大连接范围或改变其目的。

解印失败造成精神之海崩溃、星契调用永久失能或残余控制入口时，事件进入重大异常调查。管理局可以追查连接痕迹与施术者，但不得仅凭痕迹任意读取相关人员的全部记忆。
