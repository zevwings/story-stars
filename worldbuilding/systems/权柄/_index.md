# 权柄目录

本目录承载常见权柄方向的基础描述与相邻权柄辨析。体系总则、掌握规则与共用边界以[星辰权柄](../星辰权柄.md)为准。

| 专题 | 权柄方向 |
| --- | --- |
| [感知与信息](感知与信息.md) | 观测、记忆、镜像、静默、梦境、幻境、真言、寄识 |
| [意识与认知](意识与认知.md) | 篡忆、情绪、欲望、心念、催眠、行止、篡心、激励、抑制、循环、回响、钝化、疏离 |
| [连接与关系](连接与关系.md) | 传念、截断、逆流、共感、灵犀、同调、投射 |
| [侵蚀与修复](侵蚀与修复.md) | 侵蚀、污染、净化、修复 |
| [现实与趋势](现实与趋势.md) | 御物、预兆、脉络 |

每个权柄条目统一使用“描述、核心机制、常见表现、基础边界”四项。专题内的“辨析”只澄清相邻权柄的边界，不增加新的权柄或能力。
