# 势力索引

本目录存放星契社会中的官方机构与主要社会力量。只有星辰管理局作为统一组织单列；星辰家族、黑市与黑暗契约者均是社会类型或松散生态，不视为统一机构。

| 文档 | 层级 | 定位 |
| --- | --- | --- |
| [星辰管理局](星辰管理局.md) | 官方机构 | 负责星契登记、监管、调查与争议裁定 |
| [星辰家族](星辰家族.md) | 权贵类型 | 以血缘、星表与资源垄断为基础的传统家族群体 |
| [黑市](黑市.md) | 地下生态 | 非法交易形成的地下环境，不是统一组织 |
| [黑暗契约者](黑暗契约者.md) | 行为类型 | 从事黑暗行为的个人或小团体，不是统一势力 |

普通城市政府、公共安全和跨地区治理作为社会背景存在，不在本目录另设势力专题；具体制度仅在剧情需要时补充。
