# 全书五阶段大纲草案

> 本页是基于 `.analysis/plotlines/` 的全书阶段施工草案，不是正式正典。已确认事实仍以 `BIBLE.md`、`story/synopsis.md`、`characters/` 与 `worldbuilding/` 为准；本页只用于后续整理 `.analysis/plotlines/`、拆分正式 `story/arcs/` 和规划章节时参考。
>
> 当前最高主轴：亲密兑现如何被铺垫、推进、受阻、协商、后果化，并最终成阵。事件、势力、协查、黑暗线与日常场域都服务这个主轴。

## 总体规模

当前推荐采用五阶段结构。

这里的“阶段”是施工大段，不等于正式正文必须分卷。正式发布时可以连续章节推进；只有某个阶段实际达到约 30 章以上、且读者阅读节奏需要明显段落标识时，再考虑在正式 `story/arcs/` 或发布目录里拆成卷。若某段最终只有十几章，不必硬拆成一卷。

整体体量建议：

- **章节数**：160-180 章。
- **总字数**：约 80-110 万字。
- **常规章字数**：4000-5000 字。
- **亲密兑现 / 修罗场 / 阶段高潮章**：6000-8000 字。
- **低压日常过渡章**：3000-4000 字。

不建议压到 100 章以内。七位女主、亲密兑现、日常蓄势、彼此知情、修罗场和协商成阵都需要独立呼吸；压得过短会使后宫结构变成默认收编，削弱“知情、自愿、可撤回”的项目承诺。

## 亲密戏总量

整本书亲密戏先按增强版预留 **85-115 场**，其中身体兑现 B 场约 **42-60 场**。

这是施工预留，不是硬性配额。正式写作时仍按人物节奏、章节密度和读者疲劳感动态增减；若某条线需要更多日常蓄势或修罗场后果，可以减少重复性亲密场，不为了凑数强行插入。

这里的“亲密戏”包含高暧昧 / A 场、正向 B 场、多人协商型亲密和少量黑暗压迫场，不等于全部都是 B 场。

| 类型 | 建议数量 | 功能 |
| --- | ---: | --- |
| 高暧昧 / A 场 | 30-38 场 | 欲望升温、边界试探、关系临界、修罗场前后压强 |
| 单人正向 B 场 | 32-42 场 | 核心兑现、关系确认、复合、后果承担 |
| 多人 / 内圈 A 场 | 10-16 场 | 同处、边界、可停点、陪伴顺序和内圈生活磨合 |
| 多人 / 内圈 B 场 | 10-18 场 | 彼此知情后的身体层协商、共同生活和成阵证明 |
| 黑暗压迫场 | 2-4 场 | 只给反派 / 黑暗线，对照正向亲密，不铺开 |

节奏建议：

- 每 2 章左右允许有一次亲密推进，但低压日常、案件后果和修罗场对话不能被挤掉。
- 每 4-6 章应有一次强亲密兑现、修罗场后果或关系边界重置。
- B 场不要只是重复“发生关系”；每一场都应改变关系状态、知情状态、嫉妒 / 追问状态、危机参与权、留宿权、陪伴顺序或内圈边界。
- 多人 / 内圈亲密必须在彼此知情之后明显增多。否则“协商成阵”会停在口头规则，身体和日常层没有兑现。

## 女主彼此知情总节奏

### 第一层：苏璃与苏晚乔

第一阶段中后段，苏璃与晏林定情并发生关系后，苏晚乔必须很快发现并追问。

苏晚乔不是新出现的追求者，而是开篇前已经与晏林高频暧昧约一年。若晏林继续享受原有暧昧却不说明苏璃关系，会直接伤害“知情、自愿、可撤回”的人物底线。

这是第一轮小修罗场，范围集中在苏璃、苏晚乔和晏林。

### 第二层：安若曦与江岚

第一阶段后段至第二阶段前段，安若曦知道苏璃与晏林已定情；江岚知道安若曦与晏林之间不只是委托关系。

这一层的重点不是争风吃醋，而是：

- 晏林是否说明与她们选择有关的信息。
- 安若曦是否能保留决定权。
- 江岚是否能承认自己不是只在替安若曦把关。

### 第三层：唐语薇

第二阶段中后段至第三阶段前段，唐语薇知道苏璃与晏林已经形成秘密内圈，同时暴露自己的星契 / 唐家身份。

唐语薇受伤的重点不是朋友有秘密，而是她突然发现苏璃与晏林已经形成自己无法进入的秘密内圈。

这也是苏璃与唐语薇线真正的修罗场，不宜拖到后期，否则唐语薇会长期停在伏笔位。

### 第四层：温婉

第三阶段，温婉逐步知道关系复杂度，但她不应被默认接纳。

她的核心是慢热确认：在长期可靠、知情边界和复杂关系后果都看清后，主动要求自己的位置。她不是调停者、贤惠收尾者或默认大度的人。

### 第五层：林芷兰

第四阶段，林芷兰通过治疗知道晏林最深秘密。

这不是全员公开，而是单点结构级暴露。她知道最深，反而制造独有压力：她该如何在自己的登记清白、治疗伦理和替晏林遮掩之间选择。

## 修罗场启动节奏

修罗场不应等到后期才开始，也不应在第一阶段一次性爆完。

推荐三层启动：

1. **第一阶段中后段：轻修罗场启动**
   - 苏璃定情后，苏晚乔发现并追问。
   - 功能：证明未命名暧昧也有责任，不能用“没有正式确认”无限拖延。

2. **第二阶段：正式修罗场开始**
   - 安若曦、江岚、苏晚乔、苏璃都开始知道晏林正在与不止一个人建立真实关系。
   - 功能：修罗场从单点吃醋变成知情边界、退出权、暂停权和关系责任。

3. **第三阶段：多人协商期**
   - 唐语薇加入后，闺蜜、恋人、家族身份、秘密内圈和选择权叠在一起。
   - 功能：正式启动协商成阵线，每场三方或多人谈话只解决一个具体边界，不解决所有关系。

一句话原则：

**第一阶段有火星，第二阶段正式烧起来，第三阶段进入协商，不在第一阶段解决修罗场。**

## 五阶段结构

### 第一阶段：点燃与首兑现

建议规模：

- **章数**：32-36 章。
- **字数目标**：16-20 万字。
- **亲密戏**：10-14 场；多人 / 内圈 B 场为 0。

核心功能：

- 建立林语心理咨询工作室、酒吧、车行、训练馆、宠物诊所等现实日常场域。
- 通过魏琳咨询展示晏林公开登记的梦境权柄、专业伦理和知情授权边界。
- 酒吧事件把安若曦的职场压力、唐家外围和低层级精神干扰带入主线。
- 苏璃与晏林完成第一段定情和首次身体兑现。
- 苏晚乔前段强出场，展示开篇前已有一年暧昧与身体信任；苏璃定情后，她必须追问。
- 安若曦与晏林建立信息边界冲突，江岚以法律和程序边界接入。
- 温婉通过安安轻量出现，只完成前三阶段埋线，不进入完整宠物诊所异常事件。
- 唐语薇作为苏璃闺蜜、车行熟人和唐家伏笔存在，不解释完整家族结构。
- 阶段末安若曦与唐家外围追求者正式切割，管理局个案协查关系开启，唐家确认晏林不是普通心理咨询师。

亲密兑现重点：

- 苏璃是首位完整兑现对象。
- 苏晚乔不一定在第一阶段完成关系，但必须完成追问和暧昧责任清算。
- 安若曦 / 江岚只推进吸引、边界和动心，不急于身体兑现。
- 第一阶段可以增加高暧昧和单人 B 场，但多人亲密只保留同处、嫉妒和边界火星，不进入身体层。

禁止提前：

- 不公开双子星结构。
- 不让唐家全面开战。
- 不让林芷兰正面抢戏。
- 不让温婉完成告白或关系确认。
- 不让多人关系被一次三方谈话解决。

### 第二阶段：正式修罗场与协查常态

建议规模：

- **章数**：34-38 章。
- **字数目标**：18-22 万字。
- **亲密戏**：16-21 场；单人 B 场 7-9 场，多人 / 内圈 B 场 1-2 场。

核心功能：

- 苏晚乔在得知苏璃关系后进入连续协商，不能长期冷处理。
- 安若曦 / 江岚完成第一次三方说开：分别确认安若曦、江岚、晏林三条关系的边界，不直接进入稳定三人关系。
- 管理局协查常态化，工作室时间表、记录流程、助理知情边界和私人生活被挤压。
- 第二阶段精神污染案正式引出林芷兰：晏林能止损但治不了，林芷兰以净化 / 修复方向协查者进入。
- 黑暗契约者线从影子进入具体案件，但不急于星辰之印全面展开。
- 唐语薇开始从“苏璃闺蜜和车行伏笔”推进为独立关系，但仍不宜立刻身体兑现。

亲密兑现重点：

- 苏晚乔完成正式关系确认，并有至少一次强亲密兑现或关系后果场。
- 安若曦 / 江岚进入明确暧昧与有限协商期。
- 林芷兰以专业协作和花店日常扎根，不靠危机催熟。
- 多人 / 内圈亲密从 A 场开始：同处、陪伴顺序、可停点和告知边界先行，少量 B 场只解决已知情小组的具体关系后果。

修罗场功能：

- 从苏璃 / 苏晚乔扩展到安若曦 / 江岚。
- 重点是告知顺序、关系复杂度和暂停权，不是女主互相审批。

禁止提前：

- 不让林芷兰治疗即坦白提前发生。
- 不让唐语薇家族回收过早压过苏晚乔和安若曦 / 江岚。
- 不让管理局掌握双子星结构。

### 第三阶段：唐家合流与内圈协商

建议规模：

- **章数**：38-44 章。
- **字数目标**：22-26 万字。
- **亲密戏**：24-30 场；单人 B 场 10-12 场，多人 / 内圈 B 场 4-6 场。

核心功能：

- 唐家中期合流：唐语薇家族身份正式拉出，唐家试图重新解释她的名册、登记、婚配、产业或职业选择。
- 安若曦发现自己的职场压迫与唐语薇被追回的家族逻辑属于同一条唐家席位争夺链。
- 唐语薇与晏林通过独立事件建立关系，苏璃不能作为许可者，唐语薇也不能被姐妹送入关系。
- 温婉线从日常扎根进入宠物诊所异常事件，再进入“谁来决定她是否退出”的核心冲突。
- 协商成阵线正式启用：苏璃 / 唐语薇 / 晏林、安若曦 / 江岚 / 晏林、苏晚乔 / 苏璃 / 晏林等三方谈话各自解决具体边界。
- 周末日常、饮食、天气、旧关系回访开始承担公开程度和陪伴顺序压力。

亲密兑现重点：

- 唐语薇完成独立定情，身体兑现放在苏璃知情和三方谈话之后。
- 温婉完成从慢热确认到主动留下的关键转折，可在第三阶段后段完成关系确认。
- 多人协商型亲密明显增加，开始承担修罗场后果、嫉妒处理、陪伴顺序和三人边界；仍不写成所有人固定大团建。

修罗场功能：

- 这一卷是全书修罗场最复杂阶段。
- 重点不是谁赢，而是谁有权知道、谁有权暂停、谁不能被默认接纳。

禁止提前：

- 不让温婉变成调停工具。
- 不让唐语薇变成天命继承人或夺权女主。
- 不把协商成阵写成完整制度文本。

### 第四阶段：深层袒露与黑暗对照

建议规模：

- **章数**：32-38 章。
- **字数目标**：18-23 万字。
- **亲密戏**：22-28 场；单人 B 场 8-10 场，多人 / 内圈 B 场 4-6 场。

核心功能：

- 黑暗契约者线升级：精神污染、黑市接口、异常补剂、星辰之印或疑似种印进入主线。
- 管理局复评压力增强，晏林登记不完整问题从关系秘密变成制度风险。
- 林芷兰进入“治疗即坦白”：晏林受污染或损伤，截断只能止损，必须主动开放精神之海；她单点看见双子星结构。
- 林芷兰的压力不是“她知道秘密所以自动进入关系”，而是她在治疗伦理、登记清白、替他遮掩和自身修复冲动之间作出选择。
- 管理局直属疗养院、合作医院和工作室回访承担黑暗后果回收，不让黑暗场无痕结束。
- 内圈关系从私人协商推进为危机责任：谁被牵连、谁有权知道、谁能决定是否介入。

亲密兑现重点：

- 林芷兰完成深层袒露后的关系推进，场数不多但重量高。
- 已确认关系者出现危机后的复合、安抚、边界重置和共同责任亲密。
- 黑暗压迫场少量使用，只给反派 / 黑暗线，对照正向亲密。
- 多人 / 内圈亲密成为危机后回稳、共同责任和秘密知情重排的身体层兑现。

修罗场功能：

- 修罗场从“谁和谁在一起”升级为“谁知道到哪一层，谁被允许承担风险”。
- 秘密知情表必须持续维护，不能让所有人突然同步知道 L4-L6。

禁止提前：

- 不让净化 / 修复 / 截断越权解除星辰之印。
- 不让黑暗场压过正向亲密主调。
- 不让林芷兰成为关系稳定器或万能治疗背景板。

### 第五阶段：公开边界与终局成阵

建议规模：

- **章数**：24-30 章。
- **字数目标**：14-18 万字。
- **亲密戏**：16-22 场；单人 B 场 5-8 场，多人 / 内圈 B 场 3-5 场。

核心功能：

- 女主阵营从“替晏林遮掩”推进到“共同决定哪些事实必须公开、哪些边界必须守住”。
- 管理局、星辰家族、黑市和黑暗契约者围绕晏林能力、女主关系、星契解释权和新秩序发生终局压力。
- 协商成阵完成，但不写主母制度、不写审批制度、不写战力阵法。
- 每个人都必须拥有自己的位置、权利、边界、责任和退出权。
- 普通社会公开程度、星契圈公开程度和内圈公开程度分开处理，不一次性全社会摊牌。

亲密兑现重点：

- 终局亲密不是堆场数，而是确认共同秩序：谁留下、怎样留下、如何重新协商。
- 多人亲密应服务最终边界和情感秩序，不写成奖励清单。

终局承诺：

- 男主与女主阵营夺回星契解释权。
- 新秩序以知情、自愿、可撤回为基础。
- 亲密不是战力收益，而是关系、救赎和共同承担的结果。
