## 能力显影

- 阶段一只显影未登记的幻境，不使用或暴露截断、逆流、暗伴星与双子星结构。
- 晏林以小范围感官遮蔽保护当事人或制造撤离空档；安若曦或现场关系人从现实位置与感知结果不一致的细节察觉异常，管理局不因本次事件直接形成记录。
- 后续独立案件以协查记录和复盘材料之间的稳定差异形成幻境疑点，设备结果只作辅证，不承担能力定性。
- 唐家从晏林对一次精神控制事件的异常准确止损开始怀疑他“不普通”，但不能据此识别双子星或完整能力结构。

## 核心女主知情

- 终局反制齐心连接链时，晏林必须以亮星维持幻境遮蔽，同时沿齐心主动暴露的连接，以暗伴星执行截断或逆流；单星双权柄无法继续解释同时出现的承载与作用结果。
- 外部人员只看见无法完整解释的行动结果；七位核心女主因承担撤离、治疗、预警或现实保护责任而取得完整事实。
- 告知按风险相关性分波：先 C 簇安若曦、江岚，再 A 簇苏璃、唐雨薇，再单独告知苏晚乔，最后在温婉决定是否进入更深危机责任前完成告知。林芷兰已单点完整知情，只参与专业风险校准，不替晏林告知或审批。
- 各波可以紧邻发生，但必须分别保留提问、暂停、拒绝参与、继续与重新协商的空间。

### 最小应急表

| 项目 | 最小内容 |
| --- | --- |
| 异常信号 | 梦醒混淆、反射性截断、承载过载 |
| 判断入口 | 晏林自报；林芷兰只判断治疗与继续施为风险 |
| 暂停权限 | 晏林本人及任何受影响参与者均可提出暂停 |
| 行动边界 | 停止调用、撤离、现实隔离、最低必要求援 |
| 信息权限 | 七位核心女主完整结构；管理局仅梦境与幻境 |
| 事后处理 | 分别复盘，不共享治疗材料与双人隐私 |

## 管理局备案

- 普通任务摘要只记录“存在清醒侧感知覆盖结果”；高权限复评材料记录“幻境方向、覆盖范围受限、不能读取真实感官、不改变现实与记忆”。
- 复评异议由晏林对超出梦境与幻境的推定提出书面更正；紧急任务只披露当次任务所需的效果、成立条件和风险，不提交暗伴星、双子星、截断、逆流或四项权柄全貌。

## 终局解释权

- 反派把双星误判成同一精神通道上的多能力扩展，忽略两颗星、双份承载与镜像反制结构；其控制方案因此集中压制亮星，给暗伴星留下反制窗口。
- 新秩序以三类可见结果呈现：晏林亲自提交最小必要说明；对越权推定提出并留下异议记录；每次材料访问和权限扩张都可追踪、可复核。
- 不把唐家或裴家整体指定为解释权对手。具体星辰家族主体须先具备持续施压、资源化研究或制度游说的独立行动；若只在终局出现一次，作为 `PL-M-007` 场景参数，不新增节点。

## 分章前残余参数

1. 阶段一幻境遮蔽发生在哪一事件、由谁最先看出感知与现实位置不一致。
2. 形成管理局幻境疑点的后续独立案件与具体复盘材料。
3. 唐家开始怀疑晏林的精神控制事件及怀疑主体。
4. 终局双星同时调用的现场分工、外部可见结果与异议记录载体。

## 升格判断

- 上述参数若只决定场景写法、告知顺序或材料措辞，进入章节大纲，不新建正式节点。
- 只有出现现有四个节点未覆盖的独立持续变化与完成判据时，才考虑新增节点。
- 不扩大管理局知情上限，不把关系知情扩张为机构、家族或普通社会知情。
