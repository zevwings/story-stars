# 阶段四编排

> 本页是阶段四施工总控，不是正式正典。已确认事实仍以 `BIBLE.md`、`story/synopsis.md`、`characters/` 与 `worldbuilding/` 为准。本页确认后，再提炼稳定结论进入 `story/arcs/阶段四.md`；未确认章序不提前登记到 `chapters/_index.md`。

## 阶段定位

阶段四承担“江岚正式进入与深层秘密外压”功能。

阶段三已经完成安若曦和唐语薇正式进入，江岚留下亲吻越界但未兑现，双胞胎完成生活型暧昧前置，管理局只摸到黑市与唐家外围的重叠影子。阶段四要把悬置的关系和外部压力压到正面：江岚正式进入，双胞胎基本挑明，林芷兰治疗即坦白但不正式进入，黑暗线第一次证明旧方法不够。

核心推进：

1. 江岚承接阶段三亲吻越界，面对安若曦和自己的私人欲望，正式定情并首次身体兑现。
2. 双胞胎从装作看不见推进到基本挑明：她们知道工作室不普通，也知道自己不是普通员工关心。
3. 黑暗契约者线从精神污染升级到疑似星辰之印或持续控制结构。
4. 管理局复评压力增强，晏林登记不完整从关系秘密变成制度风险。
5. 林芷兰在治疗中单点看见晏林双子星结构，但不因此正式进入关系。
6. 管理局直属疗养院第一次正式承担黑暗后果回收。
7. 唐家线不全面开战，但强硬派确认唐语薇已经不是孤立对象。
8. 温婉继续慢热观察，可更明确动心，但不抢阶段四正式进入位。

## 阶段四主线

### 一、江岚正式进入

江岚阶段四必须承接阶段三亲吻越界，不能把那一吻当作误会擦掉。

推进要点：

- 她先面对安若曦，承认那一吻不是替闺蜜把关时的失控，而是自己的私人欲望。
- 安若曦可以介意、吃醋、要求暂停，但不能替江岚退出。
- 晏林不能把安若曦知情当成追求江岚的通行证；他必须分别回答两段关系。
- 江岚正式定情前，要确认自己不是借“保护若曦”进入这段关系。

首次 B 场关键词：

- 克制破口后的清醒选择。
- 不是职业压力下的失控。
- 不是安若曦背书。
- 不是用身体证明她没有判断错。

阶段终点：

- 江岚正式进入明确关系。
- 她从外部法律顾问和边界提醒者，推进为内圈边界执行者。

### 二、双胞胎基本挑明

双胞胎阶段四不正式进入关系，但必须从生活背景升为关系候选人。

她们挑明两件事：

1. **工作室不正常**：协查预约、特殊访客、临时取消、晏林疲惫和管理局来访，都已经超过普通咨询工作室的范围。
2. **她们不是普通员工关心**：留饭、守门、玩笑和吃味都已经越过单纯同事情分。

推荐场：

- 许知夏摊开异常日程、被改掉的预约、特殊访客记录和重复出现的保密要求。
- 许知秋用客服腔开玩笑，最后把玩笑放下：“我们可以装不知道，但你不能真把我们当不知道。”
- 晏林承认自己也不是没有感觉，但不能在直接雇佣关系和信息不对等下推进。

阶段边界：

- 双胞胎仍不掌握双子星结构。
- 她们可以知道星契异常存在、管理局协查存在、晏林有秘密。
- 她们不能无条件接触咨询隐私或管理局材料。
- 她们不能在阶段四亲吻或 B。
- 后续正式进入前，必须处理雇佣关系压力：转岗、扩编、合伙、外部行政托管或其他方式择一。
- 许知夏和许知秋是两个人，不写成同步接受或绑定套餐。

阶段终点：

- 双胞胎成为半知情工作室守门人。
- 感情基本挑明，三人共同暂停，留给阶段五处理正式关系与雇佣边界。

### 三、黑暗线升级

阶段四黑暗线要推进到“旧方法不够”。

事件建议：

- 出现第一名疑似星辰之印受害者，或一个表现为持续控制结构的精神侵蚀个案。
- 受害者存在持续指令、局部抵抗、关系响应异常或长期行为偏移。
- 晏林能截断持续输入、协助闭海和梦境复盘，但不能解除控制结构。
- 林芷兰能净化污染和修复结构损伤，但不能解印。
- 管理局能收容、复核、保全证据，但无法保证制度不被家族或黑市绕开。

黑市推进：

- 黑市明确有中介、药剂、目标介绍和观星资源交易链。
- 苏晚乔训练馆异常补剂线可在本阶段变成明确黑市接口。
- 唐家强硬派或外围人物可以与黑市接口发生交易，但不等同于唐家整体。

阶段边界：

- 不解印。
- 不推出统一黑暗组织首领。
- 不把黑市写成地下帝国。
- 不把唐家写成黑暗契约者总后台。
- 黑暗场少量使用，只给反派 / 黑暗线，并必须落后果。

阶段终点：

- 内圈确认：精神污染、星辰之印、黑市和管理局流程已经不是单人能处理的问题。
- 阶段五才进入解释权终局与公开边界。

### 四、管理局复评与疗养院回收

阶段四管理局线从个案协查升级为制度压力。

推进要点：

- 管理局要求晏林补充说明协查中多次超出普通梦境权柄的表现。
- 复评可以启动，但不能直接查穿双子星结构。
- 管理局直属疗养院第一次正式承担黑暗后果回收：收容、稳定、复核、转介、回流。
- 疗养院不是软禁场，也不是万能治疗院。

工作室后果：

- 工作室需要重分权限：什么由许知夏 / 许知秋处理，什么只由晏林处理，什么必须转管理局或合作医院。
- 双胞胎挑明后，晏林不能继续让她们站在门口替他挡人，却不给她们真正安全的位置。

阶段终点：

- 管理局怀疑晏林登记不完整，但仍无完整证据。
- 工作室从“普通咨询工作室 + 隐性协查点”推进为“有半知情守门人的风险接口”。

### 五、林芷兰治疗即坦白

林芷兰阶段四可以看见最深秘密，但不正式进入关系。

推荐触发：

- 晏林在对抗黑暗契约者时遭通道反侵入、精神污染或结构损伤。
- 截断只能止损，不能修复。
- 林芷兰治疗必须要求晏林主动开放精神之海。
- 晏林明知可以区域开放隐藏暗伴星，却选择完整开放。

治疗后果：

- 林芷兰单点看见双子星结构、川境与登记不完整事实。
- 她必须在治疗伦理、登记清白、替晏林遮掩和自身修复冲动之间判断。
- 她不能马上用“我喜欢你”解释隐瞒，也不能因为知道秘密就自动进入关系。
- 她可以说需要时间判断自己该怎么做。

阶段终点：

- 林芷兰成为知道最深秘密的人。
- 她不正式进入关系，保留为最后进入位。

### 六、温婉慢热观察

温婉阶段四可以比阶段三更清楚地看见晏林身边的危险和关系复杂度，但仍不抢阶段四正式进入位。

可写方向：

- 通过安安、宠物诊所或一次普通求助，温婉看见晏林不是单纯忙。
- 她不逼问，不默认接纳，也不替任何人调停。
- 她可以明确动心，甚至表达“我不是不知道，只是在等你们什么时候愿意说”。

阶段终点：

- 温婉进入确认前置。
- B 场建议后移，不与江岚兑现和林芷兰治疗即坦白挤在同一阶段。

### 七、黑市家族线正式落地（补入）

> 本小节为黑市家族线在本阶段的落地补入建议，尚未拍板；确认后并入上文「三、黑暗线升级」与推荐章序，并回应 [全书阶段总览](全书阶段总览.md) 中“黑市家族首个出场业务”的待确认项。黑市家族独立于唐家，详见 [黑市家族线](../plotlines/黑市家族线.md)。

阶段四是黑市家族从“影子”变“家族节点”的关键阶段：**首个家族出场人物 + 对外壳子一并落地**。

- 挂载「19. 训练馆补剂线」：异常补剂正式接到黑市家族的药剂流通端，露出**药物控制**链（更难离开、更难判断、更难说清），不只是单次买卖。
- 挂载「11–15. 黑暗结构露面」：结合星辰之印 / 持续控制个案，露出**人体实验影子**——分散、可追责的试验链，不写成大型实验室。
- **首个家族出场人物**（先作中介、药商或善后人之一）在本阶段现身，对外壳子（医药流通 / 器材供应 / 老字号）同时坐实。
- 首个出场业务需拍板：星表、药剂、实验还是清场；当前建议以**药剂 + 实验影子**切入，承接阶段二训练馆补剂伏笔。

阶段边界：

- 不写成地下王、不垄断全部黑市。
- 不与唐家合并；唐家强硬派若与黑市家族交易，也只是**外包关系**，不是同一势力。
- 暗杀外包、伪造洗白留到阶段五终局清算。
- 药物不增幅或恢复星辰之力。

## 推荐章序

> 以下是候选章节节点，不代表最终章号。实际写作时可按正文密度合并或拆分。目标规模暂按 32-38 章控制。

### 第一段：江岚兑现

1. **亲吻之后**
   - 功能：承接阶段三江岚亲吻越界。
   - 重点：不能当误会擦掉。

2. **先告诉若曦**
   - 功能：江岚面对安若曦，说清那一吻是自己的欲望。
   - 边界：安若曦可以介意，但不能替她退出。

3. **分别回答**
   - 功能：晏林分别回答安若曦与江岚，不用一句“都在意”抹平差异。

4. **江岚定情**
   - 功能：江岚正式确认关系。
   - 重点：她不是借安若曦背书进入。

5. **江岚首次兑现**
   - 功能：江岚首次 B 场。
   - 边界：具体场景后续进入亲密专项设计。

6. **亲密后的边界**
   - 功能：江岚确认身体关系不等于放弃程序和记录。

### 第二段：双胞胎不装了

7. **异常日程表**
   - 功能：许知夏摊开工作室异常节奏。
   - 边界：只谈流程事实，不读咨询隐私。

8. **客服腔失效**
   - 功能：许知秋用玩笑说破，最后不再玩笑。
   - 关键：`我们可以装不知道，但你不能真把我们当不知道。`

9. **不是普通员工关心**
   - 功能：双胞胎挑明感情不是单纯同事情分。
   - 边界：不正式确认关系，不亲吻，不 B。

10. **安全的位置**
   - 功能：三人讨论雇佣关系、知情边界和工作室权限。
   - 结果：双胞胎成为半知情守门人。

### 第三段：黑暗结构露面

11. **回流受害者**
   - 功能：疗养院或管理局回流一个精神污染后果更重的受害者。

12. **持续指令**
   - 功能：显影疑似星辰之印或持续控制结构。
   - 边界：不直接命名全部机制。

13. **截断不够**
   - 功能：晏林能止损，但不能解开控制结构。

14. **修复不够**
   - 功能：林芷兰能修复损伤，但不能解除印记。

15. **疗养院收容**
   - 功能：正式启用管理局直属疗养院作为黑暗后果回收端。

### 第四段：复评压力

16. **补充说明**
   - 功能：管理局要求晏林解释协查表现。

17. **登记疑点**
   - 功能：管理局怀疑登记不完整，但不掌握双子星结构。

18. **工作室重分权限**
   - 功能：双胞胎挑明后，工作室重新划分普通预约、协查接待、保密流程。

19. **训练馆补剂线**
   - 功能：苏晚乔训练馆异常补剂接入黑市。
   - 边界：药剂不恢复或增幅星辰之力。

20. **唐家强硬派退一轮**
   - 功能：唐家不全面开战，但确认唐语薇身边已有内圈支撑。

### 第五段：治疗即坦白

21. **通道反侵入**
   - 功能：晏林在黑暗线冲突中受创。
   - 边界：危机只负责袒露与看清，不直接兑换林芷兰关系。

22. **治疗授权**
   - 功能：林芷兰要求主动开放精神之海。

23. **完整开放**
   - 功能：晏林选择完整开放，而不是区域隐瞒。

24. **双子星单点暴露**
   - 功能：林芷兰看见双子星结构与登记不完整。

25. **我需要时间**
   - 功能：林芷兰不立刻进入关系，而是判断自己该如何面对治疗伦理和登记风险。
