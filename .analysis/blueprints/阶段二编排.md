# 阶段二编排

> 本页是阶段二施工总控，不是正式正典。已确认事实仍以 `BIBLE.md`、`story/synopsis.md`、`characters/` 与 `worldbuilding/` 为准。本页确认后，再提炼稳定结论进入 `story/arcs/阶段二.md`；未确认章序不提前登记到 `chapters/_index.md`。

## 阶段定位

阶段二承担“正式修罗场与协查常态”功能。

阶段一已经完成点燃：苏璃首兑现、苏晚乔追问、安若曦确认晏林有资格站在身边、温婉轻量埋线、管理局个案协查入口开启。阶段二要把这些结果变成持续压力，而不是让它们停在事件结尾。

核心推进：

1. 苏晚乔在追问后进入连续协商，完成正式关系确认，并有强亲密兑现或明确关系后果场。
2. 苏璃开始面对苏晚乔的存在，学习嫉妒、告知边界和关系后果，不写成默认大度。
3. 安若曦 / 江岚 / 晏林完成第一次三方说开，拆清选择权、告知权、暂停权和退出权。
4. 江岚从法律顾问和闺蜜位置，进入独立私下面与独立动心节点。
5. 管理局个案协查常态化，工作室时间表、记录流程、助理知情边界和私人生活被挤压。
6. 第二阶段精神污染案正式引出林芷兰；晏林止损，林芷兰净化与修复。
7. 黑暗契约者线从影子进入具体案件，但不急于星辰之印全面展开。
8. 唐语薇从车行伏笔推进到独立事件和独立吸引，但不立刻身体兑现。

## 阶段二主线

### 一、苏晚乔连续协商

阶段一追问之后，苏晚乔不能长期冷处理。

阶段二需要完成：

- 晏林承认：未命名暧昧同样产生责任，“没有正式确认”不能无限延迟选择。
- 苏晚乔知道苏璃关系已经成立，但她受伤重点不是输给苏璃，而是自己这一年是否被认真对待。
- 苏璃知道苏晚乔不是临时出现的人，而是晏林开篇前已经长期回应的关系。
- 三人不靠一次谈话解决后宫结构，只先解决“继续隐瞒是否允许”“原有暧昧能否照旧”“苏晚乔是否有权知道”。阶段二中后段允许这组修罗场进入三人 B 场，但该场只解决苏璃 / 苏晚乔 / 晏林三人的具体边界，不代表协商成阵完成。

阶段终点：

- 苏晚乔进入明确关系；若阶段一未 B，本阶段前段必须完成 B 场兑现。若阶段一已 B，本阶段至少安排一次高强度亲密后果场。
- 苏璃开始承认嫉妒和不安，不被写成立刻大度或默认主母。

### 二、安若曦 / 江岚 / 晏林三方说开

阶段二的安若曦线不是身体兑现，而是第一次把三人边界说清。

要拆的问题：

- 安若曦与晏林之间，哪些信息关系到她的风险和选择，必须告知。
- 江岚是否只是替安若曦把关，还是也拥有自己的判断和私人感受。
- 晏林是否会利用闺蜜分歧扩大自己的位置。
- 安若曦和江岚分别有没有暂停权、退出权和继续参与权。

阶段终点：

- 安若曦与晏林进入明确暧昧和有限协商。
- 江岚开始脱离“替安若曦审查晏林”的单一位置。
- 三人不进入稳定三人关系，不用一场谈话解决所有未来关系。

### 三、管理局协查常态化

阶段一卷末只是入口，阶段二要让协查开始改变日常。

可写压力：

- 工作室普通预约与协查委托冲突。
- 双胞胎察觉来访者类型、保密要求和晏林疲惫程度异常。
- 协查材料是否经过工作室行政流程产生边界问题。
- 江岚要求授权、记录、紧急处置范围更清楚。
- 管理局确认晏林梦境安抚和复盘有价值，但也开始积累能力边界疑点。

阶段边界：

- 管理局不是晏林官方靠山。
- 工作室不是管理局分部。
- 协查材料不能替代现实证据。
- 晏林仍只以登记梦境契约者身份参与，不公开幻境、截断、逆流和双子星结构。

### 四、林芷兰正式入场

推荐采用第二阶段新精神污染案，不复用安若曦案余波。

入场逻辑：

- 管理局发现精神污染受害者。
- 晏林能截断仍在输入的伤害、协助闭海、进行梦境安抚和复盘，但不能净化污染或修复结构损伤。
- 管理局按登记档案委托林芷兰入场。
- 林芷兰负责净化与修复；她的治疗必须有目标知情、自愿和主动开放，昏迷、强制压制或第三方代同意不能成立。

阶段终点：

- 晏林和林芷兰形成专业互认：他止损不越权，她治疗必先授权。
- 案件之外，晏林通过花店回访或治疗对象后续安排，看见她的生活侧。
- 不进入“治疗即坦白”；林芷兰不在阶段二看见完整双子星结构。

### 五、黑暗契约者具体案件

阶段二可以让黑暗契约者从影子变成具体施术者，但不建立统一组织。

可启用内容：

- 熟练精神侵蚀者。
- 黑市中介、药剂流通、目标介绍或观星资源线索。
- 管理局记录与污染结构分析。
- 苏晚乔训练馆异常补剂线作为黑市接口影子。

阶段边界：

- 不让星辰之印成为主战场。
- 不把黑市写成全知全能。
- 不把唐家写成黑暗契约者总后台。
- 每个具体施术者必须有自己的权柄、连接方式、成立条件和行为后果。

### 六、唐语薇独立事件

阶段二要让唐语薇从“苏璃闺蜜和车行伏笔”推进为独立关系萌芽。

可选事件方向：

- 客户索赔。
- 改装责任边界。
- 资深技师失误嫁祸。
- 商业投诉。
- 唐家外围人物进入车行试探，但不正式回收她。

关系原则：

- 事件必须发生在唐语薇自己的车行、技术、责任、学习资历或经营压力中。
- 晏林可以帮助她厘清风险和异常，但不能替她谈判、联系唐家或决定处理方式。
- 晏林出于保护欲做多一步后，唐语薇要校正他。
- 唐语薇真正动心的原因不是被救，而是晏林做错以后能够停下，并把主动权归还给她。

阶段终点：

- 唐语薇与晏林形成独立吸引和信任。
- 苏璃察觉两人变化，但不能把唐语薇送入关系，也不能默认自己已经接受。
- 唐语薇不接吻、不身体兑现；正式三方谈话可放在阶段二后段或阶段三前段。

### 七、黑市家族线显影（补入）

> 本小节为黑市家族线在本阶段的显影补入建议，尚未拍板；确认后并入上文主线与推荐章序。黑市家族独立于唐家，对外以医药流通商、星契器材供应商或地方老字号等壳子存在，详见 [黑市家族线](../plotlines/黑市家族线.md)。

阶段二只让黑市家族以**供应端影子**出现，不出家族正脸，不给家族名。

- 挂载「18. 训练馆异样」：异常补剂作为药剂流通接口影子，指向一个稳定但看不清的上游供货方，不止损于单个施术者。
- 挂载「30. 阶段二尾钩」：留下星表货源或登记残片的交易痕迹，暗示有人把“知道你是谁”做成生意。
- 对外壳子先行：读者只看到普通医药 / 器材 / 观星耗材供应商，不看到“黑市家族”自称。

阶段边界：

- 不出家族正脸、不给家族姓氏。
- 不把黑市家族并进黑暗契约者的单个施术者，也不并进唐家。
- 星表、药剂两条功能先行，人体实验、暗杀、洗白留到阶段四之后。

## 推荐章序

> 以下是候选章节节点，不代表最终章号。实际写作时可按正文密度合并或拆分。目标规模暂按 30-36 章控制。

### 第一段：追问后的关系后果

1. **追问之后**
   - 功能：承接阶段一苏晚乔追问，晏林不能用沉默拖延。
   - 重点：明确“没有正式确认”不能抹掉一年暧昧责任。

2. **苏璃知道苏晚乔**
   - 功能：苏璃第一次正面面对苏晚乔不是后来者。
   - 边界：苏璃可以嫉妒，不写成默认大度。

3. **不能照旧**
   - 功能：苏晚乔拒绝恢复原有含混日常。
   - 重点：约饭、送回家、训练后的暧昧动作都必须重新定义。

4. **第一次连续协商**
   - 功能：晏林、苏璃、苏晚乔先解决告知与暂停边界。
   - 边界：不解决全部多人关系，不形成完整内圈制度。

5. **苏晚乔关系确认**
   - 功能：苏晚乔正式进入明确关系。
   - 后续：若阶段一未 B，本章段默认承接首次 B 场；若阶段一已 B，本章段转为强亲密后果场。具体亲密场另行设计。

### 第二段：协查挤压日常

6. **个案委托**
   - 功能：管理局以阶段一事件后续为基础，正式委托晏林参与新个案。
   - 边界：个案委托和有偿协助，不给执法权或档案权限。

7. **预约冲突**
   - 功能：协查挤压普通咨询安排，双胞胎第一次感到工作室节奏异常。
   - 重点：普通员工不能无条件接触协查材料。

8. **江岚补记录**
   - 功能：江岚要求授权、记录和紧急处置边界制度化。
   - 关系：她开始与晏林形成独立工作默契。

9. **安若曦继续靠近**
   - 功能：安若曦在协查压力中确认自己仍愿意继续靠近晏林。
   - 边界：动心不等于交出判断权。

10. **三方说开**
   - 功能：安若曦 / 江岚 / 晏林第一次明确拆分关系和选择权。
   - 边界：不进入稳定三人关系。

### 第三段：林芷兰入场与污染案

11. **污染受害者**
   - 功能：第二阶段新案出现，污染结构超出普通短时精神干扰。
   - 边界：不是安若曦案余波，不复用第一阶段低级干扰。

12. **晏林只能止损**
   - 功能：晏林截断输入、协助闭海、梦境安抚，但治不了污染和损伤。
   - 重点：展示他能力边界，不靠主角万能解决。

13. **复原向协查者**
   - 功能：管理局委托林芷兰入场。
   - 重点：登记完整的林芷兰，与登记残缺的晏林形成制度对照。

14. **治疗授权**
   - 功能：林芷兰坚持知情、自愿和主动开放，拒绝昏迷或第三方代同意。
   - 边界：治疗授权不能被管理局命令、亲密关系或善意替代。

15. **止损与修复**
   - 功能：晏林与林芷兰完成第一次专业协作。
   - 结果：他止损不越权，她治疗必先授权。

16. **花店回访**
   - 功能：案件之外第一次进入林芷兰生活侧。
   - 重点：花店是现实经营，不异能化。

### 第四段：黑暗接口与生活回压

17. **污染结构**
   - 功能：管理局分析污染痕迹，确认施术者更熟练。
   - 边界：只定义本案需要的连接方式和后果，不批量命名黑暗体系。

18. **训练馆异样**
   - 功能：苏晚乔线接入异常补剂影子。
   - 边界：药剂不增幅星辰之力，不恢复星辰之力。

19. **温婉稳定出现**
   - 功能：宠物诊所 / 安安 / 低压生活承接高压案件后的换气。
   - 边界：不进入完整宠物诊所异常事件。

20. **苏璃的嫉妒**
   - 功能：苏璃面对苏晚乔已确认关系后的真实反应。
   - 重点：她可以不安、吃醋和占有，但不能用先来身份压人。
   - 触发：苏璃介意苏晚乔与晏林开篇前一年的身体信任、训练前史和直给方式；苏晚乔也不接受自己像“后来被通知的人”。

21. **苏璃 / 苏晚乔三人 B 场**
   - 功能：苏璃、苏晚乔和晏林第一次把嫉妒、占有、确认、可停点和告知边界放到身体层验证。
   - 类型：争风吃醋后的边界确认型，不是争宠表演，也不是补偿奖励。
   - 边界：苏璃不是主母，苏晚乔不是被允许加入的人，晏林不能用身体亲密逃避说明。
   - 事后：必须有清醒状态下的确认，明确陪伴顺序、以后谁有权知道什么、哪里可以停、下一次如何重新协商。
   - 后果：这场不代表协商成阵完成，只证明最早知情的苏璃 / 苏晚乔 / 晏林可以进入身体层共处。

### 第五段：唐语薇独立萌芽

22. **车行责任事件**
   - 功能：唐语薇自己的技术、责任或资历压力出现。
   - 边界：不由苏璃转交任务。

23. **晏林做多一步**
   - 功能：晏林出于保护欲越过一点边界。
   - 关键：唐语薇校正他：“我让你帮我看清楚，不是让你替我选。”
   - 亲密功能：动作可以停在擦灰、扶稳、递工具或短暂靠近；重点是他能在她说停时真的停。

24. **唐语薇自己处理**
   - 功能：她依靠车况敏感度、学习能力、经营判断和行动处理问题。
   - 结果：晏林承认越界并把决定权还给她。

25. **苏璃看见后退**
   - 功能：苏璃察觉唐语薇与晏林的变化。
   - 边界：苏璃不当许可者，唐语薇不被姐妹送入关系。

### 第六段：阶段二收束

26. **协查材料回流**
   - 功能：污染案材料、工作室记录和管理局复核产生能力边界疑点。
   - 边界：管理局仍不知道双子星结构。

27. **江岚私下面**
   - 功能：可安排周末漫展 / 私下兴趣场，让晏林看见江岚非职业面。
   - 边界：不替代法律线，也不让双胞胎传播或追问账号。
   - 亲密功能：给江岚一次非职业高暧昧，不靠安若曦背书。

28. **安若曦有限协商**
   - 功能：安若曦与晏林确认继续靠近，但保留判断权和暂停权。
   - 边界：不身体兑现；可以给身体临界场，必须明确主动允许、可停和撤回权。

29. **林芷兰留下联系方式**
   - 功能：她不成为固定搭档，但后续协查和花店回访有自然入口。
   - 边界：不因一次案件自动进入关系。

30. **阶段二尾钩**
   - 功能：黑市接口、异常补剂或污染施术者留下更深线索。
   - 边界：不进入星辰之印主战场；将唐语薇家族回收压力留给阶段三。
