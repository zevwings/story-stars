# 阶段一编排

> 本页是阶段一施工总控，不是正式正典。已确认事实仍以 `BIBLE.md`、`story/synopsis.md`、`characters/` 与 `worldbuilding/` 为准。本页确认后，再提炼稳定结论进入 `story/arcs/阶段一.md`；未确认章序不提前登记到 `chapters/_index.md`。

## 阶段定位

阶段一承担“点燃与首兑现”功能。

核心推进：

1. 建立林语心理咨询工作室、酒吧、车行、训练馆和宠物诊所等生活场域。
2. 通过魏琳咨询展示晏林公开登记的梦境权柄、专业伦理、授权和记录边界。
3. 通过酒吧事件把安若曦的职场压力、唐家外围和低层级精神干扰接入主线。
4. 苏璃与晏林完成第一段定情和身体兑现。
5. 苏晚乔作为开篇前已存在约一年的暧昧关系强出场，并在苏璃关系确认后追问。
6. 安若曦确认晏林有资格站在她身边，但不在阶段一身体兑现。
7. 温婉只做慢热生活埋线，不抢主案。
8. 卷末留下管理局个案协查入口和唐家关注升级。

## 四线并行深度

| 人物 | 阶段一功能 | 推进深度 | 阶段一终点 |
| --- | --- | --- | --- |
| 苏璃 | 首位关系兑现，确定本书亲密主调 | 定情 + B 场兑现 + 星契边界坦白 | 与晏林进入明确亲密关系 |
| 苏晚乔 | 直球追问，把既有暧昧逼成责任 | 关系确认 + 强亲密推进；默认阶段一末进入 B 场或等价强后果场 | 确认晏林不能继续含混处理她 |
| 安若曦 | 主案当事人，确认判断权与信任边界 | 明确暧昧 / 信任 / 告知边界，不身体兑现 | 确认晏林有资格站在她身边 |
| 温婉 | 慢热生活入口，补宠物诊所与城市日常 | 轻量出现 2-3 次，不告白、不确认 | 读者记住她与安安、诊所和温柔职业面 |

阶段一的关系原则：

- 苏璃和苏晚乔可以直给，而且阶段一必须让两人的亲密都显影；苏璃靠熟络暧昧与共同旅行，苏晚乔靠开篇前一年高频关系。
- 安若曦可以“确定”，但不“兑现”；她的爽点是判断权、告知边界和被尊重的选择。
- 温婉只扎根，不进入唐家主案，不承担调停者功能。

## 亲密节奏表

阶段一不能只有案件推进。亲密推进应每 3-4 个章节节点出现一次，每 5-6 个章节节点有一次强亲密、关系后果或边界重置。

| 区间 | 亲密功能 | 推荐强度 |
| --- | --- | --- |
| 3-5 | 苏璃在酒吧事件中主动保护安若曦，晏林看见她不只是暧昧对象 | A 场前置 / 欲望升温 |
| 6 | 苏晚乔训练馆前置，身体信任和一年暧昧先显影 | 高暧昧 / 专业触碰边界 |
| 9-10 | 苏璃确认主动入局，车行日常压出旅行前欲望和玩笑后的真心 | A 场 |
| 11-13 | 苏璃定情、首次 B 场、亲密后星契边界坦白 | B 场 + 关系确认 |
| 14-15 | 苏晚乔发现变化并追问，拒绝继续含混 | 修罗场后果 / 强 A 场 |
| 21 | 苏璃亲密后日常、苏晚乔追问后的暂停或继续、温婉低热生活触感 | 低压亲密 / 后果场 |
| 25-26 | 安若曦卷末确认晏林有资格站在身边，但不身体兑现 | 高张力边界场 |

阶段一最低亲密配置：

- 苏璃：1 次 B 场，1 次亲密后日常。
- 苏晚乔：默认 1 次 B 场；若节奏需要推迟到阶段二开头，阶段一末必须有等价强 A 场和明确关系后果。
- 安若曦：2-3 次高张力边界场，不 B。
- 温婉：1-2 次低热生活触感，不确认关系。

## 配角使用原则

配角只承担场域运转、证据链、生活气和世界观接口，不单独开传记式支线。

| 场域 | 配角功能位 | 阶段一用法 |
| --- | --- | --- |
| 工作室 | 合作医生 | 魏琳转介、评估材料、诊断边界；可只通过材料和电话出现 |
| 工作室 | 外部督导师 | 授权、记录、保密例外提醒；阶段一不独立成章 |
| 酒吧 | 店长 / 老板 | 支持苏璃调监控、留账单、应对投诉和商业压力 |
| 安若曦职场 | 直属上级 | 维稳、劝退、资源压力，不必写成反派 |
| 安若曦职场 | 项目同事 | 提供职场侧见证、消息提醒和证据补口 |
| 唐家外围 | 追求者 | 阶段一现实压迫核心，不能升格为家族核心继承人 |
| 唐家外围 | 低级契约者 | 短时精神干扰或证据混乱执行节点 |
| 训练馆 | 馆长 / 老学员 | 展示苏晚乔职业生活和晏林现实搏斗前置 |
| 宠物诊所 | 护士 / 常客 | 承接温婉慢热出场和安安回访 |
| 车行 | 资深技师 | 证明唐语薇仍在学习阶段，关键安全判断需复核 |

## 推荐章序

> 以下是候选章节节点，不代表最终章号。实际写作时可按正文密度合并或拆分。目标规模暂按 24-28 章控制。

### 第一段：专业入口与事件点燃

1. **魏琳咨询**
   - 功能：建立晏林的公开梦境权柄、咨询伦理、未成年人授权与梦境辅助边界。
   - 配角：合作医生通过转介材料出现；许知夏、许知秋承担工作室流程。
   - 结尾：咨询记录完成，工作室闭店。

2. **工作室闭店与酒吧转场**
   - 功能：从专业状态切到下班生活，展示双胞胎、预约、记录和工作室真实运转。
   - 配角：外部督导师可只通过记录备注或电话提醒出现。
   - 结尾：晏林前往苏璃工作的酒吧。

3. **酒吧里的异常客人**
   - 功能：苏璃正式入场，安若曦带着职场压力出现。
   - 配角：酒吧店长 / 老板承担监控、账单和员工支持。
   - 关键：苏璃先靠现实观察发现安若曦状态不对，不急着解释星契异常。

4. **拒绝失效**
   - 功能：唐家外围追求者把安若曦的拒绝重新解释成“仍可谈判”。
   - 配角：追求者、同伴、偷拍者或被安排的证人形成现实越界链。
   - 关键：偷拍视频、带离、证言布置或身体状态异常选择一组主事件。

5. **晏林介入**
   - 功能：晏林判断精神异常，协助安若曦安全离场。
   - 配角：低级契约者留下最低可追踪痕迹，不展开黑市组织。
   - 关键：晏林可低烈度使用未登记能力，但不暴露双子星真相。

### 第二段：前置关系与第一次边界

6. **训练馆日常**
   - 功能：苏晚乔前置登场，展示她与晏林开篇前已有一年高频暧昧。
   - 配角：馆长或老学员证明苏晚乔职业场域真实运转。
   - 关键：训练中的身体信任只按专业教学成立，不倒推恋爱关系。

7. **不完整的说明**
   - 功能：安若曦发现晏林隐瞒了与她风险相关的信息。
   - 关系：她不要求晏林交出全部秘密，只要求他不要把保留说成完整。
   - 核心句：`你可以保留自己的秘密，但不能替我删掉与我有关的风险。`

8. **安若曦职场压力回声**
   - 功能：补足酒吧事件不是孤立偶发，而是职场资源、项目评价和私人追求持续压缩。
   - 配角：直属上级维稳，项目同事提醒或提供证据缺口。
   - 关键：唐家外围压力落在现实制度里，不写成单纯纨绔纠缠。

9. **苏璃的选择**
   - 功能：苏璃确认自己不是旁观酒吧事件，而是主动入局者。
   - 关系：她与晏林既有暧昧被事件推到必须正视的位置。
   - 边界：苏璃不未经同意使用情绪权柄。

10. **车行日常**
   - 功能：唐语薇作为苏璃闺蜜、车行学徒型技师和晏林熟人出现。
   - 配角：资深技师复核关键安全判断。
   - 边界：只埋唐家身份和预感权柄，不抢安若曦主事件。

### 第三段：首兑现与轻修罗场启动

11. **旅行照旧**
   - 功能：苏璃与晏林按原计划出发，确认两人仍愿意面对原有暧昧。
   - 关键：旅行不是调查任务，不由酒吧事件或星契能力推动欲望。

12. **苏璃定情与首次兑现**
   - 功能：苏璃与晏林正式说出喜欢，完成第一段身体关系。
   - 边界：定情先于身体关系；清醒、自愿、可撤回；酒精与星契不推动亲密。
   - 后续：具体 B 场另行进入亲密设计流程，本页只保留剧情功能。

13. **秘密坦白**
   - 功能：苏璃坦白情绪权柄，晏林承认自己是星契者并说明仍有不能交代部分。
   - 边界：两人约定不得用星契能力影响彼此感情、欲望和亲密选择。

14. **苏晚乔发现变化**
   - 功能：苏晚乔从晏林状态、训练反应或约饭节奏中察觉他与苏璃关系已变。
   - 关系：她不是后来者，而是已有一年暧昧的人。
   - 关键：她可以生气和受伤，但不长期冷处理。

15. **苏晚乔追问**
   - 功能：第一轮轻修罗场启动；晏林必须回答既有暧昧的责任。
   - 边界：未知情、未协商时停下身体兑现；快速推进不跳过知情、自愿、可撤回。
   - 终点：苏晚乔进入明确关系协商，并默认推进到阶段一末的强 A 场或 B 场；若不 B，必须留下清楚的身体停点、继续意愿和关系后果。

16. **温婉与安安**
   - 功能：温婉轻量正面出现，建立宠物诊所、安安和慢热职业面。
   - 配角：护士、常客或宠物主人承担生活事件。
   - 边界：不进入完整宠物诊所异常事件，不告白、不确认关系。

### 第四段：证据、制度与卷末收束

17. **证据整理**
   - 功能：安若曦整理酒吧前后的现实证据。
   - 关键：把职场压迫、私人骚扰、星契干扰和证言污染分层。
   - 配角：项目同事、酒吧店长和监控记录回收前段信息。

18. **江岚入局**
   - 功能：江岚以闺蜜和法律顾问身份接入。
   - 冲突：她不替安若曦决定，但要求晏林把授权、记录和责任写清楚。

19. **工作室制度缺口**
   - 功能：暴露晏林过去靠个人自觉维持边界的风险。
   - 配角：双胞胎、外部督导师或合作医生提供现实流程压力。
   - 结果：工作室开始形成更清楚的记录、授权和紧急处置流程。

20. **真言盲区**
   - 功能：安若曦发现“不能说谎”不等于“说出的就是客观事实”。
   - 选择：认知错误、记忆干扰、回避陈述三者择一。
   - 边界：真言不能替代证据、调查和人的判断。

21. **低压生活换气**
   - 功能：用早餐、夜宵、训练馆或车行短场消化关系后果。
   - 可承接：苏璃亲密后的日常变化、苏晚乔追问后的暂停 / 继续或首次兑现后的后果，温婉第二次轻量出现。
   - 边界：生活气必须服务关系变化，不单独扩餐饮地图。

22. **最安全的退出方案**
   - 功能：江岚提出低风险路线，引发她与安若曦的决定权冲突。
   - 关系：晏林不能借闺蜜分歧扩大自己的位置。
   - 关键：安若曦明确“由谁承担损失，必须由当事人决定”。

23. **集团反击**
   - 功能：唐家外围压力从个人追求升级为项目、职位、声誉和私人安全的组合压迫。
   - 配角：直属上级、项目同事、追求者各自回收前文。
   - 关键：安若曦必须为自己的选择支付明确代价。

24. **隐藏变量**
   - 功能：江岚依据已知结构作出合理误判，晏林面临再次隐瞒风险的诱惑。
   - 结果：晏林把可说明的风险、能力边界和退出选项拆开，让当事人选择。

25. **卷末决裂**
   - 功能：安若曦与追求者及其越界方式正式决裂，留下事件记录。
   - 结果：安若曦确认晏林有资格站在她身边，但不进入身体兑现。

26. **管理局协查入口**
   - 功能：管理局提高对晏林能力边界与工作室案件的关注，以个案委托方式开启松散有偿协查。
   - 配角：管理局经办人只给最低称谓，不展开部门树。
   - 钩子：唐家确认晏林不是普通心理咨询师。

## 阶段一终点状态

### 关系状态

- **苏璃**：已定情，已完成首次身体兑现，知道晏林是星契者但不知道双子星结构和全部权柄。
- **苏晚乔**：已追问并进入明确关系；默认已完成首次身体兑现。若阶段一正文最终不兑现，必须停在强亲密后果场，并把 B 场前移到阶段二开头。
- **安若曦**：与晏林形成明确信任和暧昧边界，确认其有资格站在自己身边；不身体兑现。
- **温婉**：完成自然认识和职业生活埋线，不进入完整关系线。
- **唐语薇**：作为苏璃闺蜜、车行熟人和唐家伏笔存在；不解释完整家族结构。

### 事件状态

- 魏琳咨询只完成问题方向定位，不完成治愈。
- 酒吧事件完成现实越界、精神干扰和证据链启动。
- 安若曦完成与唐家外围追求者及其越界方式的现实决裂。
- 管理局个案协查关系开启，但不成为晏林官方靠山。
- 唐家提高对晏林的关注，但不全面开战。

### 信息暴露状态

- 晏林公开层仍是心理咨询师和登记梦境契约者。
- 苏璃知道晏林是星契者，并知道仍有不能交代部分。
- 安若曦知道晏林隐瞒了与能力相关的信息，并要求涉及自身风险的信息必须告知。
- 江岚知道工作室存在授权、记录和紧急处置边界问题，但不掌握完整双子星结构。
- 苏晚乔知道晏林的现实风险和关系变化，是否知晓星契细节另行拍板。
- 管理局只怀疑晏林登记资料不完整，不掌握完整真相。

## 禁止提前

- 不公开双子星结构、暗伴星、截断、逆流完整真相。
- 不让唐家全面开战，不建立完整家谱、继承制度和全部派系。
- 不展开黑市固定组织、完整交易网络或黑暗契约者总后台。
- 不让林芷兰正面抢戏；最多卷末以处置链缺口或善后名单作为钩子。
- 不让温婉完成告白、关系确认、完整宠物诊所异常事件或多人关系席位。
- 不把苏晚乔写成苏璃定情后才出现的新追求者。
- 不让江岚替安若曦决定，也不让江岚成为安若曦和晏林关系的审批人。
- 不让多人关系被一次三方谈话解决。
- 不设计全球、海外或跨国星契体系。
