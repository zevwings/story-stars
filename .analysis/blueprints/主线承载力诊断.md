# 主线承载力诊断

> 本文是分析层诊断，回答“现有 `story/arcs/` 与 `story/plotlines/` 是否撑得起主线”，不是正式正典，也不改动正式 `story/`。
>
> 正式事实仍以 `BIBLE.md`、`story/`、人物卡、世界观文件与已定稿章节为准。本文提出的补强方向均属未拍板方案；拍板前不得升格为正式节点或阶段变化，作者确认后再落对应真源或 `.analysis/blueprints/待确认/`。

## 一、缘起与问题

作者提出：按 `story/arcs/`、`story/plotlines/` 现有的全部内容，是否太少、撑不撑得起主线。

## 二、核定范围

本次诊断已读取：

- `story/arcs/` 全六篇（阶段一至五、尾声）与 `arcs/_index.md`。
- `story/plotlines/` 全部主线节点（`PL-M-001` 至 `PL-M-005`）。
- `story/plotlines/_index.md` 全量索引（主线 5、副线 14、亲密线 12 共约 31 个正式节点）与 `_status/pending.md`、`active.md`。
- 代表性副线节点：`PL-S-008`（非法种印首案）、`PL-S-009`（黑市链串联）。
- `characters/` 目录与主卡清单。
- `BIBLE.md` 关于主线与色带的定位段落。
- `story/synopsis.md`、`story/timeline.md`。

未通读的部分：`worldbuilding/` 全部、全部副线与亲密线节点正文、`.analysis/plotlines/` 施工稿。因此本文结论限于**结构层**，不下沉到单节点的写法评价。

## 三、当前结论

不是“太少”，是“偏”。

- 绝对量不小：约 31 个正式节点，五阶段每阶段上百行编排。
- 但重量几乎全部压在“男主 × 女主”这条线上。
- 真正撑不起的，是外部主线里的**反派 / 星辰契约黑暗线**——问题不在文件数量，而在缺人、缺变奏、缺引擎。

## 四、主线的两层口径

`BIBLE.md` 已经区分两条线（见 `BIBLE.md` 情欲主色与色带段）：

- **正向主线 = 男主 × 女主**，全书最高频主色。
- **反派 / 黑暗契约者 = 并行的黑暗线**，走黑暗场双声部，仅对敌、不落男主对女主。

“撑不撑得起”必须分这两层量，否则结论会打架。

## 五、第一层：正向主线——充分，甚至偏厚

- 12 个亲密节点（`PL-I-001` 至 `PL-I-012`）＋ 一批关系副线（坦白、簇内知情、跨簇可见）。
- 五簇结构完整：A 苏璃—唐雨薇、B 苏晚乔、C 安若曦—江岚、D 温婉、E 许知夏—许知秋，另有林芷兰与配角。
- 阶段一至五对每位女主每阶段的关系状态、知情边界、亲密推进与禁止项都有编排。

结论：这一层不是短板，是当前最结实的部分，暂不需要补。

## 六、第二层：外部主线——需再拆两条

外部主线不是铁板一块，内部承载力差异明显：

- **唐家继承线**（`PL-M-005` ＋ `PL-S-012`）：相对结实。有具体人物（唐雨薇、哥哥、强硬派、内部保护者）、有制度冲突（名册、财产、继承资格）、有唐雨薇的选择弧。这条不是问题。
- **反派 / 星辰契约黑暗线**（`PL-M-002`、`PL-M-003`、`PL-M-004` ＋ 黑市系 `PL-S-008/009/010/011`）：真正的空洞。下一节展开。

## 七、反派黑暗线的三个病灶

### 7.1 反派是“一张没有脸的方法体系”

- `characters/` 现有 15 张主卡，全部为男主、女主与男配（唐雨宸、唐雨霖、阮清黎、徐清、沈宁），**没有一张反派卡**。
- “核心反派”“核心施术者”在全部主线节点里都是纯功能位；每个节点都在强调“不揭身份、不写姓名、不建统一组织”。
- 结果：从阶段二追到阶段五，读者追的是一条匿名黑市链 ＋ 一种精神控制方法，缺一个能恨、能怕、能记住、能与晏林正面博弈的人。
- `PL-M-004` 名为“核心反派责任链清算”，但被清算的主体目前为空——没有主的清算，缺少分量。

> 说明：留白本身是正确的施工纪律（防止过早把姓名、组织写死）。病灶不在“暂不揭穿”，而在于连一个供全书追踪的**反派轮廓**都没有立起来。

### 7.2 事件类型单一，是同一种案子在放大

- 抽象看，`PL-M-001` → `PL-S-006` → `PL-M-002` → `PL-S-010` → `PL-M-003` → `PL-S-008` → `PL-M-004`，几乎都是“精神侵蚀个案 → 晏林截断止损 → 林芷兰治疗 → 管理局追责”的同构循环，只是规模从个人升到链条。
- 缺变奏与峰值：没有晏林能力反噬 / 暴露的高危时刻、没有一次真正的内部背叛、没有管理局从“受控合作”翻脸成对手、没有把主角逼到墙角的中段危机。
- 主线事件密度不足：阶段三至五跨度很长，主线 `PL-M` 只有 002 / 003 / 004 三个点，中段主要靠副线 `PL-S` 填充。

### 7.3 最大的金手指没有被当外部引擎用

- 双子星 / 暗伴星 / 截断 / 逆流，目前几乎只在“谁知道晏林真身”这个**关系知情边界**里发挥作用。
- 它本应驱动外部主线：登记不完整 → 管理局复评本可是巨大外压，但 `PL-S-005` / `PL-S-007` 被定位成“受控合作、必要更正备案”，冲突烈度被主动调低。
- 反派为什么存在、终局赌什么，目前与这个金手指没有强绑定。
- 对照：情感线有清晰引擎（五簇 ＋ 情欲主色 ＋ 知情差），外部主线缺一个同级别的引擎。

## 八、两个容易被漏算的账

### 8.1 反派单薄不只是情节问题，也是色带供给问题

- 尺度口径 v2 的四档色带谱里有“黑暗场”一档；`BIBLE.md` 规定黑暗场**只能落在对敌 / 反派 / 黑暗契约者身上**，不落男主对女主。
- 即反派是最深那档色带的**唯一合法供给源**。反派越是“没有脸的方法体系”，黑暗色带就越缺稳定的人来承载。
- 推论：补反派，顺带把黑暗色带喂饱；不补反派，最深一档色带将长期缺料。

### 8.2 反派宜为主题的对手盘，不是外挂

- 全书核心命题（`PL-M-003` 已点明）：“谁有权解释一个人的欲望与选择”。
- 顺此命题，具名反派的最优解是**与晏林同源、却走到操控极端的人**：一样能碰欲望与精神，晏林守边界（截断即止、非治疗非报偿），他越界（强种、剥夺退出权）。
- 这样反派立起来不费力，还能反衬男主——这也是把“立反派”列为杠杆点的原因。

## 九、可拍板的三个支点

均为方向，不在本文落实为节点：

1. **立一个具名核心反派**——身份、动机，以及他与晏林 / 星辰契约 / 唐家的关系。
2. **给黑暗线加事件变奏 ＋ 一个中段高危峰值**（放阶段三或四），打破“截断—治疗—追责”的同构。
3. **把金手指升格为外部悬念引擎**——让管理局复评烈度、终局赌注与双子星结构强绑定。

支点关系：**支点 1 是杠杆**。反派一旦有人、有动机，支点 2 的变奏与支点 3 的引擎会顺势长出来。

## 十、待作者拍板的前置问题

**反派 / 黑暗线在本书里是“背景板”还是“第二引擎”？** 两个方向的结论相反：

- 若定位为**背景板**（本书就是情欲主色的多女主故事，外部冲突只负责给情感施压）：现状其实够用，甚至可以再收，避免查案线挤占主色频率；三个病灶不算病，属有意留白。
- 若定位为**第二引擎**（希望读者除了记住七位女主，也记住一个够分量的对抗骨架）：三个病灶须补，尤以支点 1（立反派）为先。

此问不拍板，后续补强方向无法确定，本文不再往下推。
