# 阶段三编排

> 本页是阶段三施工总控，不是正式正典。已确认事实仍以 `BIBLE.md`、`story/synopsis.md`、`characters/` 与 `worldbuilding/` 为准。本页确认后，再提炼稳定结论进入 `story/arcs/阶段三.md`；未确认章序不提前登记到 `chapters/_index.md`。

## 阶段定位

阶段三承担“唐家合流与双线正式进入”功能。

阶段二已经完成正式修罗场启动：苏璃 / 苏晚乔 / 晏林完成第一轮三人边界确认，安若曦和江岚进入有限协商，唐语薇通过车行独立事件与晏林建立吸引，管理局协查和黑暗契约者案件开始常态化。阶段三要把这些压力合流，让安若曦和唐语薇正式进入关系，同时让协商成阵从口头边界进入可反复执行的生活秩序。

核心推进：

1. 安若曦从阶段二的有限协商推进到正式定情与首次身体兑现。
2. 唐语薇家族身份正式被唐家重新翻出，但不写成回家夺权。
3. 唐语薇与晏林在苏璃知情、三方谈话之后正式定情并首次身体兑现。
4. 江岚越界到亲吻，承认私人欲望，但清醒后暂停，不进入 B 场。
5. 唐家温情兄长作为有限保护者出场，给唐语薇消息、旧物和余地，不替她决定。
6. 管理局通过梦境追溯摸到黑市和唐家外围交易影子，但不直接查穿幕后。
7. 工作室日常、双胞胎、饮食、天气和低压场景继续承担生活气与关系后果；双胞胎开始出现生活型暧昧前置。
8. 温婉不强进入，只保持稳定生活线和慢热存在感，不告白、不确认、不 B。

## 阶段三主线

### 一、安若曦正式进入

阶段二的安若曦已经确认晏林有资格靠近，但她还没有把自己交给这段关系。阶段三需要让她在唐家线和管理局线的双重压力下主动选择，而不是被危机推倒。

推进要点：

- 唐家线让她确认：自己承受的职场压迫，与唐语薇被追回的家族逻辑同源。
- 晏林不能只说“我会保护你”，而要把涉及她的风险、能力边界和退出选项摊开。
- 安若曦不是被打动后软化，而是在看清代价后仍然选择靠近。
- 首次 B 场必须写成清醒、判断、主动允许和可撤回，不写成安抚奖励。

阶段终点：

- 安若曦正式进入明确关系。
- 她保留判断权、告知权、暂停权，不因身体兑现而默认放弃边界。
- 江岚知道这段关系已经越过原来的“闺蜜把关”范围，为自己的越界埋压。

### 二、江岚亲吻越界

江岚阶段三可以越界，但不能兑现到 B 场。她的功能是让安若曦 / 江岚 / 晏林这组三角从“理性说开”进入私人欲望的破口。

推荐触发：

- 唐家、协查材料或黑市线索带来高压共处。
- 江岚发现自己已经不是纯粹替安若曦审查晏林。
- 晏林在她靠近时停住，江岚反而主动吻他，或在一次情绪失控后亲吻成立。

吻后处理：

- 江岚必须清醒暂停，不能顺势进入 B 场。
- 她明确这件事不能当没发生，也不能继续往下走。
- 她需要先面对安若曦，也要确认自己不是借“保护若曦”进入这段关系。

阶段终点：

- 江岚承认私人欲望存在。
- B 场后移到阶段四，阶段三只留下清楚的悬置压力。

### 三、唐家回收与温情兄长

唐家合流不是唐语薇回家夺权，而是唐家重新试图解释她是谁。

阶段三唐家压力可从“翻她的名字”开始：

- 有人重新查看她是否仍在唐家名册上。
- 有人关注她是否已经登记星契。
- 有人试图把她的职业、住处、关系和未来开店计划纳入家族可控范围。
- 有人把安若曦职场事件与唐语薇的回收放进同一条席位争夺链。

温情兄长建议作为有限保护者出场：

- 他知道唐语薇在哪里，但这些年没有逼她回家。
- 他第一次出场可以只是来车行送饭、送旧工具箱或旧车库钥匙。
- 他提醒她：“他们最近在翻你的名字。”
- 他给她消息、旧物和几天余地，不给命令。
- 他仍有家族思维，容易下意识替她挡、替她安排，因此必须被唐语薇校正。

兄长边界：

- 他不能替唐语薇解决唐家。
- 他不能成为唐语薇自由的授予者。
- 他不能用温情包装控制。
- 他不能压过苏璃 / 唐语薇 / 晏林三方关系主线。

修车行方向：

- 阶段三不必立刻开店，但要让“自己的修车行”成为唐语薇明确说出口的目标。
- 资源来源不写成哥哥甩钱，而是唐语薇拿回本来属于自己的东西：旧工具、母亲或外婆留下的一点钱、被冻结的小额分红、旧车库记忆或合法客户资源。
- 店名、地址、经营方向和第一批客户必须由唐语薇自己决定。

阶段终点：

- 唐语薇不再只是拒绝回家，而是开始决定如何处理自己和唐家的关系。
- 她的职业目标从“在车行成长”推进到“将来有自己的店”。

### 四、唐语薇正式进入

唐语薇正式进入必须晚于苏璃知情和三方谈话。

推进顺序：

1. 唐家压力进入车行或她的日常。
2. 兄长提醒她，但不替她处理。
3. 她的预感权柄在车行、骑行或家族压力事件中暴露。
4. 苏璃发现她和晏林之间真的动心。
5. 苏璃、唐语薇、晏林三方谈话。
6. 唐语薇明确问晏林：“你要不要认真追我？”
7. 正式定情。
8. 首次 B 场。

三方谈话要拆清：

- 苏璃有权嫉妒和受伤，但不是许可者。
- 唐语薇不是来问苏璃能不能把晏林分给她。
- 晏林不能用“还没做什么”逃避关系责任。
- 三人不能用退让代替表达。
- 唐语薇与晏林的关系必须独立成立，不能依附苏璃。

首次 B 场关键词：

- 抢回节奏。
- 不是被允许。
- 不是被保护收编。
- 晏林在她说停时必须停，在她主动时必须认真回应。

阶段终点：

- 唐语薇正式进入明确关系。
- 苏璃与唐语薇的姐妹关系受伤但不破裂，并进入新的修复周期。

### 五、管理局黑市追溯

阶段三可以通过管理局协查把黑市线和唐家外围牵到一起，但不能直接查到最终结果。

推荐用法：

- 管理局委托晏林做梦境复盘或污染受害者追溯。
- 梦境里出现碎片：车行附近的人、唐家项目影子、匿名中介、异常补剂、观星资源、某个熟悉但不能确认的家族称谓。
- 晏林能看见心理材料和象征性线索，但不能把梦境当铁证。
- 江岚负责提醒证据边界，安若曦负责识别职场与产业链条，唐语薇负责识别唐家内部口吻或旧物痕迹。

阶段边界：

- 不把管理局写成主角靠山。
- 不让晏林获得内部档案权限。
- 不让梦境复盘直接定罪。
- 不把唐家写成黑暗契约者总后台。
- 不展开统一黑市组织。

阶段终点：

- 管理局只确认“唐家外围与黑市接口存在可疑重叠”。
- 唐家只确认晏林和女主阵营正在逼近某些不该被看见的链条。
- 黑市和星辰之印主战场继续留到阶段四。

### 六、生活线与双胞胎

阶段三压力复杂，必须用工作室和城市日常把关系落地。

双胞胎用法：

- 许知夏负责预约、记录、时间表和材料流程，发现协查事项挤压普通咨询。
- 许知秋负责前台、来访者感受和外侧观察，察觉晏林身边来的人越来越不普通。
- 两人开始意识到自己对晏林不是普通员工关心，但仍用玩笑、流程和留饭把情绪压住。
- 她们可以吃味，可以短暂停顿，可以在闭店后把话说到一半又绕回玩笑。
- 她们可以察觉异常节奏，但不能自动知道星契真相。
- 她们不能利用咨询隐私开玩笑，也不能随意传播江岚、唐语薇或安若曦的信息。

双胞胎暧昧前置：

| 类型 | 写法 | 功能 |
| --- | --- | --- |
| 习惯性亲近 | 闭店后收拾桌面、递外套、给晏林留饭、顺手替他改掉不合理日程 | 让亲近从长期生活里长出来 |
| 玩笑失效 | 照常调侃“老板今晚约谁”，听见安若曦或唐语薇名字后短暂停顿 | 暗示吃味，不明说 |
| 守门压力 | 管理局、女主和异常来访者频繁出现，她们替晏林挡场面，却仍被排除在真相外 | 推到阶段四“我们不装了” |
| 流程追问 | 晏林试图把协查说成普通委托，她们按普通流程反问责任归属 | 让感情和职业边界同时升温 |

阶段三不能写成：

- 双胞胎正式告白。
- 双胞胎亲吻或身体兑现。
- 双胞胎知道双子星结构。
- 双胞胎越权读取咨询记录或协查材料。
- 许知夏与许知秋作为绑定套餐同步动心；两人必须有独立反应。

生活场类型：

| 场型 | 阶段三功能 |
| --- | --- |
| 工作室开门前 / 闭店后 | 展示晏林疲惫、双胞胎维持秩序、协查挤压日常与暧昧前置 |
| 车行夜宵 | 承接唐语薇家族压力和苏璃闺蜜关系 |
| 雨天 / 饭点 / 临时加班 | 给安若曦亲密后生活后果，不只停在 B 场 |
| 花店回访 | 保持林芷兰存在感，不推进治疗即坦白 |
| 宠物诊所 / 安安 | 保持温婉慢热，不强进入主关系 |
| 周末短休 | 让内圈关系执行陪伴顺序和可停点 |

### 七、黑市家族线显影（补入）

> 本小节为黑市家族线在本阶段的显影补入建议，尚未拍板；确认后并入上文「五、管理局黑市追溯」与推荐章序。黑市家族独立于唐家，详见 [黑市家族线](../plotlines/黑市家族线.md)。

阶段三让黑市家族的**壳子第一次隐约成形**，但仍不出正脸。

- 挂载「17. 梦境碎片」：观星资源残影、匿名中介、星表买卖痕迹里，让某个供应商 / 保管方 / 老字号名号**反复出现一次以上**，形成可追的家族接口感。
- 与唐家的关系按“**疑似重叠但不同源**”写：梦境里“唐家项目字样”和黑市接口同框，但**是两个后台**，不能让读者默认黑市＝唐家后台。
- 挂载「18. 三人识别」：江岚划证据边界、安若曦识产业链、唐语薇识唐家口吻时，明确“这条供货 / 情报链不属于唐家内部口吻”。

阶段边界：

- 家族仍不出正脸、不拍板姓氏。
- 不让梦境碎片把黑市家族坐实成铁证。
- 人体实验、暗杀、洗白不在阶段三展开。

| 温婉 / 宠物诊所 | 低压生活与慢热存在感，不承担阶段三兑现压力 |
| 林芷兰 / 花店 | 协查后回访和生活侧稳定，不治疗即坦白 |

## 推荐章序

> 以下是候选章节节点，不代表最终章号。实际写作时可按正文密度合并或拆分。目标规模暂按 34-40 章控制。

### 第一段：安若曦正式选择

1. **协商之后的安静**
   - 功能：承接阶段二安若曦 / 江岚 / 晏林三方说开。
   - 重点：安若曦不是被安抚，而是在重新整理自己的判断权。

2. **唐家压力同源**
   - 功能：安若曦发现职场链条与唐语薇被翻名册有交叉。
   - 边界：她只能看见产业与人事层面的重叠，不能掌握完整唐家结构。

3. **风险说明**
   - 功能：晏林把涉及安若曦的能力边界、协查风险和退出选项说清。
   - 重点：他不再把“我会处理”当成告知。

4. **安若曦主动靠近**
   - 功能：安若曦确认继续，不是因为安全，而是因为她愿意。
   - 亲密功能：强 A 场或身体临界。

5. **安若曦定情**
   - 功能：明确关系成立。
   - 边界：保留判断权和暂停权。

6. **安若曦首次兑现**
   - 功能：首次 B 场。
   - 边界：清醒、自愿、可撤回；具体场景后续进亲密专项设计。

7. **亲密后的判断权**
   - 功能：B 场后果不以甜蜜抹平风险。
   - 重点：安若曦确认身体关系没有让她失去判断权。

### 第二段：江岚破口

8. **江岚看见结果**
   - 功能：江岚面对安若曦与晏林关系正式成立。
   - 重点：她不能再把自己的情绪解释成职业判断。

9. **材料夜审**
   - 功能：江岚与晏林高压共处，整理唐家与协查材料。
   - 边界：法律证据、梦境材料和个人猜测必须分开。

10. **亲吻越界**
   - 功能：江岚承认私人欲望破口。
   - 边界：亲吻后暂停，不进入 B 场。

11. **不能当没发生**
   - 功能：江岚要求这件事先面对安若曦，也面对自己。
   - 结果：阶段四江岚兑现线留下明确钩子。

### 第三段：唐家翻名与兄长出场

12. **车行门口的饭**
   - 功能：唐语薇兄长温情出场。
   - 可写锚点：饭盒、旧工具箱、旧车库钥匙、手上的旧伤。
   - 重点：他不进工位打扰，只提醒她家里在翻她的名字。

13. **旧车库**
   - 功能：唐语薇回忆最早拆车、出走与被保护的复杂感。
   - 边界：不一次讲完唐家前史。

14. **强硬派来信**
   - 功能：唐家压力从人情变成现实动作。
   - 方向：名册、登记、职业归属、婚配资格或家族活动邀请择一。

15. **晏林做错的保护**
   - 功能：晏林试图替唐语薇挡掉一部分压力。
   - 结果：唐语薇校正他，兄长也被校正：可以帮，别替她决定。

### 第四段：梦境追溯与黑市影子

16. **协查新材料**
   - 功能：管理局拿到阶段二污染案后续或新受害者材料。
   - 边界：个案委托，不给晏林内部权限。

17. **梦境碎片**
   - 功能：晏林通过梦境追溯看见黑市接口影子。
   - 可见物：匿名中介、异常补剂、唐家项目字样、车行周边路线、观星资源残影。
   - 边界：梦境不是铁证。

18. **三人识别**
   - 功能：安若曦识别产业链条，唐语薇识别唐家口吻，江岚划出证据边界。
   - 结果：唐家与黑市只是疑似相连，不查穿。

19. **唐语薇预感暴露**
   - 功能：车行、骑行或回家压力中，唐语薇的预感权柄显影。
   - 边界：预感只给危险信息，不强化身体或车辆。

### 第五段：苏璃发现与三方谈话

20. **苏璃看见火**
   - 功能：苏璃确认唐语薇和晏林不是普通熟人动心。
   - 重点：她可以嫉妒、受伤和害怕失去闺蜜。

21. **不是许可**
   - 功能：苏璃与唐语薇先单独冲突。
   - 核心：唐语薇不是来问苏璃能不能把晏林分给她。

22. **三方谈话**
   - 功能：苏璃 / 唐语薇 / 晏林拆清感情、隐瞒、退让和知情边界。
   - 边界：只解决这一组三人关系的最低共识，不解决全员协商成阵。

23. **认真追我**
   - 功能：唐语薇主动向晏林要明确回答。
   - 核心句：`现在只问你，要不要认真追我？`

### 第六段：唐语薇正式进入

24. **定情**
   - 功能：唐语薇与晏林正式确认关系。
   - 边界：不是苏璃允许，不是晏林保护收编。

25. **首次兑现**
   - 功能：唐语薇首次 B 场。
   - 关键词：抢回节奏、主动选择、说停就停、清醒回应。
   - 后续：具体 B 场另行进入亲密设计流程。

26. **姐妹修复**
   - 功能：苏璃和唐语薇不经过晏林修复一次关系。
   - 重点：闺蜜关系不是后宫关系的附属品。

27. **自己的店**
   - 功能：唐语薇明确说出将来开自己修车行的目标。
   - 边界：阶段三不必立刻开店，只让目标扎根。

### 第七段：生活回压与双胞胎前置

28. **工作室异常节奏**
   - 功能：双胞胎察觉协查、来访者和晏林疲惫越来越异常。
   - 边界：她们仍按普通人写，不自动知道星契。

29. **预约表上的名字**
   - 功能：许知夏发现苏璃、苏晚乔、安若曦、唐语薇出现频率和时间段都不普通。
   - 亲密功能：她用玩笑盖过去，但第一次意识到自己不是单纯在替老板排班。
   - 边界：不窥探隐私，不追问关系细节。

30. **闭店后的饭**
   - 功能：双胞胎给晏林留饭，饭凉了又热第二次。
   - 亲密功能：许知秋照常补刀，话到一半卡住，改成把筷子递过去。
   - 边界：她们不要求晏林马上解释，只让读者看见委屈开始成形。

31. **普通委托**
   - 功能：晏林试图用“普通委托”带过协查风险。
   - 关键：双胞胎不追问星契，只问如果按普通委托走，出了事是否也按普通责任算。
   - 结果：阶段四“我们不装了”有了职业与情感双重前置。

### 第八段：阶段收束

32. **低压饭点**
   - 功能：苏璃、苏晚乔、安若曦、唐语薇的陪伴顺序开始变成实际生活问题。
   - 边界：不写成全员固定大团建。

33. **温婉与安安**
   - 功能：温婉稳定出现，给高压阶段换气。
   - 边界：不告白、不确认、不 B，不承担调停。

34. **花店回访**
   - 功能：林芷兰保持专业与生活侧存在。
   - 边界：不治疗即坦白。

35. **阶段三尾钩**
   - 功能：唐家确认唐语薇不再只是出走女儿，管理局确认黑市接口疑点未断。
   - 后续：阶段四进入更深黑暗对照、江岚兑现、双胞胎基本挑明和林芷兰治疗即坦白。
