# 世界观待确认

> 本页只登记未拍板的公共设定。角色专属能力与人物经历见 [人物](人物.md)。

## 首卷世界观决策板

### 首卷必须拍板

- [ ] **唐家外围压力的合法外壳**：集团业务、职场权力、项目/应酬/评价/资源撤回如何形成现实压迫链。
- [ ] **低级契约者干扰口径**：酒吧事件采用哪类精神干扰、成立条件、连接方式、授权链与失败后果。
- [ ] **管理院注意晏林的触发点**：第一卷中哪一次能力表现让管理院怀疑其登记资料不完整。
- [ ] **事件记录归属**：卷末事件由管理院、警方、集团内部程序或多方记录留存，决定后续唐家如何承压。
- [ ] **江州首卷空间关系**：林语心理咨询工作室、酒吧、训练馆、修理店、宠物诊所、安若曦职场的共同活动半径。
- [ ] **知情普通人边界**：魏琳咨询、酒吧人员、工作室助理、普通证人分别能知道到什么程度。

### 随剧情启用

- [ ] 管理院中央与地方机构层级、跨城管辖、舆论管控、记忆干预和知情普通人处置流程。
- [ ] 唐家在星辰家族体系中的正式位置，以及与管理院、星表和黑市的利益关系。
- [ ] 合作医院、督导师、诊疗转介、精神检测设备与合法医疗边界。
- [ ] 黑市网络、黑暗契约者小团体、药剂供应链、跨省/境外协作机制。

### 可以暂缓

- [ ] 架空纪年、架空国名、境外城市专名、现实品牌替代策略。
- [ ] 远期城市映射、跨国制度、黑市固定场所和后期组织专名。
- [ ] 未进入首卷的药剂细则、检测设备供应者与技术体系差异。

## 随剧情启用

### 权柄与能力

- [ ] 是否另设可与植物或其他生命沟通的星辰权柄；灵犀当前只作用于动物。

### 势力与组织

- [ ] 唐家在星辰家族体系中的位置，以及与管理院、星表的利益关系。
- [ ] 具体星辰家族、管理院内部派系与制度名称。
- [ ] 管理院与地方政府、警察系统及星辰家族之间的权力结构。
- [ ] 是否设置独立于精神之海五境的监管风险分级。
- [ ] 跨省、跨国的保密与超凡事件协作机制。
- [ ] 黑市的具体交易网络、固定场所或组织是否需要进入正式设定。
- [ ] 黑暗契约者是否形成具体反派组织；不预设覆盖地下世界的统一机构。
- [ ] 普通社会是否存在星辰之力的都市传说或模糊认知。

### 专名与地理

- [ ] 架空纪年专名。
- [ ] 架空国名。
- [ ] 东京系亚洲邻都与纽约系西方都会的城市专名。
- [ ] 摩托车等现实品牌的架空专名策略。
- [ ] 药剂、黑市组织、精神侵蚀手段与星辰之印是否需要正式专名。
- [ ] 江州主要职业空间的具体位置与共同都市活动半径。
- [ ] 长安、临安、姑苏、锦城、汴京、神都／洛邑、奉天、江夏、穗城、鹏城、津门、金陵等城市映射是否启用。
- [ ] “恭州”是否作为江州下半城老城区或码头区的旧地名。

### 技术

- [ ] 星契观测、精神检测设备的具体技术体系、供应者、可靠性与监管差异。
