# 世界观待确认

> 本页只登记未拍板的公共设定。角色专属能力与人物经历见 [人物](人物.md)。

## 归属规则

- 首卷事件压力、能力暴露、空间调度、记录归属和知情边界归入对应 `plotlines/` 文件。
- 本页只保留公共机制、制度、专名、地理、技术与势力结构中尚未拍板的设定问题。

## 随剧情启用

### 权柄与能力

- [ ] 是否另设可与植物或其他生命沟通的星辰权柄；灵犀当前只作用于动物。

### 势力与组织

- [ ] 具体星辰家族、管理局内部派系与制度名称。
- [ ] 管理局与地方政府、警察系统及星辰家族之间的权力结构。
- [ ] 是否设置独立于精神之海五境的监管风险分级。
- [ ] 跨省、跨国的保密与超凡事件协作机制。
- [ ] 黑市的具体交易网络、固定场所或组织是否需要进入正式设定。
- [ ] 黑暗契约者是否形成具体反派组织；不预设覆盖地下世界的统一机构。
- [ ] 普通社会是否存在星辰之力的都市传说或模糊认知。

### 专名与地理

- [ ] 架空纪年专名。
- [ ] 架空国名。
- [ ] 东京系亚洲邻都与纽约系西方都会的城市专名。
- [ ] 摩托车等现实品牌的架空专名策略。
- [ ] 药剂、黑市组织、精神侵蚀手段与星辰之印是否需要正式专名。
- [ ] 江州主要职业空间的具体位置与共同都市活动半径。
- [ ] 长安、临安、姑苏、锦城、汴京、神都／洛邑、奉天、江夏、穗城、鹏城、津门、金陵等城市映射是否启用。
- [ ] “恭州”是否作为江州下半城老城区或码头区的旧地名。

### 技术

- [ ] 肉体兴奋药剂的药效时长、配方边界、成瘾表现与管制等级。
- [ ] 星契观测、精神检测设备的具体技术体系、供应者、可靠性与监管差异。
