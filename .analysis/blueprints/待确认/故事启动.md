## 第一卷骨架

### 第一章开场

第一章先从林语心理咨询工作室的一次入梦咨询开始，展示晏林已经公开登记的梦境权柄、现实职业状态与知情授权边界；咨询结束后再转入酒吧，由苏璃与安若曦接入第一卷主线。

具体咨询结构、梦境内容、家长沟通与后续引导统一见 [魏琳咨询线](../../剧情线/魏琳咨询线.md)，本页不重复保存。

### 首卷必须拍板

- [ ] **女主入场批次**：当前推荐为安若曦、苏璃承担首卷主轴；苏晚乔第一卷前段强出场并兑现训练前置；唐语薇、温婉轻埋线；江岚从法律入口接入；林芷兰压到第一卷末短露面或第二卷正式入场。
- [ ] **首卷主线发动机**：安若曦遭遇唐家外围追求者的现实施压与低级星契干扰，酒吧事件是第一次公开显影。
- [ ] **苏璃定情线位置**：苏璃与晏林的短途旅行、关系确认与身体关系发生在酒吧事件之后，并与安若曦证据线并行推进。
- [ ] **苏晚乔追问线位置**：苏晚乔必须在苏璃定情前完成足够出场；苏璃定情后，晏林必须及时面对与苏晚乔既有暧昧的责任。
- [ ] **江岚法律入口**：安若曦介绍江岚认识晏林与工作室；江岚先处理法律、授权、记录与责任边界，不直接作为恋爱通道。
- [ ] **唐语薇首卷功能**：先作为苏璃闺蜜、修理店日常入口与唐家伏笔存在，不提前解释唐家外围结构。
- [ ] **温婉首卷功能**：通过安安、宠物诊所与晏林精神状态异常埋线，不抢唐家主线。

## 首卷能力暴露顺序

- [ ] 晏林：公开梦境能力用于咨询；幻境、截断、逆流只在必要时局部暴露，且优先维持“登记不完整但真相未明”的制度张力。
- [ ] 安若曦：真言不替代证据，首卷必须出现一次“不能说谎不等于客观真相”的盲区。
- [ ] 苏璃：情绪权柄是否在首卷主动使用仍待确认；酒吧事件优先靠现实观察、调酒师经验与现场处理能力成立。
- [ ] 苏晚乔：御物权柄是否首卷暴露待确认；普通搏斗训练必须先作为现实能力前置并在唐家外围冲突中兑现。
- [ ] 唐语薇：预感权柄首卷建议只埋伏，不急于公开；星契秘密暴露可后移。
- [ ] 温婉：灵犀首卷只通过安安和动物当前情绪埋线，不进入完整案件。
- [ ] 江岚：脉络权柄是否首卷出手待确认；第一卷优先呈现法律与程序边界能力。
- [ ] 林芷兰：净化、修复不进入第一卷主战场，避免压缩安若曦、苏璃与唐家外围主线。
