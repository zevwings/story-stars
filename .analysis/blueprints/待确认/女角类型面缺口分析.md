## 三、四个结构性缺口

**① 气质色系：「清·温·静」四连扎堆。** 安若曦 / 江岚 / 温婉 / 林芷兰 第一眼都是素雅柔光系（月白·水仙·暖玉·木兰）。功能区分得开（判断 / 守门 / 归处 / 修复），但第一眼气质容易糊。属既有资产的优化项，不必加人。

**② 情感姿态：约六成在「藏」，主动 / 攻击性稀缺。** 真正外放扑近的只有苏晚乔、唐语薇两人，且都偏年轻莽撞的「冲」。缺成熟主导型，缺危险 / 占有欲强的异色。

**③ 年龄带过窄（最该补）。** 24–27，全部 ≤ 男主晏林（27），扎堆 24 岁（六人）。没有年上、没有 28+ 熟龄、没有 21 以下青涩。后宫长篇依赖年龄层次制造关系质感差，目前是一整层「轻熟同龄」。

**④ 立场清一色（最该补，且是剧情漏洞）。** 9 位全是内圈、正向、善良底色。反派侧 0 女性、管理局侧 0 女性——唐家反派全是男性纨绔，唐语薇兄长（男）是温情内线，管理局无具名女性。这让中期修罗场与全书势力斗争缺女性对手戏；`BIBLE.md` 调高了黑暗场占比，却无黑暗侧女性锚点。

> 权柄面（真言 / 情绪 / 灵犀 / 净化修复 / 脉络 / 御物 / 预感）覆盖均衡、符合「精神系收敛」，不建议动。
