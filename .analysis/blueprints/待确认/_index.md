## 分类

- [人物](人物.md)：人物前史、职业细节、星契落点、圈层入口与关系事件。
- [世界观](世界观.md)：星契公共机制、制度、势力、技术、专名与地理。
- [故事启动](故事启动.md)：第一卷骨架、人物入场顺序、线索与运行层启用条件。

## 处理规则

1. 新出现的未定事项先进入本目录，不写进正式真源。
2. 作者确认后，先回写对应正式文件，再从本目录删除该项。
3. 同一问题只保留一个主条目；跨模块影响写在主条目下，不重复登记。
4. 模板占位、尚未产生的章节状态、空索引不属于设定待确认，不登记在此。
5. 历史废案与讨论过程移入 `.analysis/archived/`，不留在本目录。
6. 本目录只记录当前问题，不记录何时拍板、何时更新或为什么更新；这些信息由 git 历史与归档材料承接。
