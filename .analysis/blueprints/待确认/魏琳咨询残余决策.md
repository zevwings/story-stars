# 魏琳咨询残余决策

> 本页只保存阶段一魏琳咨询尚未拍板的场景参数与未来升格判断，不是正式正典。已经确认的开场功能、咨询边界与阶段结果读取 [`story/arcs/阶段一.md`](../../../story/arcs/阶段一.md)。

## 正式承接

阶段一已经确认：

- 魏琳咨询用于展示晏林登记在册的梦境权柄、心理咨询职业和知情授权原则。
- 梦境辅助服从现实评估、本人同意、最低干预与醒后现实核验，不替代诊疗或当事人选择。
- 第一次咨询只定位问题方向，不完成治愈。

上述结论不在本页重复展开或重新定义。

## 候选场景备忘

- 魏琳暂按 17 岁高二学生设计，经合作医院评估后转介；监护授权不能替代本人同意。
- 候选心理冲突为：好友遭同伴排挤后，她因恐惧选择沉默，继而把返校理解为对好友的背叛。
- 候选梦境意象为末班车：她在站台等待已经不会出现的好友；晏林只让车门多停几秒，是否上车仍由她决定。
- 梦醒后必须通过现实访谈核验，不把梦境象征当作事实、诊断或证据。
- 当次现实行动暂定为穿便服走到学校附近，并保留不进入校园的选择。
- 家长沟通只披露魏琳同意或安全处置必需的信息，不公开完整梦境、好友身份与无关隐私。
- 后续恢复可通过工作室日常简短反馈，不默认把魏琳发展为核心人物线。

## 待确认

1. 魏琳、家长、学校与医院分别能知道到什么程度。
2. 魏琳后续咨询在阶段一之后的出现频率，以及是否只通过工作室日常反馈进度。
3. 学校能够提供的保护措施、医院复诊意见与渐进返校节奏。
4. 年龄、问题根因、末班车意象和首次现实行动是否作为第一章最终方案采用。

## 升格判断

- 咨询流程、梦境意象、对白、后续频率与返校步骤只进入章节大纲，不建立正式剧情节点。
- 只有定稿后产生必须跨章继承的独立案例后果，并具备明确变化主体和完成判据时，才考虑新建正式副线节点。
- 魏琳保持单元来访者定位，不因开场示范自动进入核心人物线。
