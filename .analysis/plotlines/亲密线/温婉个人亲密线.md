# 温婉个人亲密线

> 本页是温婉与晏林跨阶段、可反复调用的亲密回路施工，不是正式正典，也不承载具体章节的完整身体流程。
>
> 关系何时成立、温婉何时知晓复杂关系与晏林风险，仍以 [温婉线](../主线/温婉线.md)、阶段编排、正式人物卡与已定稿章节为准。具体 A／B 场设计进入 `intimacy/chapters/chXXX/scene_NN/`。

## 一句话定位

温婉在完整知道关系结构及影响自身安全与选择的必要事实、并主动选择晏林之后，以迎接、更衣、沐浴、身体照料、主动取悦、陪睡与晨间整理组成“专属侍奉夜”，让长期负责观察和收束的晏林暂时彻底卸力；她不因此知道双子星完整结构。与此同时，她必须逐步学会说出自己的需要，并在后段把照料权完整交还给晏林。

## 核心回路

> 主动留人 → 接管当晚的生活节奏 → 晏林逐步交出决定与照料责任 → 温婉获得专属观察与服侍权限 → 晏林真实失守 → 温婉说出自己的需要 → 照料权反转或互换 → 次日生活权限留下变化。

这条线的核心不是新奇器具，而是两件事同时成立：

- **晏林的奖励**：从进门到入睡不必继续观察、安排和接住任何人，可以完整接受温婉的照料与主动取悦。
- **温婉的奖励**：晏林只在她面前卸下从容，让她拥有当晚排他的注意力、观察权与安排权；她也能在需要时要求回报，而不是靠无限付出来换位置。

## 启用前提

1. 温婉与晏林已经独立确认关系，不以安安、诊所事件或脆弱状态代替定情。
2. 温婉已经知道晏林身边存在复杂关系，并获得足以影响自己选择的真实信息。
3. 首次完整兑现必须发生在清醒、无职业权力交换、无危机催熟的情况下。
4. “今晚交给我”只授权当晚已协商的内容，不构成长期服从、随时服务或永久性可用。
5. 服务项目、身体接触、道具、留宿与场后照料分别可以接受、删减、暂停或撤回。

## 一、她的服侍不是贤惠劳动

### 关系定义

温婉的服侍属于**主动建立的亲密仪式**，不是家务分工，也不是固定身份。

- 她可以享受准备、照料、观察和让晏林卸力的过程。
- 晏林接受服务时必须给出真实反馈，不能把她的周到当成默认义务。
- 温婉可以只做其中一段，也可以中途改变主意；已经准备好空间不等于同意全部身体层内容。
- 温婉疲惫、生病、工作受挫或单纯不想服务时，无须给出补偿。
- 日常家务仍按共同生活规则分担，不能把情趣协议扩张成她长期承担两个人的生活劳动。

### 主动方式

温婉的主动不靠高声命令，而靠完整、清楚、已经想好的安排：

- 主动说“今晚别走”或“今晚交给我”。
- 接过外套、钥匙和手机，但是否暂存工作设备由晏林明确同意。
- 让晏林坐下、闭眼或停止继续处理事情。
- 按顺序完成更衣、沐浴、放松与亲密照料。
- 晏林想立刻反过来照顾她时，她可以明确说“今晚先不用管我”。
- 她真正需要回报时，也必须亲口说出来，不等晏林猜。

## 二、专属内容谱

### 1. 迎接与卸载

- 接过外套、钥匙与随身物品，给晏林一个从工作状态退出的现实动作。
- 准备饮水、换洗衣物与安静空间，不追问他暂时不想谈的事件细节。
- 用“今晚想自己来，还是交给我”完成当次总入口确认。
- 晏林可以只接受休息与照料，不自动进入身体兑现。

### 2. 更衣与整理

- 温婉替他整理袖口、衣领和随身物品，身体距离逐级缩短。
- 她可以检查疲劳、肌肉紧张或现实擦伤，但不把兽医职业能力迁移成人体诊疗。
- 晏林逐渐停止自己完成每一步，形成身体与决定权的有限交付。
- 更衣结束后再次确认是否继续，不能把前一段同意顺延到底。

### 3. 沐浴与身体照料

- 调整水温，准备毛巾与清洁用品，完成洗发、冲洗、擦干与吹发等伴侣照料。
- 重点写温度、掌心、头发、肩颈和呼吸变化，不写成行业服务流程。
- 私人浴室是两人的核心场域；诊所清洁区、治疗区及任何职业空间不得调用。
- 洗浴同意不自动包含后续亲密，浴室内也保留明确停止与离开条件。

### 4. 放松与感官照料

- 肩颈、背部、手臂与腿部按摩，以低速、连续触感逐步卸下晏林的控场习惯。
- 可使用热敷、柔软遮眼、身体安全按摩油和低亮度暖光。
- 温婉通过“轻一点、保持、继续还是停”要求晏林开口反馈，不把沉默当满意。
- 她享受的是让他真实放松并被自己看见，而不是完成一套技术项目。

### 5. 主动取悦

- 温婉可以明确提出当晚先以晏林的感受为主，让他只负责接受与反馈。
- 她的身体主动必须清楚，不靠羞怯默认，也不靠晏林自行理解她“应该愿意”。
- 晏林的真实失守是这一段的关系奖励：他不再维持老司机式从容，也不把反馈包装成对她的指导。
- 完整身体流程留给具体 B 场设计，本线只登记主动权、确认点与场后后果。

### 6. 睡眠与晨间延续

- 场后由温婉完成或协助完成清洁、补水、保暖和入睡安排。
- 她不在晏林睡着后悄然退出，也不要求自己整夜保持照顾者状态。
- 晨间可以延续整理衣物、头发、早餐与出门准备，但双方共同处理昨夜用品。
- 每次调用至少留下一个生活权限变化：固定睡衣、备用洗漱用品、可留宿时段、晨间叫醒方式或下一次约定。

## 三、三种主要场型

### A. 纯照料夜

> 迎接 → 卸载 → 沐浴／按摩 → 陪睡 → 晨间回到现实。

- 可以只做强 A 场或低浓度兑现。
- 适合压力后回稳，但不能用晏林虚弱换取关系确认。
- 关系变化落在他第一次主动承认“今晚不想自己撑”。

### B. 专属侍奉夜

> 温婉主动邀请 → 明确当晚范围 → 更衣／沐浴 → 感官照料 → 主动取悦 → 晏林彻底卸力 → 场后与晨间延续。

- 这是本线最完整、最具辨识度的单人兑现模式。
- “完全”只指当晚体验链完整，不代表晏林获得对温婉的整体支配权。
- 每次复现必须改变一项主动权、照料方式或生活权限，不能只换地点重复流程。

### C. 照料权反转夜

> 温婉准备继续服务 → 晏林发现她仍在压住自己的需要 → 暂停并询问 → 温婉明确说出想要什么 → 晏林逐项复现她曾给予的照料 → 温婉只负责接受。

- 这是温婉成长轴的必需回路，不是附加彩蛋。
- 她最难交出的不是身体，而是“我现在不照顾任何人”的许可。
- 后续稳定期可以从单向反转发展成双向轮换，由当晚需要而非固定性别决定谁照料谁。

## 四、专属仪式与道具

### 固定仪式

- 一句总入口确认：“今晚想自己来，还是交给我？”
- 一件只在她住处使用的柔软睡衣或浴袍。
- 一条专属暖色浴巾，承担沐浴与场后照料的记忆回收。
- 一盏由温婉决定何时调暗、但双方都能随时打开的床边灯。
- 一个临时收纳手机与钥匙的小盒，放入前必须得到同意，任何一方可随时取回。
- 三档口令：保持、放慢、停止；身体反应不能替代口头或动作反馈。

### 首选器物

- 身体安全的按摩油、热敷用品与柔软毛巾。
- 低温、明确标注可用于皮肤的按摩蜡烛；先试温、避开敏感与破损皮肤，不以普通香薰蜡烛替代。
- 柔软遮眼：作用是让晏林暂停观察，不制造恐惧或取消表达通道。
- 支撑枕、床边靠垫与可调暖光。
- 后期可少量使用细调型感官工具，但不建立实验、挑战或器械收集路线。

### 明确避让

- 不使用 Cos、女仆装、主人称呼或角色扮演证明服侍感，避免复制安若曦／江岚标签。
- 不使用诊所设备、宠物用品、医疗用品或职业制服进入亲密玩法。
- 不以惩罚、任务考核、家务评分或服务不合格制造刺激。
- 不发展二十四小时服从协议、永久随叫随到或生活劳动单边化。
- 不使用录像、私拍与系列回看作为核心奖励。

## 六、阶段推进

### 第一阶段：只让生活权限增长

- 通过安安、诊所与工作室附近日常建立低频熟悉。
- 可以出现递水、顺路送还、短暂留饭等生活照料，不情色化正常职业接触。
- 每次出场至少改变一项现实权限，但不告白、不留宿、不身体兑现。

### 第二至第三阶段：看见疲惫但不借机越线

- 温婉看见晏林可靠外壳下的疲惫，能够提供“今晚别独处”的现实落点。
- 第一次请求留下只完成脆弱交付与陪伴，不以危机、失眠或精神异常推动完整亲密。
- 晏林也要在没有安安和事件时主动见她，证明他要的是温婉本人。

### 第四阶段：确认前置

- 温婉知道关系复杂度与危险，开始判断自己是否愿意进入。
- 可以出现第一次清醒状态下的私人邀约、沐浴前停顿或强 A 场，但不抢正式关系确认。
- 她需要确认晏林不会因为喜欢她的稳定，就默认她承担所有人的照料。

### 第五阶段：正式确认与首次完整兑现

- 温婉主动说出“我留下是因为我想要你”，独立确认关系。
- 首次完整兑现采用低复杂度专属侍奉夜：只启用迎接、沐浴／按摩、主动取悦与场后照料，不一次展示全部仪式和器物。
- 晏林必须在次日用现实行动回应，不把她的主动收成一次性的温柔奖励。

### 尾声段：反转与稳定复现

- 温婉能够主动安排完整一夜，也能明确说今晚不想照顾任何人。
- 至少完成一次照料权反转，由晏林完整接过她曾给予的步骤。
- 成阵后保留温婉独占夜，不因多人关系稳定便把她的照料自动开放给内圈其他人。
- 后期以旧仪式的新含义维持新鲜感，不靠持续增加器具。

## 七、预留场景单元

建议预留 **9—11 个场景单元**，其中 **4—6 个完整兑现位**；数量是弹性上限，重叠功能应合并。

1. 初识期的现实照料：只建立可靠与生活权限，不情色化诊疗。
2. 晏林状态异常后，温婉第一次说“今晚别走”，只完成陪伴。
3. 无危机状态下第一次主动邀请，证明她想留的是晏林本人。
4. 正式确认后的第一次完整兑现：低复杂度专属侍奉夜。
5. 第一次沐浴服侍：晏林停止自己完成每一步，学会接受。
6. 第一次遮眼与感官照料：晏林失去观察权，只能真实反馈。
7. 第一次完整晨间延续：固定睡衣或洗漱用品进入生活。
8. 温婉第一次主动说出“现在轮到你照顾我”，完成照料权反转。
9. 晏林主动复现她的整套照料，温婉第一次整晚不承担收尾。
10. 成阵后的独占夜：她明确索要只属于两人的整晚，不负责安顿其他关系者。
11. 稳定期无道具夜：只回收入口问句、暖灯和晨间位置，证明关系不依赖器物成立。

## 八、场后复盘

- 当晚只完成身体舒适、补水、清洁、保暖和情绪回稳，不强迫立即总结全部感受。
- 次日再确认哪些步骤喜欢、哪些太多、下一次希望保留或删除什么。
- 晏林必须询问温婉是否因持续照料而疲惫，以及她是否真正得到满足。
- 温婉可以回答“这次我只想给”，但连续多次单向服务时必须触发关系检查，不能用人设替代她的需求。
- 复盘结果应改变下一次顺序、角色或生活权限，不写成技术评分表。

## 十、多人关系边界

## 十三、拆章前待定

- 第一次“今晚别走”与第一次清醒主动邀约之间间隔几个生活场景。
- 首次完整兑现采用温婉住处还是安静酒店，以及是否将沐浴与第一次 B 场合并。
- 第一次遮眼由谁提出，是否只做强 A 场。
- 温婉第一次要求回报发生在首次完整兑现当晚，还是在后续独立反转场。
- 成阵后第一次独占夜发生在什么共同生活矛盾之后。
- 固定仪式最终保留暖色浴巾、床边灯、睡衣还是手机收纳盒中的哪两至三件。

这些参数不阻塞本线成立；未定内容不得提前写成正式人物偏好或定稿事件。
