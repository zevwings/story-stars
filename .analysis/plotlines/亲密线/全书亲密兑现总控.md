# 全书亲密兑现总控

> 本页是亲密线路的导航与去重总控，不复制 [全书阶段总览](../../blueprints/全书阶段总览.md) 中的总数、阶段比例与人物配额。各阶段真正发生什么，仍由阶段编排、人物主线与具体章节决定。

## 作用

- 登记已形成多场、跨阶段回路的亲密线。
- 确保每条线都有独立的观看方式、主动权结构、场后余波与关系变化。
- 防止多人亲密滑成固定大团建，也防止各线只换角色、服装或地点重复同一流程。

## 后续增线标准

只有同时满足以下条件，才新建独立亲密线：

1. 至少需要跨两个阶段或承接三个以上场景。
2. 存在明确且不可被其他人复制的亲密回路。
3. 每次调用能够改变关系、知情、嫉妒、陪伴顺序、暂停权或共同生活边界中的至少一项。
4. 已有人物主线无法在不被亲密场淹没的情况下承接。

未达到该门槛的单场或两场设想，直接在阶段编排或具体 `intimacy/chapters/` 场景卡中承接，不继续增目录。
