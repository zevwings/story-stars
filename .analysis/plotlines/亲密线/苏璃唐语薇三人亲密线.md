# 苏璃·唐语薇·晏林三人亲密线

> 本页承接苏璃、唐语薇与晏林跨阶段、可反复调用的亲密回路，属于施工层，不是正式正典。三人的定情、嫉妒、秘密交叉和首次三方协商，以 [苏璃和唐语薇线](../主线/苏璃和唐语薇线.md) 为准；具体 A／B 场须在章号确定后进入 `intimacy/chapters/chXXX/scene_NN/`。

## 一、线路定位

本线采用“一条总线、三条回路”，不拆成三个平级文件：

1. **苏璃／晏林：设局与看穿。**
2. **唐语薇／晏林：实验与控制权翻转。**
3. **苏璃／唐语薇／晏林：规则、行动与收束在三人间流转。**

三条回路的启用顺序、知情后果和陪伴边界高度绑定。三人共同亲密不能吞掉两条独立关系，也不能把苏璃写成许可者、唐语薇写成闺蜜附带、晏林写成只负责享受两种主动的人。

线路总命题是：

> 苏璃习惯给真心留退路，唐语薇习惯抢回行动的主动；晏林必须分别接住她们，又不能用“保护”替任何人决定。

## 二、共同启用门槛

### 苏璃独立回路

- 苏璃与晏林先完成明确关系确认。
- 第一次身体兑现发生在两人双人旅行中，定情先于身体层。
- 酒精和情绪权柄不推动欲望、不削弱判断。

### 唐语薇独立回路

- 唐语薇与晏林先通过不依附苏璃的独立事件建立吸引与信任。
- 苏璃已真实知情，但知情不构成许可或追求通行证。
- 唐语薇主动要求明确关系；第一次独立兑现不与三人场合并。

### 三人共同回路

- 苏璃、唐语薇与晏林三段关系各自成立。
- 三人已谈过嫉妒、偏心、陪伴顺序、暂停方式和是否接受共同身体层。
- 首次三人场不是为了证明苏璃大度、唐语薇敢抢或晏林有能力同时满足所有人。
- 每次共同场重新确认参与意愿；稳定关系不构成永久多人授权。

## 三、回路 A：苏璃与晏林

### 核心结构

> 苏璃设局 → 晏林入局但不被带走 → 点破真假 → 控制权回摆 → 玩笑停下 → 真心被接住 → 事后嘴硬依赖。

### 苏璃的主动

- 她用话、装束、问答或小道具设置两层意思，给自己保留“我没这么说”的退路。
- 她可以先掌握遥控、规则或眼罩，也可以故意露出能被识破的破绽。
- 她不是每局必输；有些场次必须让晏林真的被她绕进去，下一次反用才有分量。

### 晏林的回应

- 他可以识破局，但不能以公开拆穿、羞辱或专业分析夺走她的体面。
- 真正有效的是指出哪句话在演、哪一次停顿是真的，并把是否继续交还给她。
- 他不能因为熟悉苏璃的套路便替代当次确认。

### 道具系统：苏璃游戏箱

- 真话／假话卡、欲望问答卡。
- 骰子、抽签盒、回合计时器。
- 可交换控制权的遥控器与单件感官工具。
- 丝带、眼罩、镜子、密封信封。
- 盲品、气味与声音辨认工具。
- 经当次同意留下的一句真话录音；不建立录像收藏标签。

### 不可替代的关系变化

- 她从“随时能退”推进到主动承认这次没有玩笑。
- 晏林从看破她推进到允许自己也被她成功设局。
- 场后留下一个只有两人懂的词、规则或未拆穿的真话，成为下一次回收点。

## 四、回路 B：唐语薇与晏林

### 核心结构

> 唐语薇提出新玩法 → 两人共同检查 → 她抢开局 → 晏林认真接招并反制 → 她痛快认下这一轮 → 疯完后黏住 → 预告下轮抢回来。

### 唐语薇的主动

- 新东西必须先经过她的好奇、研究和明确提出，不由晏林把她打扮或绑入一个既定位置。
- 她可以抢控制器、先给晏林试、主动改变回合规则。
- 她的交付是确认对方接得住以后自愿扑进去，不是被驯服或惩罚后的服从。

### 晏林的回应

- 他负责和她共同检查材质、档位、快拆与停止方式，但不能借“安全”包办她的选择。
- 反制必须回应她的宣战，而不是把她的活力压灭。
- 他要允许唐语薇真正赢下部分回合，也要接受自己成为实验对象。

### 道具系统：唐语薇实验箱

- 模块化震动、气压／吸附与温感工具。
- 双遥控或双方分别控制不同功能的设备。
- 需要两人共同确认的“双钥匙启动”。
- 软皮腕带、踝带、腿环、眼罩与降噪耳机。
- 玻璃／金属感官工具与支撑垫。
- 挑战卡、回合计时器与安全检查表。

### 职业设备禁区

- 不把工业束线带、维修胶带、润滑油、清洗剂、电动工具、车辆升降或固定设备转作身体用品。

### 不可替代的关系变化

- 她确认晏林喜欢的是自己的热、闹和好奇，不是对她的纵容。
- 晏林学会在不替她决定的前提下接住高强度主动。
- 每轮复盘都留下一个下一次可以翻转的输赢结果，而不是只追求新器具。

## 五、苏璃与唐语薇的横向关系

### 闺蜜关系必须独立存在

- 两人保留不经过晏林的骑行、夜宵、争执、护短与修复。
- 苏璃可以最先看懂唐语薇动心，却不能替她说、替她退或把她送入关系。
- 唐语薇可以逼苏璃停止玩笑，却不能把“说直”当成对苏璃的道德要求。

- 唐语薇为了苏璃主动后退。
- 苏璃发现后并不感激，因为这同样是在替她决定她能否承受嫉妒。
- 唐语薇逼苏璃承认介意；苏璃逼唐语薇承认喜欢。
- 两人先修复姐妹关系，再讨论是否愿意共同进入三人身体层。

### 女女亲密边界

- 两人的闺蜜与共同关系不自动生成女女身体关系。
- 若未来发展苏璃／唐语薇独立身体吸引，必须另有双人表达、确认和可撤回过程；不能把三人场中的接触倒推成独立恋爱事实。

## 六、回路 C：三人共同亲密

### 核心结构

> 苏璃设定表面规则 → 唐语薇把规则推进行动 → 晏林分别确认并接住 → 控制权在三人间流转 → 偏心或嫉妒显形 → 当前回合暂停／调整 → 场后重新协商。

### 三人功能分工

- **苏璃**：制造局、语言回合和真假张力，但不能永久担任主持者。
- **唐语薇**：点火、试验和打破僵局，但不能用热烈把苏璃逼成旁观者。
- **晏林**：负责分别回应、收束风险与安顿余波，但不能以控场者身份垄断所有主动。

| 控制牌 | 功能 | 默认倾向但不固定归属 |
| --- | --- | --- |
| 设局牌 | 决定本轮表面规则、问题或隐藏条件 | 苏璃 |
| 实验牌 | 决定启用哪一种工具与首先作用于谁 | 唐语薇 |
| 收束牌 | 降速、暂停、重新分配注意力或结束当前回合 | 晏林 |

- 三张牌在稳定期必须允许交换。
- 持有某张牌只获得该项功能，不获得对其他两人的整体处置权。
- 一人要求暂停时，当前共同回合整体暂停，不能只把她移出后继续。

### 共同道具边界

- 一场只设一个主要机制：控制权轮转、感官限制、问答反馈或单件遥控。

- 晏林不使用笼统的“你们”；关键确认必须分别叫名字、分别得到回应。

## 七、阶段推进

### 阶段一：苏璃先成立

- 车行和酒吧三人日常只负责既有熟悉与轻微偏移。

### 阶段四：回路稳定与互换
