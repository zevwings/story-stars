# 反派欲望与压迫线

> 本页承接核心反派跨阶段的正向恋爱、交易性亲密、正向转黑暗与纯压迫回路，属于施工层，不是正式正典。核心反派的行动、阵营与阶段交锋见[反派主线](../主线/反派主线.md)；精神侵蚀与星辰之印机制见[黑暗契约者线](../副线/黑暗契约者线.md)。
>
> 本页不承载某章的完整身体流程。章号与事件确定后，正向 B 场进入 `intimacy/chapters/chXXX/scene_NN/` 的正向设计流程；黑暗压迫场在已有 `design.a.md` 后进入同场 `design.b.dark.md`。所有参与者必须年满十八岁。

## 一、线路定位

反派身份不自动把其所有亲密判定为黑暗场。每段关系按当次是否知情、自愿、可拒绝、可停止、可撤回来定性。

本线保留两条色带：

1. **正向色带**：核心恋人、共犯恋人、攀附者与部分利益对象，可以拥有真实恋爱、交易性亲密与正向 B 场。
2. **黑暗色带**：当撤回被否认，或药物、精神侵蚀、身份与资源使对象无法离开，关系转入黑暗 B／压迫场。

线路总命题：

> 反派可以真诚地爱少数人，也完全懂得怎样尊重拒绝；真正的黑暗在于，只承认少数人配拥有这种边界。

## 二、对象结构

| 类型 | 关系逻辑 | 定性 | 核心变化 |
| --- | --- | --- | --- |
| 核心恋人／共犯 | 真爱、平等、完全知情 | 正向恋爱＋正向／深色 B | 她是普通人，知道反派所做的一切，仍主动继续共犯并承担独立责任 |
| 攀附者 | 主动以身体、陪伴或关系换资源与身份 | 自愿交易型 B | 高估自己的特殊性，随利益变化重估关系 |
| 利益对象 | 双方利用，以亲密交换情报、权限、掩护或把柄 | 交易型 B，按当次信息判断 | 床上与场后都发生筹码转移 |
| 救助依赖者 | 前期被真实帮助并自愿靠近，后期撤回被拒绝 | 正向 B 转黑暗 B | “曾经同意”被改写成永久处置权 |
| 纯压迫对象 | 没有恋爱成立，被当作控制、享乐或实验对象 | 黑暗 B／压迫场 | 受害、反抗、证据、恢复与清算 |

猎奇实验对象不单独设长期关系位，可作为纯压迫对象的一种案件形态接入星辰之印或药剂线。

## 三、正向回路 A：核心恋人／共犯

核心恋人只设一位。她是普通人，不是星契者；她知道反派的篡心、节点网络、黑色产业与受害后果，并在完全知情的前提下仍继续帮助他。她不是反派的良心、赎罪券、被篡心的受害者或被挟持软肋，而是主动共犯。

两人的恋爱与身体关系可以真实成立；反派对她遵守知情、自愿与可撤回，恰好证明他理解边界，只是选择不把同等边界给予其他人。

### 核心回路

> 并肩处理事务 → 私下卸下角色 → 彼此试探真实立场 → 明确愿意与停止方式 → 正向 B → 场后继续谈利益或分歧 → 终局选择重新定义关系。

### 正向边界

- 双方清醒，能够说不、暂停、离开并重新协商。
- 可以使用主导、臣服、束缚、羞耻或危险感等深色情趣，但不能依赖职业权力、药物、星印、恐惧或求生压力。
- 核心恋人可以是唯一能命令反派停下的人，但这不证明反派尊重所有人的边界。
- 反派可以照顾、道歉、让步、吃醋和害怕失去；不能由此冲淡受害者账单。
- 核心恋人若参与犯罪，需要承担自己的责任，不因爱情自动转为无辜证人。

### 关系变化

- 反派并非不会爱、不会停，而是选择性给予完整人格待遇。
- 核心恋人必须面对“我得到的尊重，是否建立在别人被剥夺尊重之上”。
- 终局时她不倒戈成为无责污点证人，而是继续站在反派一边，并就自己参与的遮掩、协助与维持行为接受追责。

## 四、正向回路 B：攀附者

攀附者主动接近反派，希望换取地位、圈层、星表资料、药剂、保护或被选中的优越感。她知道关系带有交换，不按受害者书写。

### 核心回路

> 她展示价值并主动靠近 → 反派评估而非追求 → 双方说清最低交换 → 自愿 B → 她以为得到特殊位置 → 场后资源兑现但关系权限没有同步扩大。

### 写作重点

- 她不是愚蠢拜金工具人，应有现实目标与可计算收益。
- 她可以享受身体、权力与资源，不因交易性质失去主体性。
- 反派履行明确承诺，体现危险来自规则选择，而不是处处赖账。
- 她的误判是把一次被选择当成长期特殊性，不是把强迫误认成爱情。
- 后续可以转为利益对象、外围共犯或主动退出，不必必然受害化。

## 五、交易回路：利益对象

双方都在利用彼此。亲密既是欲望，也是谈判、试探与互留把柄的空间。

可交换星表与家族情报、药剂渠道、目标介绍、集团权限、诊疗记录、公开身份掩护、证据延迟或交易切割。

### 核心回路

> 带着条件见面 → 确认本次关系与交换边界 → B 场中继续试探 → 一方交出部分筹码 → 场后核验承诺 → 新把柄或新债务形成。

### 边界

- 信息不完全不自动等于身体非自愿；必须判断被隐瞒内容是否直接影响当次安全、对象身份或同意基础。
- 若以曝光、伤害亲友、断药、职业毁灭或无法退出的债务逼迫继续，从该次起转入黑暗线。
- 每场必须改变现实利益，不能只用“互相利用”包装重复身体场。
- 利益对象可以拒绝下一次交易，旧交易不构成永久身体权限。

## 六、转黑回路：救助依赖者

这是本线最重要的贯穿型受害关系。反派前期确实帮助过她，她也可以真心爱过反派并自愿进入正向 B 场。

可用前史包括：帮助她压住精神失衡；从家族、黑市债务或另一段控制关系中带她离开；提供住所、药物、工作和安全；在合法服务失效时接住她最狼狈的状态。

黑暗不是从“反派身份曝光”自动开始，而从她明确暂停、拒绝、离开或撤回授权后，反派仍继续介入开始。

反派的自我合理化可以是：她的拒绝只是病情反复；她被外界误导；离开后一定会再次失控；过去的主动证明她真正需要继续；既然是自己救回来的，就有责任替她完成调整。

### 选择权曲线

1. **真实救助**：她拥有选择，关系与 B 场成立。
2. **依赖形成**：住所、药物、工作、秘密与情感集中到反派一人。
3. **第一次撤回**：要求暂停精神介入、亲密或共同生活。
4. **撤回被解释掉**：反派不承认拒绝具有真实意义。
5. **控制加深**：药物、记录、星印、经济与关系隔离共同压缩出口。
6. **留下证据**：利用反派对“她不会真的离开”的自信制造误导、求救或记录。
7. **清算与恢复**：她的选择参与破局，后续进入治疗、证言、生活重建与长期关系后果。

### 转黑后的硬边界

- 身体反应必须注明来自药物、精神干扰、恐惧应激、自然生理或其他明确来源，不能写成心理愿意。
- 表面配合可以是减伤、保护他人、拖延、保留证据或维持清醒，不能被叙述认可为迎合。
- 过去的爱情可以造成留恋、混乱与哀伤，但不能取消当下拒绝。
- 救援不能替她完成全部行动；至少一个关键证据、误导、拒绝或破坏由她自己完成。

## 七、黑暗回路：压迫享乐与实验

### 反派快感来源

- 把明确拒绝重新解释成服从。
- 让对象怀疑自己的记忆、欲望与身体。
- 迫使对方说出并不相信的话。
- 把僵住、失声、求饶或求生配合视为战利品。
- 证明人格、依恋和退出权可以被重新编排。

### 双声部铁律

| 同一细节 | 受害方声部 | 压迫方声部 |
| --- | --- | --- |
| 僵住 | 恐惧、麻木或判断断裂 | 服从已经形成 |
| 眼泪 | 痛苦、羞耻或求生失控 | 自己终于抵达她最深处 |
| 表面配合 | 减少伤害、拖延、保护他人 | 她承认自己离不开 |
| 身体反应 | 药性、精神干扰或自然生理 | 身体证明反派才是对的 |
| 求饶或失声 | 崩溃、策略或无法继续表达 | 控制完成的标志 |

固定判断：快感是压迫方的，账单是受害方的；文本不认同压迫方；受害方痛苦必须与压迫方快感同重量在场。

### 后果接口

- 身体损伤、药物依赖、睡眠和进食障碍。
- 记忆不可信、关系响应异常、持续指令或星印残留。
- 对触碰、医疗、咨询、睡眠和亲密的长期困难。
- 病历、交易记录、药剂批次、星表编号或地点线索。
- 对反派核心恋人、共犯与资源后台的知情冲击。
- 管理局收容、疗养院复核、合作医院治疗与工作室心理支持。

黑暗场结束不能立刻转入甜蜜救援或正向 B。救援、医疗、询问、证据保全、日常恢复与重新建立身体自主权需要独立场景承接。

## 八、多向关系之间的知情

### 核心恋人

- 她知道攀附者、利益对象、救助依赖者和压迫对象的存在。
- 她知道哪些人已经撤回、哪些关系已经转为强制控制，不得以「不知道具体后果」保留无辜余地。
- 她本人不被篡心，对反派的支持与共犯是自己的选择。

### 攀附者与利益对象

- 可以知道核心恋人存在，不自动拥有挑战或取代她的位置。
- 交易关系中的排他、保密和公开身份必须分别约定。
- 发现黑暗受害者后，必须选择继续合作、切割、利用证据或帮助掩盖。

### 依赖者与压迫对象

- 反派可能用核心恋人的待遇证明“自己不是施暴者”，文本不能接受这种比较。
- 受害者不需要证明反派从未爱过任何人，才能证明自己遭受了侵害。
- 核心恋人被尊重，恰好证明反派知道怎样尊重边界。

## 九、场景总量与阶段分配

| 场型 | 建议数量 | 功能 |
| --- | ---: | --- |
| 核心恋人正向 B | 2—4 | 建立真实爱情、边界反差与终局选择 |
| 攀附／利益型 B | 2—3 | 资源交换、身份与把柄变化 |
| 救助依赖者正向 B | 1—2 | 证明前期关系曾真实成立 |
| 黑暗压迫 B | 2—4 | 撤回失败、控制加深、实验或终局失败控制 |

数量是施工预留，不是配额。阶段建议：

1. **阶段一**：只留攀附或利益关系影子，不展开黑暗 B。
2. **阶段二**：核心恋人首次正向 B；救助依赖者仍处于真实帮助与自愿关系。
3. **阶段三**：利益型 B 推进唐家／黑市接口；救助依赖者第一次撤回失败，首个黑暗场成立。
4. **阶段四**：控制加深或实验型黑暗场；核心恋人继续协助遮掩与节点维持。
5. **阶段五**：终局失败控制与证据清算；核心恋人仍继续共犯，正向恋爱不能用 B 场跳过责任结算。

## 十、线路接口

- **反派主线**：核心反派身份、阶段动作、与晏林交锋及终局清算。
- **黑暗契约者线**：精神侵蚀、星辰之印、药剂和连接条件。
- **黑市家族线**：资源、记录、场所与善后；亲密对象不是家族提供的工具。
- **管理局协查线／疗养院线**：证据、收容、复核与回流。
- **工作室日常线／合作医院线**：心理支持、医疗转介与长期恢复。
- **全书亲密兑现总控**：控制黑暗场比例，确保反派亲密不压过男主与女主的正向主色。

## 十一、不得写偏

- 不因反派有正向恋爱和正向 B 场而洗白黑暗行为。
- 不把所有关系对象都写成受害者，也不把所有交易关系美化成爱情。
- 不把核心恋人写成唯一完整女性，其余对象全部工具化。
- 不让攀附者因为追求利益而“活该”遭受后续侵害。
- 不让救助依赖者因曾经主动、爱过或享受过身体关系而失去撤回权。
- 不把身体反应写成愿意，不把求生配合写成迎合。
- 不从压迫场直接转入恋爱、原谅或正向 B；新的自愿只能从选择权恢复后重新建立。
- 不让七位女主进入反派男性的情色兑现；完整场使用成年配角。
- 不在本施工线提前写完整身体流程；具体正向／黑暗场按章进入对应设计卡。

## 十二、待确认

1. 核心恋人的姓名、身份、是否结契，以及知情共犯或逐层知情的路径。
2. 核心恋人与反派的关系是否排他，如何看待攀附者与利益对象。
3. 核心恋人的正向亲密标签、主导关系与明确禁区。
4. 攀附者与利益对象是否分设，还是由同一人从攀附转为稳定交易伙伴。
5. 救助依赖者的姓名、职业、原始困境与最初真实获救事件。
6. 救助依赖者第一次明确撤回的内容：精神介入、共同生活、药物、身体关系还是全部关系。
7. 首个黑暗场的选择权削弱方式、身体失控来源与场后证据。
8. 是否保留独立实验型受害者，还是将星辰之印实验并入救助依赖者线。
9. 核心恋人何时知道受害者已经撤回，以及她作出的独立选择。
10. 终局失败控制由受害者留下哪一种证据、误导或破坏完成关键转折。
