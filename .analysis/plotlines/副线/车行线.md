# 车行线

> 本页是车行、骑行圈、机械职业、责任链和唐语薇日常场的施工设计，不是正式正典。角色事实以 `BIBLE.md`、`story/synopsis.md` 与 [苏璃和唐语薇线](../主线/苏璃和唐语薇线.md) 为准。

## 剧情定位

车行线负责把“手上活”和“责任归属”写出来。

唐语薇不是家族办公室里被安置的人，而是在工位、工具和客户问题里建立自我位置的人。车行既是她的职业落点，也是唐家逻辑、骑行圈关系和现实信用的交叉面。

## 主要功能

- 给唐语薇提供独立于唐家的职业场。
- 给苏璃、晏林和唐语薇的日常交集提供现实接口。
- 给客户索赔、责任边界、改装争议和师徒关系提供剧情材料。
- 给唐家外围压力留下“不靠直说家族，也能施压”的入口。

## 角色位

| 角色位 | 功能 | 备注 |
| --- | --- | --- |
| 车行老板 / 师傅 | 工位、技术判断、接单范围 | 可承担职业权威 |
| 资深技师 | 复核、补位、维修规范 | 可做唐语薇成长镜像 |
| 客户 | 维修需求、投诉、口碑传播 | 现实压力来源 |
| 骑友 / 试车对象 | 圈层温度、日常交叉 | 承接骑行半径 |
| 配件商 / 供应链 | 价格、时效、质量风险 | 可与黑市边缘相连 |

## 可承载的关系面

- **正向亲密**：一起试车、递工具、擦掉机油、保养后顺手送一程、把对方安全送回去。
- **黑暗亲密**：借师徒关系压人、把事故责任往下推、用资源和名声换沉默。

唐语薇的关系感必须建立在“我让你帮我看清楚，不是让你替我选”的边界上。

## 阶段骨架

### 一、车行日常

维修、保养、客户解释、试车和夜间收工构成基本节奏。

### 二、独立事件

客户纠纷、改装责任、资深技师失误、商业投诉或唐家外围人物进入车行，让她面对独立选择。

### 三、外部压力

唐家不一定直接现身，但它可以通过资源、名册、解释权和“你该回去了”形成压力。

### 四、后续扩展

若后续接入黑市、异常补剂或唐家授权链，车行线可以作为追责节点保留。

## 不得写偏

- 不把车行写成豪门产业中心。
- 不把唐语薇写成靠家族堆出来的技术人物。
- 不让修车变成“顺手谈恋爱”的背景板。
- 不把客户纠纷直接拔高成家族战争。
- 不让车行职业现实被星契替代。

## 待确认

1. 车行正式名称、街区与经营规模。
2. 唐语薇的师傅、资深技师与长期客户结构。
3. 车行是否有固定骑行圈或改装圈合作。
4. 唐家外围压力先从哪类现实纠纷切入。
