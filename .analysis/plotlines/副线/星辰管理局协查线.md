# 星辰管理局协查线

> 本页是剧情线施工层，不是正式故事真源。制度真源见 `worldbuilding/factions/星辰管理局.md`；晏林的能力、登记与暴露边界见 `characters/major/晏林/契约.md`。

## 定位

星辰管理局协查线是官方线，也是星辰之力进入现代社会秩序后的制度线。它不按正邪阵营展开，而围绕登记、保密、协查、公共安全、失控处置与解释权争夺推进。

晏林与管理局是松散有偿协查关系，不是从属关系。管理局可以在特定案件中委托晏林提供梦境安抚、梦境复盘和受害者询问辅助；晏林只以登记在册的梦境契约者身份协助，不主动暴露幻境、暗伴星、截断、逆流与双子星结构。

## 核心功能

- 让星辰之力从个人能力扩展为社会制度问题：谁登记、谁监管、谁处置、谁解释。
- 给后续独立案件提供制度后果：越界行为进入正式流程后，管理局会介入、复核、善后或追查。
- 增加晏林的长期压力：他越能解决精神类案件，管理局越会怀疑他的登记资料不完整。
- 区分管理局与星辰家族：管理局维护公共秩序，星辰家族争夺星表资源与解释权；二者可以合作、竞争，也可以互相掣肘。

## 管理层决策接口

沈宁是星辰管理局副局长、局内二把手，负责把分散个案中的能力边界疑点收束为管理层需要承担的风险与决策。日常委托、材料收取和现场协调仍由经办人负责，独立复评仍由复评人员主持；沈宁不兼任所有岗位，也不把管理局变成主角阵营的靠山。

她的决策原则是：允许陈述和拒绝，但不因体谅抹掉记录、绕过复评或替任何一方作无证据判断。她同时拒绝两种偷懒方案：无证据地把晏林定为危险人物，或因其有用而长期默许登记残缺。

### 沈宁的阶段动作

- **阶段二**：只在材料背后出现，要求继续合作但保留能力疑点，并确认林芷兰有权拒绝无授权治疗。
- **阶段三**：多案材料出现稳定差异后，与晏林进行一次正式但非审讯式会面，只索取与公共安全和协查风险有关的说明。
- **阶段四**：不亲自主持自己批准过的协查决定的复评；既不删减不利记录，也不允许把推测写成违法事实。
- **阶段五**：为“个案受控合作＋必要更正备案＋分层公开”承担签字责任；完整双子星结构不进入普通档案。

### 能力与生活边界

- 沈宁的单星契约星辰持有收束向的静默权柄，可以近距离静默域收束向外泄露的精神信号；不物理隐身、不消除监控或痕迹、不切断已建立的稳定连接。
- 她不进入晏林后宫，不与晏林发生暧昧、恋爱或身体关系。
- 她的长期伴侣徐清不在其直接管理链内，不因伴侣身份获得额外权限，也不参与她负责个案的经办、复评或决策。

## 阶段一边界与后续入口

安若曦酒吧事件不报警、不进入管理局检测，也不形成警方、管理局或集团正式事件记录。管理局不因本案知道受雇欲望契约者、晏林的截断能力或登记资料可能不完整。

晏林与管理局的首个个案委托关系由后续独立精神污染案件建立。届时管理局先确认其登记范围内的梦境安抚、梦境复盘与询问辅助具有协查价值，再由该案自身的处置结果逐步形成能力边界疑点；不得回接安若曦酒吧事件作为触发证据。

## 推进节奏

1. **登记内协查**：晏林只使用梦境方向能力，提供受害者安抚、梦境复盘和询问辅助。
2. **能力边界疑点**：管理局发现他的协查效果过于稳定，或现场结果超出普通梦境权柄解释范围。
3. **复评压力**：管理局以合规流程发起问询、补充说明或复评，晏林需要在配合与隐藏之间周旋。
4. **解释权冲突**：星辰家族、黑市或黑暗契约者利用管理局规则漏洞，推动晏林和女主阵营意识到旧秩序不能只靠管控维持。

## 高端商务会所联合任务

- 阶段二会所异常药剂事件、重复自我解释与撤回陈述，使普通警察怀疑强迫陪侍、场外交易、药物控制与隐私侵害，管理局则怀疑存在未登记精神能力。
- 阶段三由普通警察负责现实犯罪调查、外围安全与紧急介入，管理局负责未登记能力、精神侵蚀与外泄判断；完整个案见[高端商务会所线](高端商务会所线.md)。
- 晏林以松散有偿协查者身份随真实商务入口进入，只负责精神异常识别、本人授权后的连接判断与必要止损，不取得搜查、抓捕、审讯或内部档案权限。
- 场外服务女孩被带离后明确求救并授权转交信息；晏林另行取得连接操作授权后截断持续回响。经理重新连接时暴露未登记能力，任务同时加深管理局对晏林登记不完整的怀疑。
- 官方不能为取证故意放任侵害发生；现场出现明确安全风险时，保护与医疗优先于继续观察。

## 写作边界

- 管理局不是主角官方靠山，也不是天然反派。
- 协查是个案委托和有偿协助，不给晏林执法权、内部档案权限或官方身份豁免。
- 梦境复盘不能直接成为铁证，只能作为线索、心理材料或证言辅助。
- 晏林不能因为协查而公开登记幻境、暗伴星、截断、逆流和双子星真相。
- 管理局处理的是危害社会、精神侵害、非法控制、恶性外泄和持续逃避监管，不因星辰契约存在本身惩罚普通契约者。

## 终局合作口径

- 晏林与管理局继续保持个人专家式的个案受控合作，对已被证据确认的登记外精神反制能力作必要更正备案。
- 管理局只在个案经办、复评负责与按任务必需的高风险负责人之间分层知情；普通协查与普通档案人员不掌握详情，完整双子星结构不进入正式档案。
- 工作室不取得官方协查或转介身份，只承担合同、排期、费用、材料分级和保密流转；普通员工不接触案情与契约档案。
- 林芷兰、江岚、安若曦可依各自职业与个案需要分别与管理局合作，但都不承担固定制度接口或代表女主阵营对接管理局。
- 结局不另设规则修订、审查权变化或证据标准改革等制度成果；只写上述合作与公开边界已能稳定运行。

## 协查报价与收入体量

管理局对晏林采用少量个案雇佣，不发固定工资，也不把他写成编外行动队。费用主要按任务难度与风险系数计算：

| 任务类型 | 单案参考价 |
| --- | ---: |
| 普通梦境安抚、复盘 | 3—6 万元 |
| 中风险精神污染案 | 10—20 万元 |
| 高风险现场介入 | 25—50 万元 |
| 极高风险、连续或跨城任务 | 50 万元以上，单独报价 |

基础报价再按现场介入、精神反制可能、跨城、夜间和连续跟进等因素上调风险系数。交通、住宿、设备、医疗、康复、误工与战损不并入服务费，另行结算。

协查失败不等于不付费：只要晏林完成约定服务，基础服务费仍应结算；若任务实际风险高于管理局事前说明，后续补偿和重新议价随之上调。协查频率保持较低，晏林因此拥有一笔可观但不稳定的额外收入。

## 世界观启用点

- 阶段一不为安若曦酒吧事件启用管理局流程；个案委托、补充说明或复评压力从后续独立案件开始。
- 管理局与警方、地方政府、集团内部程序、星辰家族之间的边界，随后续真正进入正式流程的案件逐步拍板。
- 是否设置独立监管风险分级，等后续精神污染或黑暗契约者案件进入管理局流程时再决定。
- 星辰契约观测、精神检测设备只按后续正式案件所需定义可靠性、记录形式和调取权限；不得用设备回溯本次酒吧事件。
