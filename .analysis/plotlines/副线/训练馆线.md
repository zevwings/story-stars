# 训练馆线

> 本页是训练馆、业余搏击、现实身体边界、异常补剂影子和苏晚乔日常场的施工设计，不是正式正典。角色事实以 `BIBLE.md`、`story/synopsis.md` 与 [苏晚乔线](../主线/苏晚乔线.md) 为准。

## 剧情定位

训练馆线负责把“身体层的现实危险”写出来。

星辰契约不提供肉搏强化，所以人物的身体能力、抗压、站位、脱离和受伤都必须回到训练、习惯和现实代价。苏晚乔在这里不是万能战斗女主，而是把“普通人也能变强，但依然会受伤”这件事写实化的人。

## 主要功能

- 为晏林的现实搏斗前置提供长期训练场。
- 给苏晚乔的职业、身体记忆和边界感提供日常落点。
- 给异常补剂、业余赛事和教练伦理提供入口。
- 让黑暗线有物理层后果，不只停在精神层。

## 角色位

| 角色位 | 功能 | 备注 |
| --- | --- | --- |
| 训练馆老板 | 场馆经营、课表、收费、对外协调 | 可以较现实、较务实 |
| 主教练 / 助教 | 技术纠正、对抗安排、学员安全 | 负责真实训练边界 |
| 固定女学员 | 日常镜像、女性自保、友谊支持 | 可承接正向亲密对照 |
| 业余选手 | 赛事、备赛、伤病与压力 | 作为竞争压力来源 |
| 场务 / 器材员 | 设备维护、记录、消毒 | 衬托日常真实感 |

## 可承载的关系面

- **正向亲密**：长期训练建立的信任、递水、处理淤青、搭把手、收工后一起吃饭。
- **黑暗亲密**：强训、越界身体接触、借指导名义控制、异常补剂下的依赖诱导。

训练馆里的“身体接触”必须先回到专业与同意，不允许直接滑成暧昧或压迫。

## 阶段骨架

### 一、日常训练

课表、打点、体能、拉伸、复盘、送回家，构成稳定日常。

### 二、现实兑现

当外部冲突进入物理层，训练中习得的站位、框架、脱离和撤离在现实里被用上。

### 三、异常补剂

训练馆或其供应链里出现不干净的补剂影子，黑市和黑暗契约者线可以从这里接入。

### 四、赛事后果

伤病、停训、赔偿、退赛、排名和口碑，把“强”变成有代价的东西。

## 不得写偏

- 不把训练馆写成地下拳场。
- 不让晏林靠训练直接拥有超常战力。
- 不把伤病写成浪漫勋章。
- 不把异常补剂写成增幅星辰契约的手段。
- 不让教练伦理被“熟了就行”抹掉。
