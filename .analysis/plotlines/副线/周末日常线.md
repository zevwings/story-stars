# 周末日常线

> 本页是周末、休息日、郊游、骑行、夜宵与普通生活场的施工设计，不是正式正典。正式事实以 `BIBLE.md`、`story/synopsis.md`、人物卡、[江州城市回声线](江州城市回声线.md)、[工作室日常线](工作室日常线.md)、[车行线](车行线.md) 与 [苏璃和唐语薇线](../主线/苏璃和唐语薇线.md) 为准。
>
> 本线只负责生活节奏与关系蓄势，不新增女主，不替代任何人物主线，不把未确认地点写成正式地理真源。

## 剧情定位

周末日常线负责让江州不只在案件、咨询、协查和冲突中出现，也在休息日、短途路、饭桌、便利店、江边风、停车场和傍晚回程里出现。

它不是“无事水剧情”。每个日常场都要至少服务一个功能：

1. 让人物从职业和危机身份里松出来。
2. 给关系推进提供低压空间。
3. 让城市空间被身体使用，而不是只被设定描述。
4. 给下一次案件或秘密压力留下对照。

本线命题：**日常不是主线之外的空白，而是亲密真正长根的地方。**

## 场型库

| 场型 | 可写场景 | 功能 | 适合人物 |
| --- | --- | --- | --- |
| 短途骑行 | 近郊山路、江边公路、返程夜风、车行出发前检查 | 关系默契、身体距离、安全边界 | 苏璃、唐语薇、晏林 |
| 郊游散心 | 栖云山、江边步道、近郊观景台、普通农家乐 | 城市呼吸、事件后降压 | 晏林、苏璃、温婉、双胞胎 |
| 夜宵回程 | 酒吧下班、车行收工、训练后补餐 | 成人日常、疲惫和照顾 | 苏璃、唐语薇、苏晚乔、晏林 |
| 买菜做饭 | 工作室闭店后、家里简单饭、临时加菜 | 普通亲密、生活能力和关系边界 | 晏林、苏璃、温婉 |
| 遛狗寄存 | 安安、宠物诊所、江边或小区步道 | 温婉入口、非案件接触 | 晏林、温婉、双胞胎 |
| 漫展 / 活动 | 周末无预约日、轻量出游、私下兴趣 | 角色反差、江岚私下交集 | 晏林、许知夏、许知秋、江岚；详见 [漫展线](漫展线.md) |
| 雨雾天停留 | 雾都天气、临时取消计划、室内聊天 | 情绪停顿、秘密边界 | 各关系线按需 |

## 第一卷用法

### 一、苏璃与晏林短途旅行

苏璃与晏林的两天一夜骑行旅行仍归 [苏璃和唐语薇线](../主线/苏璃和唐语薇线.md) 承担关系定情。本线只补生活层：

- 出发前车况检查、装备、天气和路线。
- 路上停靠、加油、吃饭、避雨、看江景。
- 两人不靠案件推动关系，而是在普通行程里把玩笑停下来。
- 旅行地点只需满足江州近郊成立，不急着升格为正式地理条目。

边界：这场旅行不让唐语薇同行，不接入调查任务，不用“只剩一间房”强迫亲密。

### 二、车行与骑行圈轻量日常

车行日常归 [车行线](车行线.md)，本线只处理“车行之外”的骑行生活。

可写内容：

- 唐语薇检查车况后，嘴上嫌弃、手上细致。
- 苏璃熟悉路线与夜风，晏林负责确认安全节奏。
- 唐语薇偶尔参加共同骑行，但第一卷完整展开最多一次。
- 骑行不是炫技场，重点是信任、速度感、边界和返程后的沉默。

### 三、案件后的降压场

安若曦事件、管理局问询或工作室异常来访之后，可以安排一个低烈度日常场。

作用：

- 让人物从“处理问题”回到“还能正常吃饭、散步、回消息”。
- 让普通城市生活反衬隐秘压力。
- 让女主之间的关系在非案件场里出现微小变化。

边界：降压场不能抹掉事件后果，也不能用轻松日常替代该承担的记录、证据和授权。

### 四、工作室休息日

若没有正式咨询、危机预约和协查任务，可以安排工作室休息日。

可写内容：

- 许知夏、许知秋拉晏林出门。
- 安安寄存或遛狗提供温婉入口。
- 晏林确认预约表、危机消息和工作安排都清空后才离开。
- 周末活动只接生活面，不碰咨询隐私。

## 中期用法

中期周末日常线可承担“多人关系开始需要协商生活”的部分。

可写方向：

- 谁能约到晏林的周末时间。
- 临时协查打断私人安排后，谁承担失落，谁愿意理解。
- 女主之间不是只在危机场相遇，也会在吃饭、骑行、遛狗、训练后恢复中互相看见。
- 多伴侣关系不只靠宣言成立，也要经过时间表、边界、公开程度和普通生活习惯磨合。

中期仍不建议把所有人固定成大团建。周末场要有组合差异，给双人、三人和多人场不同功能。

## 与其他线接口

- **苏璃和唐语薇线**：骑行、旅行、车行外日常和三人关系偏移。
- **车行线**：车辆检查、维修后试车、骑行圈客户和职业边界。
- **酒吧线**：下班夜宵、深夜回程、成年人社交后的清醒边界。
- **工作室日常线**：休息日、闭店后、双胞胎拉晏林离开工作状态。
- **漫展线**：从周末活动场型中独立出来，专门承接江岚私下 coser 反差、双胞胎出行和活动场边界。
- **温婉线**：安安、宠物诊所、遛狗和低压慢热接触。
- **探亲与家庭回声线**：家庭电话、普通饭桌和休息日探望。
- **江州城市回声线**：日常场提供城市正常运行的底色，反衬异常只在缝隙里留痕。
- **协商成阵线**：中后期把周末时间、陪伴顺序和公开程度转成关系协商材料。

## 世界观启用点

- 江州近郊骑行路线、江边步道、观景台、农家乐、夜市或活动场馆，先作为候选生活场，不直接写进正式地理真源。
- 若某地点反复承担剧情功能，再回写 `worldbuilding/regions/`。
- 摩托车品牌、装备、维修耗材和路线名优先用泛称或架空专名；等正文反复调用时再统一命名。

## 不得写偏

- 不把周末日常写成无功能水场。
- 不让所有女主长期固定大团建，冲淡各自关系质地。
- 不用郊游、骑行或夜宵跳过正式关系确认。
- 不让骑行变成危险炫技或星契能力展示场。
- 不让休息日破坏工作室职业可信度。
- 不把候选地点提前升格为江州正式地理。
- 不新增女主或新势力。

## 待确认

1. 第一卷是否保留一场完整周末日常，还是只用短场穿插。
2. 苏璃与晏林两天一夜旅行的近郊目的地、路线和天气。
3. 唐语薇是否在第一卷参加一次共同骑行，若参加，发生在苏璃旅行前还是后。
4. 工作室周末无预约日是否与漫展场合并，还是另设普通郊游 / 遛狗场。
5. 温婉的宠物诊所日常是否通过安安寄存进入周末线。
6. 江州哪些生活地点会反复使用，值得后续升格为正式地理条目。
