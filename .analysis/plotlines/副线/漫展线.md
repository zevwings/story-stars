# 漫展线

> 本页是周末漫展、架空同人活动、coser 私下面和工作室休息日出行的施工设计，不是正式正典。正式事实以 `BIBLE.md`、[江岚主卡](../../../characters/preentry/江岚/主卡.md)、[安若曦和江岚线](../主线/安若曦和江岚线.md)、[许知秋和许知夏线](../主线/许知秋和许知夏线.md)、[工作室日常线](工作室日常线.md) 与 [周末日常线](周末日常线.md) 为准。
>
> 本线只负责生活场、私下面反差和低压交集，不新增女主，不把现实 IP、真实展会或真实地点写入正典。

## 剧情定位

漫展线负责把工作室、双胞胎、江岚和晏林从职业空间带入城市生活空间。它不是大型娱乐地图，也不是二次元题材支线，而是一条低压日常副线。

核心功能：

1. 给江岚一次离开律师身份、离开安若曦闺蜜身份的私下面场。
2. 给许知夏、许知秋展示工作室之外行动力和分寸感。
3. 给晏林一个练习“认出别人私下面后如何守边界”的场景。
4. 给后续亲密和协商提供生活气，而不是只靠案件、证据和授权谈话推进。

本线命题：**一个人在职业场里有边界，在私人场里也有边界；真正的靠近，是先确认对方愿不愿意被认出。**

## 启动条件

漫展场必须满足以下前提：

- 当天无正式咨询、无危机预约、无管理局协查任务。
- 晏林已确认工作室预约表、紧急消息和待处理记录。
- 许知夏、许知秋的出行理由只来自休息日、团建或个人兴趣，不使用咨询隐私和来访者信息制造玩笑。
- 江岚与晏林此前已经有至少一次职业合作或边界摩擦；漫展不是她的第一次人物入口。

## 第一核心节点：周末无预约日

### 触发

工作室连续被咨询、记录和协查挤压后，出现一个真正空出来的周末。许知夏抢到活动票，许知秋以“老板强制休息”或“员工心理健康建设”为名把晏林拖出工作状态。

晏林可以抗拒热闹，但不能因为责任感模糊就留在工作室空耗。他确认没有待处理事项后才出门。

### 场景功能

- 许知夏负责路线、时间表、抢票和现场热闹。
- 许知秋负责补刀、收场、控制调侃边界。
- 晏林第一次看见双胞胎在工作室之外的生活节奏。
- 工作室日常从职业空间扩展到城市生活。

## 第二核心节点：江岚私下 coser 偶遇

### 触发

晏林在漫展现场偶遇以 coser 身份参加活动的江岚。她不是职业 coser，也不是靠角色营业谋生，只是在私人时间认真参与自己喜欢的事。

### 写法重点

职业场景里的江岚是刚转正律师：克制、清楚、把授权、记录和责任拆得很明白。

漫展里的江岚可以是甜的、轻的、认真整理发夹和裙摆的，会在镜头前露出更柔和的笑，但这不是换人格。她仍然知道边界，仍然会判断谁可以靠近、靠近到哪一步。

### 晏林反应

晏林认出她后，第一反应不是当众叫破，也不是用熟人身份套近乎，而是确认她是否愿意在这个场合被认出。

可写动作：

- 他没有直接喊“江律师”。
- 他避开围观者，等江岚先看见他。
- 他先问一句“现在方便打招呼吗”。
- 他没有向双胞胎解释她的职业身份或与安若曦的关系。

### 阶段变化

- 江岚确认晏林在私下场合也懂得守边界。
- 晏林第一次看见江岚不作为律师、不作为安若曦闺蜜、不作为选择权守门人时的私人面。
- 双胞胎看出两人认识，但能把调侃停在安全范围内。

## 可扩展节点

| 节点 | 功能 | 边界 |
| --- | --- | --- |
| 排队与让位 | 展示晏林如何处理拥挤、拍照、陌生人请求 | 不写成英雄救美或肢体占有 |
| 道具修补 | 让许知夏、许知秋或晏林帮忙处理小事故 | 不引出真实 IP 或星契能力 |
| 临时避雨 | 接节气与天气线，给低压停留 | 不用避雨强迫亲密 |
| 活动结束夜宵 | 接饮食烟火线，形成回程余韵 | 不在公众场合泄露江岚身份 |
| 再次约定 | 江岚决定是否下次主动告诉晏林活动信息 | 不能等同恋爱确认 |

## 与江岚主线的关系

漫展线不是江岚的独立恋爱主线。江岚与晏林的信任和吸引仍然必须通过 [安若曦和江岚线](../主线/安若曦和江岚线.md) 里的法律边界、选择权冲突、误判修正和三方说开逐步建立。

本线只承担一个关键补面：让晏林和读者看见江岚在无须替别人预判风险时，仍然是一个有私人爱好、有审美、有松弛感的人。

## 与其他线接口

- **工作室日常线**：提供周末无预约日、双胞胎拉晏林出门的启动条件。
- **周末日常线**：漫展是周末活动的一种，不替代骑行、郊游、夜宵和其他休息日场型。
- **许知秋和许知夏线**：双胞胎通过组织出行展示工作室外的生活面，但不借此推进知情层级。
- **安若曦和江岚线**：江岚私下面反差服务其独立吸引，不靠安若曦背书。
- **饮食烟火线**：活动结束后的夜宵、分账和回程余韵可作为收束。
- **江州城市回声线**：漫展作为普通城市文化活动，只提供生活底色，不扩展星契社会公开程度。

## 世界观启用点

- 展会名称、场馆名称、作品名、角色名全部架空。
- 江州是否存在固定大型活动场馆，先作为候选城市空间，不直接升格为正式地理真源。
- 若漫展场馆后续反复承担剧情功能，再回写 `worldbuilding/regions/`。
- 普通参与者默认不知道星辰之力；本线不用于公开星契社会。

## 不得写偏

- 不使用现实 IP、真实角色、真实展会名或真实组织名称。
- 不把江岚写成职业 coser、擦边营业者或靠流量谋生的人。
- 不让晏林当众叫破“江律师”、安若曦闺蜜或工作室法律顾问身份。
- 不让双胞胎公开传播、追问账号、逼江岚解释或利用工作室信息开玩笑。
- 不在有正式咨询、危机预约或协查任务时安排出行。
- 不让漫展替代江岚的法律线、安若曦线或正式恋爱确认。
- 不新增女主、新势力或现实娱乐圈支线。
