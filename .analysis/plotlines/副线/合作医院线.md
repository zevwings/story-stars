# 合作医院线

> 本页是合作医院、精神科支持、转介链、危机干预和现实诊疗边界的施工设计，不是正式正典。医疗事实以 `worldbuilding/`、`BIBLE.md` 与相关人物线为准。

## 剧情定位

合作医院线负责把“异常不等于万能解释”落回现实医疗系统。

它不是管理局的附属部门，也不是晏林的后台，而是工作室、来访者、家属和协查案件之间的现实缓冲层。这里处理助眠、复诊、转介、药物、急性危机和家属沟通，但不替代星辰契约判断，更不替代法律责任。

## 主要功能

- 维持晏林咨询职业的现实可信度。
- 给魏琳、温婉、林芷兰及其他来访者提供可回收的医疗落点。
- 为精神污染、长期失眠、急性惊恐、药物使用和自伤风险提供现实归口。
- 给管理局协查提供医疗材料的边界参照。

## 角色位

| 角色位 | 功能 | 备注 |
| --- | --- | --- |
| 合作精神科医生 | 风险评估、药物、复诊建议 | 只做专业判断，不替晏林写剧情答案 |
| 督导师 / 资深心理师 | 伦理提醒、个案督导、职业边界校正 | 可与工作室建立长期往来 |
| 门诊护士 / 评估人员 | 分诊、流程、记录 | 只按现实职业写 |
| 转介社工 / 协调员 | 家属沟通、转介衔接、复诊安排 | 可承接高压家庭场景 |

## 可承载的关系面

- **正向亲密**：成年伴侣的共同就医、照护、等待与回家，写信任和分担，不写成工具化依赖。
- **黑暗亲密**：家属越权、强制留观、借治疗名义控制、把脆弱当成可支配资源。

这些关系面只作为现实伦理对照，不抢主线亲密位。

## 阶段骨架

### 一、转介起点

魏琳、温婉或其他来访者相关的异常、失眠或惊恐样事件先回到现实医疗，合作医院确认需要晏林介入的部分与不能介入的部分。

### 二、边界确认

晏林在助眠、梦境辅助或协查前，先完成现实评估与授权；医院只提供医学链条，不接管星辰契约解释权。

### 三、危机回收

当个案出现持续失眠、自伤风险、药物依赖、急性崩溃或精神污染余波时，医院负责现实处置，工作室负责后续心理收束。

### 四、后续复诊

复诊和回访让“事件已经过去”变成“现实后果还在”，也为温婉、林芷兰等慢热线留出持续接触空间。

### 五、高端商务会所案件接口

被带离会所的成年女孩先进入合作医院的现实处置链，而不是直接留在晏林身边。

- 医院负责药物筛查、伤情记录、睡眠与急性惊恐评估、必要留观和后续复诊。
- 本人清醒后分别决定是否接受心理支持、是否授权把相关信息交给警方与管理局、是否继续参与取证。
- 医院不负责判断星辰之印，也不把亲密、出台或求救经历等同于自愿承担后果。
- 若发现星辰契约污染或持续控制疑点，再转管理局直属疗养院；普通身体与心理恢复仍可留在合作医院完成。

## 不得写偏

- 不把医院写成万能治疗点。
- 不把精神疾病等同于星辰契约异常。
- 不让晏林绕过医学边界做诊断。
- 不让家属口头授权替代本人边界。
- 不把危机干预写成高光代替后续责任。
- 不把被救离现场写成自动成为主角随行者或亲密关系成员。
