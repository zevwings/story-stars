# 酒吧线

> 本页是苏璃所在酒吧、夜生活社交、外侧等待区与酒后边界的施工设计，不是正式正典。酒吧事实以 `story/synopsis.md`、`BIBLE.md` 与 [苏璃和唐语薇线](../主线/苏璃和唐语薇线.md) 为准。

## 剧情定位

酒吧线承担城市夜色、普通社交、情绪松动和外部压力显影。

它不是猎艳场，也不是纯案件场，而是成年人的社交密度空间：有人来喝酒、聊天、放松、短暂停靠，也有人在这里把试探、控制、误解和越界伪装成“氛围”。苏璃在这里首先是调酒师；她可以处理进入自己视线和职责范围的现实越界，但不负责识破所有异常。

## 最小空间结构

- **吧台**：苏璃的主要工作位，也是晏林当晚停留和观察大厅的位置。
- **开放大厅**：承接普通客流、服务动线和视线交叉。
- **高背半开放卡座**：唐雨霖、安若曦与随行人员所在位置；从吧台能够看见人员状态，但听不清完整谈话。
- **后场与出入口**：服务人员与客人的正常离场动线；暂不扩展包厢、暗门或复杂安保结构。

安若曦事件不使用封闭包厢。欲望契约者伪装成唐雨霖随行者，在卡座内维持连接；晏林从吧台发现异常，以给陌生美女请酒的普通搭讪方式进入近距离。苏璃不参与识别、截断或离场安排。

## 主要功能

- 给苏璃提供稳定的职业场域。
- 给安若曦事件提供第一处现实交叉入口。
- 给晏林和苏璃的关系提供日常、夜宵、玩笑和停下来再说的空间。
- 给普通都市层的亲密、误会和边界试探提供对照。

## 角色位

| 角色位 | 功能 | 备注 |
| --- | --- | --- |
| 酒吧老板 | 场地规则、经营压力、外部协调 | 可是本地熟人，也可只负责出场边界 |
| 资深调酒师 / 同事 | 日常协作、看场、分流 | 可承接苏璃的侧面日常 |
| 门口安保 | 出入控制、冲突拦截 | 负责把越界挡在门外 |
| 常客 / 熟客 | 日常噪声、关系温度、口碑传播 | 不都要进主线 |
| 合作方 / 物业 / 供应链 | 现实压力来源 | 可与唐家外围压力互相映照 |

## 可承载的关系面

- **正向亲密**：成年人之间的陪酒、送回家、留一杯不逼问、在喧闹里确认对方状态。
- **黑暗亲密**：借酒施压、偷拍视频、诱导失衡、把拒绝说成“再想想”。

重点不是把欲望写热，而是把知情、自愿、可撤回写稳。

## 阶段骨架

### 一、日常营业

酒吧以稳定经营为主，苏璃在这里完成工作、看人和接住日常噪声。

### 二、异常显影

安若曦事件进入酒吧后，现实纠纷、外部追逐和轻度星辰契约干扰在这里交叠显影。晏林独自判断异常，以“美女，请你喝一杯？”为切口接近卡座并截断欲望连接；截断只中止施术者与安若曦之间的输入，不摧毁施术者的星辰契约。

### 三、证据留存

账单、监控和时间线可以在酒吧保留，但苏璃不因职业身份自动成为案件调查者。监控只能记录晏林普通搭讪、安若曦离席、欲望契约者随后离场等现实动作，不能直接拍下精神截断。

### 四、事后回声

经营方压力、物业关系、投诉、外部合作和熟客流言，构成酒吧的长期后果层。

## 不得写偏

- 不把酒吧写成灰色法外场。
- 不让每个常客都变成阴谋角色。
- 不用酒精替代情感建立。
- 不用暧昧替代同意。
- 不让苏璃只作为陪聊工具存在。
