# 都市星契 · 文风总纲（STYLE）

> 本文件是本项目**文风真源**：正文写作限制、作者腔、叙述口径、角色语气底线与表达偏好以本文为准。
>
> 若本文与 `BIBLE.md` 冲突，以 `BIBLE.md` 为准（事实层）；文风方法与 story-writer 公共规则冲突，以本文为准。
> 人物具体说话方式仍以人物卡与上下文为准；本文只保存项目级共性口径。

## 一句话定义

- 贴身跟男主的**第三人称限知、细腻沉浸的甜宠都市异能后宫**。亲密为主调、日常蓄势、修行点缀；**甜是主旋律，创伤与黑暗是让甜更有重量的张力局部，终归被爱治愈**。

## 叙述口径

- **视角**：第三人称限知，主跟男主；关键情感 / 救赎节点可短暂切入当场女主。女主内心保留悬念，靠男主的观察与共鸣桥段渐次揭开，不提前全知剧透。
- **时态 / 层次**：常规过去时叙述；现实层与精神层切换时明写时间刻度（现实几秒、精神几幕，见战斗语法）。
- **叙述距离**：贴身沉浸——近距离贴人物的感官、呼吸、心理，尤其亲密与情感戏。
- **信息释放**：女主的创伤、秘密、异能代价渐次释放（前期隐秘并行 → 中期暴露）；悬念靠男主限知视角自然制造。

## 语言原则

- **句式**：细腻绵密为主，长短句配合节奏——亲密与情感戏放慢、留呼吸；战斗与冲突收紧、利落。
- **用词**：具体、有温度、感官精准；克制辞藻，不堆华丽（修辞为"更准"不为"更漂亮"）。
- **对话**：生活化、有人物区分度；甜与暧昧靠对话的拉扯与人味托出，不靠旁白直接说甜。
- **幽默（诙谐鲜活）**：日常、暧昧与亲密前奏中的自然幽默。拒绝段子腔与网络热梗，依靠角色性格的碰撞、理性吐槽、情境反差以及熟稔之后的“损嘴、反讽与自嘲”来体现人味。
- **影视感（镜头与空间）**：严肃、战斗与高张力场景中，保持镜头切换明确。动作、姿态与具体的空间物理关系要交代清楚，避免玄幻式的“悬空解释”与概念乱飞。
- **抒情**：克制、为情感服务，不无病呻吟；情绪藏在细节与动作里。
- **解释密度**：低——重展示轻解释。设定 / 异能 / 战斗不堆料，每场只端一到两个轴，读者在情节里自然理解。

## 角色声音

- **主要人物（男主）**：外冷手狠、见过黑暗，对外狠、对内守边界；话少、精准，偶有直白利落的锋利感；内心细腻（限知视角里可见他的观察与克制）。
- **配角群（7 女主）**：各有差异化的声音、身体语言与心理节奏，随角色卡定；避免同质化甜妹。
- **叙述称呼**：男主与女主用名 / “他” / “她”。
- **禁用称呼或口癖**：禁油腻旧网文称呼（娇躯、玉体、樱唇等）；禁功能词外露；禁半截词 / 碎片词滥用。病灶词表随写作积累（见「反模板化」）。

## 题材与氛围

- **主色调**：**甜宠轻快为主旋律**——明亮、暧昧、有生活气。创伤、救赎、黑暗线是**张力局部**（暗礁与深度），不盖过甜的底色；伤后回归被爱治愈，是甜的深化而非沉溺。
- **爽点（双轨爽感）**：
  1. **情感轨（核心）**：关系推进、暧昧拉扯中的“占有与被占有”、信任交付、修罗场中的妥协与共生。
  2. **剧情轨（网文爽感）**：冲突兑现快，不拖泥带水。男主利用双子星暗伴星（截断、逆流）进行利落的防反与降维压制，反转干净。
- **虐点**：**节制**——创伤 / 误会 / 退出威胁 / 黑暗场（特例）制造张力，但**必回甜、必救赎**，不沉溺虐；黑暗场落后果后走关系修复。
- **悬念**：女主秘密、异能代价、势力博弈、黑暗线渐次揭开；男主限知视角自然承载。
- **日常**：占篇幅大（定调第二位）——都市 / 职场 / 学院 / 疗养 / 关系日常，是甜宠与暧昧的主场、亲密的蓄势场；写扎实、有生活质感，不当过场。
- **亲密 / 高张力场景**（正向 HD 为主体）：
  - 三基调：**写实 ＋ 美感修辞、纯写实不用异能滤镜、张弛结合**。
  - 美感落点：感官之美（真实有温度）、情感之美（信任 / 交付 / 被看见）、节奏之美（张弛留白）。
  - 软支撑四红线（不功利化）：亲密不回星力、不涨战力、不提数值；只在情感 / 救赎维度；因果方向——先有欲望情感才有亲密，收益是自然结果不是目的。
  - **人物清醒可拒绝**：女主推拒来自羞耻 / 别扭 / 情趣拉扯；绝对主导可成立，但那是情趣、不是压迫。
  - 黑暗场（特例）走**双声部铁律**：快感是施暴方的、账单是受害方的；文本不认同施暴方；痛苦与快感同重量在场；必落后果，不用甜蜜余韵收尾。不混入正向 HD。

## 反模板化

- **禁油腻旧网文亲密词**：娇躯、挺立、花蕊、樱唇、电流窜过……病灶词表随写作积累。
- **拒设定信息倾倒**：尤其战斗 / 异能，重展示轻解释，每场只端一两个轴。
- **拒甜宠悬浮**：甜要有日常与信任的根，不空甜、不无来由的宠。
- **拒女主同质化**：每位的声音、反应、身体语言要能区分。
- **保留人味**：人物的别扭、迟疑、笨拙与真实反应，不磨成完美工具人。

## 风格锚点与应用场景

为了实现本故事的叙事诉求与情绪节奏，写作时应根据场景类型，动态切换以下五种文风锚点：

- **克制叙事型（15% 严肃/战斗底色）**：
  - *适用*：严肃剧情推进、势力背景展开、战前侦察。
  - *要求*：叙述稳重，只写动作与事实，少喊叫，少用异能虚浮滤镜。
- **爽感推进型（20% 气场/反转）**：
  - *适用*：面临挑衅、黑市周旋、反派踢馆、战斗反击。
  - *要求*：节奏加快，男主展现冷酷且游刃有余的气场，冲突即时兑现，反转干净利落。
- **互动松弛型（25% 日常/暧昧）**：
  - *适用*：诊所日常、下班同行、修罗场边缘、多角色同处一室。
  - *要求*：对话有来有回，充满拉扯、温度与细微的性格摩擦感。
- **情感救赎/抒情沉浸型（15% 情绪高潮/亲密）**：
  - *适用*：创伤揭开时、心理诊疗入梦、生死交付、情感确立与亲密递进。
  - *要求*：不仅是描写氛围与感官，更要刻画“被看见的创伤、卸下防备的脆弱、灵魂上的信任交付与治愈”，长短句配合，营造极强的包裹感与情绪美感。
- **诙谐鲜活型（10% 趣味拉扯）**：
  - *适用*：日常拌嘴、亲密前奏的逗弄、事后余温的轻松、关系试探中的损嘴。
  - *要求*：用性格差制造幽默（如医生的理智吐槽 vs 女主的别扭嘴硬），展现角色的别扭与人味。
