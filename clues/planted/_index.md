# 已埋线索索引

> 本页只负责 `planted` 线索目录，不横向展开其他状态线索或章节正文。

本目录存放已经在正文中埋下、但尚未明显指向答案的线索。每条线索一个 Markdown 文件。

## 用途

- 登记已经埋下、但角色和真相还不能一次说破的线索。
- 写续章时，用来确认某条线现在能否继续压住、轻轻加重或暂时不碰。

## 目录内容

- 放入处于 `planted` 状态的线索正文。

## 建档模板引用

- 新建线索时，可复制上级线索模板，填写后放入本目录。

## 读取口径

- 写续章时，这一层通常是线索读取的第一站。
- 重点不是立刻回收，而是确认“现在能不能再压一下”。
- 如果只需要判断是否存在某条长线，先停在本页即可。

## 使用提醒

- `planted` 不等于“还很远”，而是“读者已被钉住，但角色和真相还不能一次说破”。
- 如果当前章节只需要保持这条线活着，优先延续它的存在感，不要急着解释答案。
- 若某条线索已经进入明确反复铺垫，再考虑转去 `hinted`。
